# 选必第二册·第三章 交变电流（精读）

> 主线：发电机原理 → 交变电流的瞬时 / 峰值 / 有效值 / 周期 / 频率 → 变压器原理与变换关系 → 远距离输电的损耗分析。
> 核心方程：$e = E_0\sin\omega t$、$U_\text{有效} = U_0/\sqrt 2$、$U_1/U_2 = n_1/n_2$、$I_1/I_2 = n_2/n_1$。

## 1. 交变电流

### 定义

- 大小和方向都随时间作周期性变化的电流叫**交变电流 (Alternating Current, AC)**。
- 与之对比：方向不变的电流叫**直流 (DC)**。
- 我们生活中的市电就是交变电流（中国为 50 Hz、220 V）。

### 交变电流的产生：发电机

#### 简单交流发电机
- 矩形线圈（$N$ 匝、面积 $S$）在匀强磁场 $B$ 中**绕垂直于 $B$ 的轴**匀速转动（角速度 $\omega$）；
- 当线圈平面**平行**磁场时（处于"中性面外"），切割磁感线速度最大，电动势瞬时值最大；
- 当线圈平面**垂直**磁场时（处于"中性面"），切割速度为 0，电动势瞬时值为 0；
- 同时**电流方向**在两个中性面之间改变。

### 中性面

- 线圈平面与 $B$ **垂直**时的位置 = 穿过线圈的磁通量**最大**的位置（$\Phi = BS$）；
- 该位置：$\Phi$ 最大但**变化率为 0** → $\varepsilon = 0$；
- 线圈每经过中性面一次，**电流方向就改变一次**；
- 每个周期内线圈经过中性面 **2 次**。

### 瞬时电动势

设从中性面开始计时（$t = 0$ 时 $\Phi$ 最大）：

$$\boxed{\,e = E_0\sin(\omega t)\,}$$

- $E_0 = NBS\omega$：最大值（峰值）
- 若从**平行磁场位置**开始计时（$\Phi = 0$）：$e = E_0\cos(\omega t)$

### 几个关键性质

- 频率：$f = \omega/(2\pi)$
- 周期：$T = 1/f = 2\pi/\omega$
- 中国市电：$f = 50$ Hz，$T = 0.02$ s，$\omega = 100\pi$ rad/s

## 2. 交变电流的描述

### 瞬时值、最大值、有效值

#### 瞬时值
- 任意时刻的电流 / 电压值。

#### 最大值（峰值）
- 一个周期内瞬时值的最大值；
- 用下标 0 表示：$U_0, I_0, E_0$。

#### 有效值（关键概念！）

> 让一个交流电与一个直流电分别通过相同的纯电阻，在**相同的时间**内产生**相等的热量**，那么交流电的有效值等于这个直流电的电流值。

对**正弦交流电**：

$$\boxed{\,I_\text{有效} = \dfrac{I_0}{\sqrt 2},\quad U_\text{有效} = \dfrac{U_0}{\sqrt 2}\,}$$

- "220 V"、"110 V" 等市电额定值都是**有效值**；
- 中国市电峰值 $U_0 = 220\sqrt 2 \approx 311$ V；
- 电表、电气设备铭牌上的电压、电流值都是有效值。

#### 平均值
- 半个周期内瞬时值的算术平均：

$$\bar I = \dfrac{2 I_0}{\pi}, \quad \bar U = \dfrac{2 U_0}{\pi}$$

- 用于计算**电量** $q = \bar I \cdot t$（半周期电量）。

### 三个值之间的换算

| 量 | 最大值 | 有效值 | 平均值 |
|---|---|---|---|
| 公式 | $U_0$ | $U_0/\sqrt 2 \approx 0.707 U_0$ | $2U_0/\pi \approx 0.637 U_0$ |
| 用途 | 写表达式、画图 | 计算电功、电热、推导功率 | 计算电量（半周期） |

### 各物理量的适用值

| 物理量 | 用哪个值 |
|---|---|
| 电流通过电阻产生的热量 | 有效值（$Q = I_\text{有效}^2 R t$） |
| 电功率 | 有效值（$P = U_\text{有效} I_\text{有效}$） |
| 电容器的击穿电压 | **最大值**（$U_0$） |
| 电流通过电解槽的电量 | **平均值**（$q = \bar I t$） |
| 交流电表读数 | 有效值 |

### 注意：不是所有交流都是正弦

- 方波、三角波等也是交变电流；
- 这些的有效值与峰值的关系**不是** $1/\sqrt 2$，需要根据定义重新计算。
- 例：方波（峰值 $I_0$，正负交替）→ 任意时刻 $|I| = I_0$ → 有效值 = $I_0$。

### 例题：手机充电器

> 充电器输入 220 V 交流（有效值），输出 5 V 直流。
- 输入瞬时电压峰值 = $220\sqrt 2 \approx 311$ V；
- 计算输入热功率时用 220 V。

### 易错提示

- **市电 220 V 是有效值**，最大值约 311 V。
- 电热设备计算用有效值。
- 元件耐压时要用最大值 → 否则瞬间击穿。
- 半周期电量用平均值，整周期电量为 0。

## 3. 变压器

### 结构

- 闭合铁芯 + 两个绝缘绕组（原线圈 / 副线圈）；
- 铁芯通常用**硅钢片叠层**（减小涡流损耗）。

### 工作原理

1. 原线圈接交流电源 → 通过铁芯产生交变磁通；
2. 该交变磁通同时穿过原 / 副线圈；
3. 由法拉第电磁感应：原 / 副线圈中分别产生感应电动势。

> 互感 + 法拉第定律 + 同一磁通 → 电压变换。

### 理想变压器假设

1. 没有漏磁——铁芯把所有磁通约束在内；
2. 没有铜损（线圈无电阻）；
3. 没有铁损（铁芯无涡流损耗）。

### 电压变换公式

$$\boxed{\,\dfrac{U_1}{U_2} = \dfrac{N_1}{N_2}\,}$$

- $N_1, N_2$：原 / 副线圈匝数
- $N_1 > N_2$：**降压变压器**
- $N_1 < N_2$：**升压变压器**

### 电流变换公式（理想变压器）

理想变压器**功率守恒**：

$$P_1 = P_2 \implies U_1 I_1 = U_2 I_2$$

→

$$\boxed{\,\dfrac{I_1}{I_2} = \dfrac{N_2}{N_1}\,}$$

- 即电流变换比与电压**相反**——升压变压器降电流，降压变压器增电流。

### 多副线圈

- 若副线圈有多组 $N_2, N_3, \dots$，则各组电压按匝数比：

$$\dfrac{U_2}{N_2} = \dfrac{U_3}{N_3} = \dfrac{U_1}{N_1}$$

- 总功率守恒：$U_1 I_1 = U_2 I_2 + U_3 I_3 + \dots$

### 变压器只能改变交流电

- 直流通过原线圈：稳定 → $\Phi$ 不变 → 副线圈无感应电动势；
- 故变压器**只对交流电起作用**。
- 通断瞬间会有瞬时电压（自感原理），不是稳定输出。

### 实际变压器

- 铜损：原副线圈电阻发热；
- 铁损：铁芯磁滞 + 涡流损耗；
- 效率：$\eta = P_2/P_1$，一般 95%–99%；
- 减小损耗：硅钢片叠层 + 铜线粗 + 优化磁路。

### 应用

- 输电（先升压再降压）；
- 调压器、电源适配器、充电器；
- 电焊机、感应炉。

### 例题：充电器变压器

> 输入 220 V，输出 5 V，原线圈 880 匝，求副线圈匝数。
- $N_2 = N_1 \cdot U_2/U_1 = 880 \times 5/220 = 20$ 匝。
- 若副线圈电流为 1 A，则原线圈电流 $I_1 = U_2 I_2/U_1 = 5/220 \approx 0.023$ A。

### 易错提示

- 变压器变换公式**只适用于交流电**。
- 比值用匝数比（不是电压绝对值）。
- 电流比与电压比**相反**（不是相同）。
- 理想变压器假设是简化模型，实际有损耗。

## 4. 电能的输送

### 远距离输电的核心问题

- 发电厂功率 $P$ 通过输电线（总电阻 $r$）输送到用户；
- 输电线上的电流为 $I = P/U$（$U$ 为输电电压）；
- 输电线上消耗：

$$P_\text{损} = I^2 r = \dfrac{P^2}{U^2}r$$

→ **提高输电电压可大幅减少损耗**！

### 高压输电的原因

- 同等功率、同等线路下，$U$ 提高 10 倍 → $P_\text{损}$ 减少 100 倍；
- 这就是为什么远距离输电要用 110 kV、220 kV、500 kV 甚至特高压 1000 kV。

### 输电系统流程

```
发电厂 (10 kV–25 kV)
   ↓ 升压变压器
高压输电线 (220 kV–1000 kV)
   ↓ 降压变压器（变电站）
中压配电 (10 kV)
   ↓ 用户变压器
低压电网 (220 V/380 V)
   ↓
家庭 / 工厂
```

### 输电损耗的两种说法（要区分）

| 公式 | 适用条件 |
|---|---|
| $P_\text{损} = I^2 r$ | **永远适用**，本质（焦耳定律） |
| $P_\text{损} = U_\text{损}^2/r = (Ir)^2/r$ | 也适用，但 $U_\text{损}$ 不是输电电压 |
| $P_\text{损} = P^2 r/U^2$ | 只在 $I = P/U$ 时（即电压乘电流等于功率）适用 |

> **不能**写成 $P_\text{损} = U^2/r$（这是把输电电压当作线路电压，错！）

### 输电线两端电压

- 实际输电线两端电压**不是** $U$，而是 $U - U_\text{损}$（用户端）；
- 输电线上电压损失 $U_\text{损} = I r$。
- 设升压后输出电压为 $U$，则用户端实际接收 $U - Ir$。

### 输电方式：交流 vs 直流

#### 交流输电（AC）
- 易升降压（用变压器）；
- 主流方式。

#### 高压直流输电（HVDC）
- 适合超长距离、大容量；
- 无感抗、电容损耗，长距离效率更高；
- 中国"特高压"工程：±800 kV 直流，全球领先；
- 但变换为直流需要昂贵的换流站。

### 例题：远距离输电

> 发电厂功率 5000 kW，输电线总电阻 $r = 4$ Ω。
- 若用 5000 V 输电：$I = 1000$ A，$P_\text{损} = I^2 r = 4 \times 10^6$ W = 4000 kW（**80% 损耗**！）；
- 若用 50000 V 输电：$I = 100$ A，$P_\text{损} = 40$ kW（仅 0.8%）。

### STSE 视角

- 我国推行特高压直流输电：西电东送（云南、四川水电送往东部）；
- 减少损耗、提高电网稳定性、节约土地资源；
- 是国家"碳达峰碳中和"战略的关键基础设施。

### 易错提示

- $P_\text{损} = P^2 r/U^2$ 中 $U$ 是**输电电压**，不是用户电压；
- 不要把"减小 $r$"作为唯一节能手段——升压更有效；
- 输电系统中多次变压（先升后降）；
- 高压输电线杆塔高、绝缘子多 = 高压表现。

## 章末公式速查

```
交流发电机
e = E₀ sin(ωt)         （从中性面计时）
E₀ = N B S ω           （最大电动势）
T = 2π/ω, f = 1/T
线圈每周期经过中性面 2 次，方向改变 2 次

有效值（正弦）
U_有效 = U₀/√2         ≈ 0.707 U₀
I_有效 = I₀/√2

平均值（半周期，正弦）
U̅ = 2U₀/π            ≈ 0.637 U₀
半周期电量 q = I̅ · (T/2) = 2I₀/π · T/2 = I₀T/π

变压器（理想）
U₁/U₂ = N₁/N₂          电压比 = 匝数比
I₁/I₂ = N₂/N₁          电流比 = 匝数反比
P₁ = P₂                功率守恒

只对交流起作用！直流通过原线圈无副线圈电动势

输电损耗
P_损 = I²r = (P/U)²·r = P² r / U²
高压输电的核心：提高 U → 大幅降低 P_损

输电线压降 U_损 = I r
用户端电压 = U − U_损
```
