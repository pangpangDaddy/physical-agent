# 选必第二册·第一章 安培力与洛伦兹力（精读）

> 主线：磁场对通电导线的作用（安培力）→ 磁场对运动电荷的作用（洛伦兹力）→ 带电粒子在匀强磁场中的圆周运动 → 质谱仪 / 回旋加速器。
> 核心方程：$F = BIL\sin\theta$、$f = qvB\sin\theta$、$r = \dfrac{mv}{qB}$、$T = \dfrac{2\pi m}{qB}$。

---

## 1. 磁场对通电导线的作用力

### 安培力的发现

- 奥斯特发现电流的磁效应后，安培进一步发现磁场反过来也对通电导线施加作用力。
- 这种力称为**安培力 (Ampère's force)**。

### 安培力的方向：左手定则

> 伸开**左手**，让**磁感线**穿入手掌，**四指**指向电流方向，**拇指**所指方向就是导线受到的安培力方向。

- 安培力方向**同时垂直**于电流方向和磁感应强度方向。
- 三者构成右手坐标系：$\vec F = I\vec L \times \vec B$（矢量形式，高中不要求）。

### 安培力的大小

#### 通电导线**垂直于**磁场时

$$F = BIL$$

- $B$：磁感应强度（T）
- $I$：电流（A）
- $L$：导线在磁场中的有效长度（m）

#### 通电导线与磁场成 $\theta$ 角时

$$\boxed{\,F = BIL\sin\theta\,}$$

- $\theta$：电流方向与 $B$ 方向夹角
- 特殊情形：
  - $\theta = 90°$：$F = BIL$（最大）
  - $\theta = 0°$ 或 $180°$（电流沿磁场方向）：$F = 0$

### 有效长度

- "有效长度" = 导线在**垂直于 $B$ 方向**的投影长度。
- 弯曲导线在匀强磁场中受力：$F = BI L_{\text{有效}}$，其中 $L_{\text{有效}}$ = 始末端点的**直线距离**。
- 闭合回路在匀强磁场中所受**合力为零**（不论形状）。

### 应用

- **电动机**：通电线圈在磁场中受力转动；
- **扬声器**：音圈在磁场中受电流变化的力 → 推动纸盆振动发声；
- **磁悬浮列车**：电磁力悬浮和驱动；
- **电磁炮 / 弹射器**：高速发射弹丸。

### 易错提示

- 安培力方向用**左手**定则（与发电机右手定则不同）。
- $L$ 是导线在磁场中的有效长度，不是导线总长。
- 安培力**始终垂直于电流和 B**，即垂直于它们所张的平面。
- 闭合通电线圈在匀强磁场中合力为零，但**合力矩不为零**——这是电动机的核心原理。

## 2. 磁场对运动电荷的作用力（洛伦兹力）

### 从安培力推导

- 安培力本质上是磁场对运动**电荷**的作用力的宏观表现。
- 设导线截面积 $S$、长度 $L$、自由电荷数密度 $n$、电荷量 $q$、定向移动速度 $v$：
  - 电流 $I = nqSv$
  - 总电荷数 $N = nSL$
  - 安培力 $F = BIL = BnqSvL$
  - 单个电荷受力 $f = F/N = qvB$

### 洛伦兹力 (Lorentz force)

$$\boxed{\,f = qvB\sin\theta\,}$$

- $q$：电荷量（C，取绝对值）
- $v$：电荷运动速度
- $\theta$：$\vec v$ 与 $\vec B$ 的夹角
- 最大值（$\vec v \perp \vec B$）：$f = qvB$
- 最小值（$\vec v$ 平行或反平行 $\vec B$）：$f = 0$

### 洛伦兹力的方向：左手定则

- 与安培力相同——伸开左手，磁感线穿入手心，四指指向**正电荷运动方向**（或负电荷运动的**反方向**），拇指指洛伦兹力方向。
- **负电荷**洛伦兹力方向与四指指向相反——务必注意符号！

### 洛伦兹力的特征

1. **始终垂直于速度**——任何时刻 $\vec f \perp \vec v$；
2. **始终垂直于磁场**——$\vec f \perp \vec B$；
3. **不做功**！——因力与速度垂直，$W = Fv\cos 90° = 0$；
4. 只**改变速度方向**，不改变速率；
5. 与电场力 ($qE$) 区别：电场力方向沿 $E$，会做功；磁场力方向垂直 $v$，不做功。

### 重力的考虑

- **微观粒子**（电子、质子、α 粒子等）：$qvB \gg mg$，重力通常**忽略**；
- **宏观带电体**（带电小球、油滴）：重力必须考虑；
- 题目特别说明"不计重力"则忽略，否则带电小球类题目要考虑。

### 易错提示

- 洛伦兹力**永远不做功** —— 与电场力相反。
- 负电荷用左手定则时，**四指方向取速度的反向**。
- $\theta$ 是速度方向与 $B$ 方向夹角，不是与导线/法线的夹角。
- 速度平行磁场时，$f = 0$，粒子做匀速直线运动。

## 3. 带电粒子在匀强磁场中的运动

### 三种情形分类

| $\vec v$ 与 $\vec B$ 关系 | 运动 |
|---|---|
| **$\vec v \parallel \vec B$** | $f = 0$，做匀速直线运动 |
| **$\vec v \perp \vec B$** | 做**匀速圆周运动** |
| **$\vec v$ 与 $\vec B$ 斜交** | 做**等距螺旋线运动**（沿 $B$ 方向匀速 + 垂直 $B$ 圆周） |

### 匀速圆周运动（$\vec v \perp \vec B$ 重点）

- 洛伦兹力提供向心力：

$$qvB = \dfrac{mv^2}{r}$$

#### 半径

$$\boxed{\,r = \dfrac{mv}{qB}\,}$$

- 与速度成正比，与磁感应强度反比。
- 同一速度下，电荷量大或质量小 → 半径小。

#### 周期

$$\boxed{\,T = \dfrac{2\pi r}{v} = \dfrac{2\pi m}{qB}\,}$$

- 周期**只与粒子的荷质比 $q/m$ 和磁场 $B$ 有关，与速度无关**！
- 这是回旋加速器能工作的关键性质。

#### 频率

$$f = \dfrac{1}{T} = \dfrac{qB}{2\pi m}$$

### 螺旋运动（$\vec v$ 斜交 $\vec B$）

- 把 $\vec v$ 分解为：$v_\parallel$（沿 $B$）+ $v_\perp$（垂直 $B$）；
- $v_\parallel$ 不受力 → 沿 $B$ 方向匀速直线；
- $v_\perp$ 受洛伦兹力 → 在垂直平面内做圆周运动；
- 合成：螺旋线，**螺距**：

$$d = v_\parallel \cdot T = \dfrac{2\pi m v_\parallel}{qB}$$

### 解题套路（圆周运动）

1. 判断粒子运动平面（与 $B$ 垂直）；
2. 找圆心：圆心在初速度方向的**垂线**上（左手定则定方向）；
3. 确定半径：用 $r = mv/(qB)$ 或几何关系；
4. 分析弦长、弧长、转角；
5. 利用周期求时间：$t = (\alpha/2\pi)T$。

### 例题：粒子进入磁场后偏转

> 电子以 $v$ 沿水平方向进入垂直纸面向里的 $B$ 区域。
- 用左手定则：手心朝里、四指指 $v$ 方向（电子带负电故取反向）→ 拇指竖直向上。
- 故电子向上偏转，圆心在初速度方向的垂直方向（上方）。
- 半径 $r = mv/(eB)$。

### 高考典型题型

1. **方框磁场区域**：粒子从某边进入，求离开位置 / 时间。
2. **环形磁场**：用几何关系判断粒子轨道偏转角。
3. **磁聚焦**：粒子从同一点不同方向射入同一 $B$，因 $T$ 相同，相遇于"对称点"。
4. **磁通量积分**：粒子轨迹包围的面积 × $B$ = 磁通量（少见）。

### 易错提示

- 圆心**始终在初速度的垂线上**，方向由洛伦兹力方向决定。
- 周期与速度无关——速度越大半径越大、弧长越长，时间不变。
- 判断粒子运动方向要带电性正确——负电荷用左手定则时取速度反向。

## 4. 质谱仪与回旋加速器

### 质谱仪 (Mass Spectrometer)

#### 结构

- 离子源 → 加速场 → 速度选择器 → 偏转磁场 → 检测板。

#### 加速段
- 电压 $U$ 把静止的离子加速：

$$qU = \tfrac{1}{2}mv^2 \implies v = \sqrt{\dfrac{2qU}{m}}$$

#### 速度选择器（选学）
- 同时存在相互垂直的电场 $E$ 与磁场 $B_0$；
- 仅当 $qE = qvB_0$（即 $v = E/B_0$）时粒子直线通过；
- 其他速度的粒子偏离 → 被挡板过滤。

#### 偏转段
- 进入偏转磁场 $B$ 后做圆周运动：

$$r = \dfrac{mv}{qB} = \dfrac{1}{B}\sqrt{\dfrac{2mU}{q}}$$

#### 测量原理
- 不同质量 / 电荷量的离子在感光板上的落点距入射点不同；
- $D = 2r = \dfrac{2}{B}\sqrt{\dfrac{2mU}{q}}$
- 由 $D$ 反推 $m/q$ → 确定离子种类。

#### 应用
- 测同位素丰度（U-235 / U-238）；
- 化学元素分析、宇宙射线粒子鉴定；
- 法医毒物分析、地质年代测定。

### 回旋加速器 (Cyclotron)

#### 结构

- 两个 D 形金属盒（D 形电极）放在均匀磁场中；
- 两 D 盒之间加**交变高频电压**；
- 带电粒子从中心释放。

#### 工作原理

1. 粒子在 D 盒缝隙中被电场加速；
2. 进入 D 盒后只受磁场作用 → 做**半圆**运动；
3. 出 D 盒时电场已**反向**，再次加速；
4. 反复加速 → 速度越来越大，半径越来越大，但**周期不变**！

#### 关键：交变电压周期 = 粒子回旋周期

$$T_\text{电源} = T_\text{粒子} = \dfrac{2\pi m}{qB}$$

→ 电源频率必须固定为**回旋频率**$f = qB/(2\pi m)$。

#### 最大动能

- 设 D 盒半径 $R$：粒子最大圆周半径 ≈ $R$；
- $R = mv_\text{max}/(qB)$ → $v_\text{max} = qBR/m$；
- 最大动能：

$$E_k^{\max} = \tfrac{1}{2}mv_\text{max}^2 = \dfrac{q^2 B^2 R^2}{2m}$$

- 只与 $B$、$R$、$q/m$ 有关，**与电压大小无关**（电压只影响达到最大速度所需的加速次数）。

#### 限制

- 当粒子速度接近光速时，相对论效应使质量增大 → 周期变长 → 与固定频率电源失同步；
- 解决：**同步加速器**（同步调节磁场或频率）。

#### 应用

- 加速质子、氘核、α 粒子；
- 核物理研究、放射性同位素生产、医学诊断与治疗。

### 易错提示

- 回旋加速器中电源频率必须等于粒子回旋频率，且不随速度变化（直到相对论效应失效）。
- 加速电压 $U$ 只影响每次加速的能量增量 $qU$，不影响最终最大能量（最终能量由 $R$ 和 $B$ 决定）。
- 质谱仪中加速段用 $qU = \tfrac{1}{2}mv^2$，偏转段用 $r = mv/(qB)$，二者结合用。

## 高频易错速查

- [ ] 安培力或洛伦兹力方向用了右手定则
- [ ] 负电荷的洛伦兹力方向没取反
- [ ] 弯曲导线安培力把全部长度代入
- [ ] 圆周运动周期与速度有关
- [ ] 洛伦兹力做功不为零
- [ ] 回旋加速器最大动能与 $U$ 有关
- [ ] 微观粒子未忽略重力 / 宏观粒子误忽略重力
- [ ] 螺旋线运动当作纯圆周运动
