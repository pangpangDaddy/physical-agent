# 选必第二册·第二章 电磁感应（精读）

> 主线：感应电流的方向规律（楞次定律）→ 感应电动势的定量规律（法拉第定律）→ 电磁感应的具体应用（涡流、电磁阻尼、电磁驱动）→ 互感和自感。
> 核心方程：$\varepsilon = -\dfrac{\Delta\Phi}{\Delta t}$、$\varepsilon = BLv$、$\varepsilon = N\dfrac{\Delta\Phi}{\Delta t}$。

---

## 1. 楞次定律

### 感应电流方向的判定

> **楞次定律**：感应电流的方向，总是要**阻碍引起感应电流的磁通量的变化**。

简记口诀：**"阻碍变化"**——
- 原磁通量增加 → 感应电流的磁场方向**与原磁场相反**（阻止增加）；
- 原磁通量减少 → 感应电流的磁场方向**与原磁场相同**（阻止减少）。

### 应用步骤

1. **判断原磁场方向**（穿过回路的 $B$ 方向）；
2. **判断 $\Phi$ 变化趋势**（增大 / 减小）；
3. **由楞次定律确定感应磁场方向**（阻止变化）；
4. **由右手螺旋定则确定感应电流方向**：
   - 让大拇指指感应磁场方向，弯曲四指方向即为感应电流方向。

### 例题：磁铁靠近闭合线圈

- 磁铁 N 极向下靠近线圈：
  - 原磁场（磁铁产生）穿过线圈向下，$\Phi$ ↑；
  - 感应电流的磁场要阻碍 → 方向向上；
  - 由右手定则 → 从上往下看，感应电流逆时针流动。

### 楞次定律的另一表述：阻碍相对运动

> 当因导体与磁场发生相对运动而产生感应电流时，感应电流的效果总是**阻碍这种相对运动**。

- 典型情境：磁铁靠近线圈，感应电流的磁场对磁铁产生**排斥力**（阻碍靠近）；
- 磁铁离开线圈，感应电流的磁场对磁铁产生**吸引力**（阻碍远离）。

### 右手定则（针对切割磁感线情境）

> 伸开**右手**，让磁感线**穿入手心**，**大拇指**指**导体运动方向**，则其余**四指**指向感应电流方向。

- 用于"导体切割磁感线"这种特殊情形；
- 与楞次定律本质一致，但更直观。

### 易错提示

- 楞次定律的"阻碍"不是"阻止变化发生"，而是"阻碍变化趋势"——变化最终仍会发生。
- 判断原磁场方向时要看回路所在平面被穿过的 $B$。
- 区分楞次定律（适用任何情形）与右手定则（只适用切割磁感线）。
- 多匝线圈中感应电流方向相同，电动势加倍。

## 2. 法拉第电磁感应定律

### 内容

> 闭合回路中**感应电动势**的大小，跟穿过这一回路的**磁通量的变化率**成正比。

$$\boxed{\,\varepsilon = -\dfrac{\Delta \Phi}{\Delta t}\,}$$

- 负号表示感应电动势的方向（由楞次定律决定）；高中常只用绝对值。
- $\Delta \Phi$：磁通量变化量（Wb）
- $\Delta t$：变化所用时间（s）
- $\varepsilon$：感应电动势（V）

### 多匝线圈

$$\boxed{\,\varepsilon = N \cdot \dfrac{\Delta \Phi}{\Delta t}\,}$$

- $N$：线圈匝数
- 多匝时电动势按匝数倍增。

### 平均与瞬时

- $\varepsilon = N\dfrac{\Delta\Phi}{\Delta t}$ 给出**平均电动势**（对应一段时间 $\Delta t$）；
- $\varepsilon = N\dfrac{d\Phi}{dt}$ 给出**瞬时电动势**（导数，选学）。

### 三种基本情境

#### 情境 A：导体切割磁感线（直导体）

$$\varepsilon = BLv\sin\theta$$

- $L$：导体在磁场中的有效长度；
- $v$：导体相对磁场的速度；
- $\theta$：$\vec v$ 与 $\vec B$ 的夹角；
- 当 $\vec v \perp \vec B$ 且 $\vec L \perp$ 两者时：$\varepsilon = BLv$（最大）。

#### 情境 B：线圈在磁场中转动（交流发电机）

- 矩形线圈面积 $S$，匝数 $N$，绕垂直于 $B$ 的轴匀速转动，角速度 $\omega$：

$$\varepsilon(t) = NBS\omega\sin(\omega t)$$

- 最大瞬时电动势：$\varepsilon_0 = NBS\omega$
- 这是交变电流的产生原理。

#### 情境 C：磁场变化引起 $\Phi$ 变化（变压器、互感）

- 线圈匝数 $N$、面积 $S$ 不变，$B$ 随时间变化：

$$\varepsilon = NS\dfrac{\Delta B}{\Delta t}$$

### 电荷量公式

> 一个闭合回路（电阻 $R$，匝数 $N$）的总通过电荷量：

$$q = I\Delta t = \dfrac{\varepsilon}{R}\Delta t = \dfrac{N \Delta\Phi}{R}$$

- 只与 $\Delta\Phi$ 和 $R$ 有关，**与 $\Delta t$ 无关**！
- 用于"快慢一次性变化"的电量计算。

### 例题：导体棒切割磁感线

> 导体棒长 $L = 0.5$ m，以 $v = 4$ m/s 在 $B = 0.4$ T 的匀强磁场中（$\vec v \perp \vec B$，$\vec L \perp$ 两者）切割磁感线，闭合回路总电阻 $R = 2$ Ω。
- $\varepsilon = BLv = 0.8$ V
- $I = \varepsilon/R = 0.4$ A
- 安培力（外力对抗）：$F_\text{安} = BIL = 0.08$ N
- 维持匀速所需拉力 $F_\text{外} = F_\text{安}$；功率 $P = F_\text{外} v = 0.32$ W = $\varepsilon I$（能量守恒）

### 力学 + 电学的综合分析

- 切割磁感线问题本质：**机械能 → 电能** 的能量转换；
- 外力做功 = 电路消耗能量 + 动能变化；
- 安培力是阻碍运动的力（楞次定律的体现）。

### 易错提示

- $\Delta\Phi$ 是磁通量**变化量**，可能是 $B$ 变、$S$ 变、$\cos\alpha$ 变；
- 多匝线圈记得乘 $N$；
- $\varepsilon = BLv$ 中 $v$ 必须**垂直** $L$；
- $q = N\Delta\Phi/R$ 与 $\Delta t$ 无关——快/慢运动产生同样电量；
- 切割磁感线时安培力总是**阻碍**运动的（楞次定律）。

## 3. 涡流、电磁阻尼和电磁驱动

### 涡流 (Eddy current)

- 当大块金属处在变化磁场中时，内部感应产生的电流**形成涡旋**——叫**涡流**。
- 涡流流过导体时**发热**，产生焦耳热。

#### 涡流的应用

| 应用 | 原理 |
|---|---|
| 电磁炉 | 高频交变磁场在锅底感应涡流 → 锅底发热 |
| 感应加热 / 熔炉 | 大功率涡流熔化金属 |
| 电磁灶 | 同上 |
| 涡流安检 | 探测金属物体（涡流改变线圈感应电压） |

#### 涡流的危害

- 变压器、电动机铁芯中涡流损耗能量；
- 解决：用**硅钢片叠层** + 涂绝缘漆 → 切断涡流回路。

### 电磁阻尼 (Electromagnetic damping)

- 物体在磁场中运动时由于电磁感应产生涡流，涡流又受到磁场作用力**阻碍运动**——表现为"电磁阻尼"。
- **能量本质**：动能 → 电能（涡流） → 内能（焦耳热）。

#### 应用

- 电流计、电压表表头的**指针快速停下**（避免长时间摆动）；
- 灵敏天平、地磁场探测、列车制动；
- 高铁电磁制动（磁悬浮列车）。

### 电磁驱动 (Electromagnetic drive)

- 与电磁阻尼相反——主动**驱动**导体运动。
- 当外磁场旋转时，闭合导体（金属盘）中感应涡流受磁场力 → 拖着金属盘**跟随旋转**。

#### 应用

- **感应电动机（异步电动机）**：定子产生旋转磁场，转子（鼠笼）跟着转，但略慢于磁场（"异步"）；
- 电度表内的铝盘旋转；
- 高速列车的牵引电机。

### 三者的本质联系

| 现象 | 磁场变化原因 | 感应电流流过的导体 | 效果 |
|---|---|---|---|
| 涡流（发热） | 磁场变 | 大块金属 | 焦耳热 |
| 电磁阻尼 | 导体运动 | 运动导体 | 阻碍运动 |
| 电磁驱动 | 磁场旋转 | 静止/慢转导体 | 驱动旋转 |

### 易错提示

- 涡流的方向也由楞次定律判断。
- 防涡流损耗的关键不是"消除磁场变化"，而是"破坏涡流回路"（叠片）。
- 电磁阻尼与摩擦阻尼区别：电磁阻尼仅在运动时才存在，停止时阻尼也消失——故不能完全制动。

## 4. 互感和自感

### 互感 (Mutual inductance)

- 一个线圈中电流变化 → 周围磁场变化 → 在另一个邻近线圈中产生感应电动势 → 这种现象叫**互感**。
- 两个线圈中产生互感电动势的现象在变压器、感应耦合中应用广泛。

#### 应用：变压器（详见第三章）

- 原线圈交流电产生变化磁场 → 在副线圈中感应交流电动势。

### 自感 (Self-inductance)

- 当**线圈自身**电流变化时，引起穿过线圈的磁通量变化 → 在线圈中**自身**产生感应电动势——这就是**自感**现象。
- 自感电动势：

$$\varepsilon_L = L\dfrac{\Delta I}{\Delta t}$$

（高中只要求定性理解，不需要计算）

- $L$：**自感系数**（电感），单位 **亨利 (H)**；
- $L$ 由线圈结构决定（匝数、形状、是否有铁芯）。

### 自感电动势的方向

- 总是**阻碍自身电流的变化**——这是楞次定律在线圈自身的应用。

### 通断电瞬间的典型现象

#### 接通电源瞬间

- 电流由 0 增大 → 自感电动势**反向**阻碍电流增加 → 电流上升**变慢**；
- 电感线圈中含灯泡时：通电瞬间灯泡**逐渐变亮**。

#### 断开电源瞬间

- 电流由稳定突然变 0 → 自感电动势**同向**维持电流 → 电感中能量通过回路释放；
- 实验现象：开关断开时灯泡突然**变得更亮**而后熄灭（自感电动势可能远大于原电压）；
- 大电感断电时可产生上千伏高压——**危险**！

### 高频应用

| 应用 | 原理 |
|---|---|
| 日光灯启动 | 通电瞬间镇流器（电感）产生高压点亮气体；稳定时限流 |
| 电感性蓄能（不间断电源 UPS） | 储存电能 |
| LC 振荡回路 | 电感 + 电容产生电磁振荡 |
| 滤波器 | 电感对高频电流阻抗大、低频小，用于平滑直流 |
| 继电器、开关跳火 | 断电瞬间自感高压 |

### 危害与防护

- 大电感（如电动机、变压器）断电时火花、电弧 → 需"灭弧装置"或并联消火花电容；
- 仪表、电脑等敏感设备旁需加滤波线路。

### 例题：灯泡与电感线圈

> 电源 + 大电感线圈 L + 灯泡 → 闭合开关时灯泡逐渐变亮；断开开关时灯泡瞬间变亮然后熄灭。
- 闭合瞬间：电流由 0 增加，$\varepsilon_L$ 反向阻碍 → 灯电压小、逐渐变亮；
- 断开瞬间：电流突减，$\varepsilon_L$ 同向维持 → 灯电流可能远大于稳态 → 瞬间更亮；
- 体现了自感"阻碍变化"的本质。

### 易错提示

- 自感是**线圈对自身电流变化的反作用**，与互感（两个线圈间）区分；
- 自感电动势方向"阻碍变化"，不是"阻止变化"——电流仍会变；
- 断电瞬间自感电压可能很高，要注意安全。

## 章末公式速查

```
楞次定律
"阻碍变化"——感应电流方向使其磁场阻碍原磁通量的变化
等价：阻碍引起感应电流的相对运动

法拉第电磁感应定律
ε = N · ΔΦ/Δt          （绝对值；多匝乘 N）

切割磁感线
ε = B L v               （v⊥L，L⊥B 时）
ε = B L v sinα          （v 与 B 夹角 α，且 L⊥两者）

转动线圈（交流电）
ε(t) = N B S ω sin(ωt)
ε₀ = N B S ω           （最大值）

电荷量
q = N ΔΦ/R              （与 Δt 无关）

涡流：变化磁场在大块金属中感应出的涡旋电流
电磁阻尼：运动→涡流→阻碍运动（能量动能→热）
电磁驱动：旋转磁场→涡流→拖动金属盘旋转

自感电动势
ε_L = L ΔI/Δt           （L: 电感，单位 H）
方向：总阻碍电流变化
通电瞬间灯逐亮，断电瞬间可能更亮
```

## 高频易错速查

- [ ] 楞次定律 "阻止变化" 而非 "阻碍变化"
- [ ] 多匝线圈忘了乘 $N$
- [ ] $\varepsilon = BLv$ 中 $L$ 与 $v$ 不垂直
- [ ] $q = N\Delta\Phi/R$ 误以为依赖 $\Delta t$
- [ ] 涡流损耗与涡流应用混淆
- [ ] 自感电动势方向写反
- [ ] 通电瞬间电流立刻达到稳态
- [ ] 断电时认为电流立刻为零（实际有自感维持）
