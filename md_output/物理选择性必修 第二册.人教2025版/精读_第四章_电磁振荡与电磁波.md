# 选必第二册·第四章 电磁振荡与电磁波（精读）

> 主线：LC 回路电磁振荡 → 麦克斯韦电磁场理论 → 赫兹证实电磁波 → 无线电波收发原理 → 电磁波谱及应用。
> 核心方程：$T_\text{LC} = 2\pi\sqrt{LC}$、$c = \lambda f = 3\times 10^8$ m/s。

---

## 1. 电磁振荡

### LC 振荡电路

- 由**电感 $L$** 和 **电容 $C$** 串联组成的闭合回路；
- 是最简单的电磁振荡回路。

### 电磁振荡过程（4 个阶段）

> 假设初始电容已充电（带电 Q₀），电流为 0。

#### 阶段 1：电容放电 → 电感储能
- 电容放电 → 电流增大；
- 电感反向感应电动势阻碍电流增大；
- 电场能 → 磁场能。

#### 阶段 2：电容放完 → 电感最大电流
- 电容电压为 0 → 电流达到最大值；
- 电场能 → 磁场能（达到最大）；
- 电感不让电流立刻停下来（自感作用）。

#### 阶段 3：电感反向给电容充电
- 自感维持电流继续流 → 给电容反向充电；
- 电流减小，电容电压反向增加；
- 磁场能 → 电场能（反向）。

#### 阶段 4：电容反向充满 → 电流再次为 0
- 电容反向充至 -Q₀，电流为 0；
- 全部能量回到电场（反向）。

→ 接下来反向重复 → 形成**周期性振荡**。

### 振荡周期

$$\boxed{\,T = 2\pi\sqrt{LC}\,}$$

(汤姆孙公式)

- $L$：电感（H）
- $C$：电容（F）
- 单位：T 单位为秒；
- **$T$ 与 $L, C$ 都成 $\sqrt{}$ 关系**——$L$ 或 $C$ 翻 4 倍 → $T$ 翻倍。

### 振荡频率

$$f = \dfrac{1}{T} = \dfrac{1}{2\pi\sqrt{LC}}$$

### 能量

- 电场能 $W_E = \tfrac{1}{2}\dfrac{q^2}{C}$（电容上）；
- 磁场能 $W_B = \tfrac{1}{2}Li^2$（电感上）；
- 振荡过程中能量在二者间转换；
- 理想情况下 $W_E + W_B = \text{常量}$（无阻尼振荡）；
- 实际：存在欧姆电阻 → 能量逐渐转为热 → **阻尼振荡**，振幅衰减。

### 电磁振荡与机械振荡的类比

| 力学 | 电磁 |
|---|---|
| 质点位移 $x$ | 电容电荷 $q$ |
| 速度 $v$ | 电流 $i$ |
| 弹性势能 $\tfrac{1}{2}kx^2$ | 电场能 $\tfrac{q^2}{2C}$ |
| 动能 $\tfrac{1}{2}mv^2$ | 磁场能 $\tfrac{1}{2}Li^2$ |
| 弹簧振子周期 $2\pi\sqrt{m/k}$ | LC 周期 $2\pi\sqrt{LC}$ |
| $m \leftrightarrow L$（惯性） | |
| $k \leftrightarrow 1/C$（恢复） | |

### 易错提示

- LC 周期与振幅（初始电荷量）**无关**——简谐性质。
- $L, C$ 越大，振荡越慢。
- 阻尼振荡逐渐衰减——需外加电源补充能量维持稳定振荡。

## 2. 电磁场与电磁波

### 麦克斯韦的统一思想

- 法拉第：**变化的磁场产生电场**（电磁感应）；
- 麦克斯韦：**变化的电场产生磁场**（对称扩展）；
- 因此变化的电场与磁场**相互激发**，形成**统一的电磁场**。

### 电磁波的产生

- 电磁场在空间**自传播** → 形成**电磁波**；
- 麦克斯韦推算其速度等于光速 $c$ → 提出"光是电磁波"；
- 1888 年**赫兹**实验证实电磁波的存在。

### 电磁波的性质

1. **横波**：电场方向 $\vec E$、磁场方向 $\vec B$、传播方向 $\vec c$ 两两垂直；
2. **不需介质**：可在真空中传播（与机械波本质区别）；
3. **真空中速度** $c = 3 \times 10^8$ m/s（基本物理常量）；
4. 介质中速度 $v = c/n$（$n$ 为介质折射率）；
5. **波速、波长、频率关系**：

$$\boxed{\,c = \lambda f\,}$$

6. **携带能量**——能量正比于振幅平方和频率平方；
7. 与一切电磁现象一致（干涉、衍射、偏振）；
8. **传递信息**——无线电、电视、移动通信、雷达。

### 物质性

- 电磁场是**物质存在的一种形式**（与实物粒子并列）；
- 微波炉：电磁波给水分子能量 → 加热；
- 收音机：电磁波在天线中感应电流 → 接收信号；
- 太阳辐射：电磁波传递能量给地球。

### 易错提示

- 电磁波**不需要介质**（区别于机械波）；
- 真空中任何频率的电磁波速度都是 $c$；
- 介质中速度变慢，频率不变，波长变短；
- $\vec E$ 与 $\vec B$ 同相、同频，不是相位差 90°。

## 3. 无线电波的发射和接收

### 发射

#### 步骤
1. **产生高频振荡电流**（如 LC 振荡）；
2. **调制**：把音频 / 视频信号加到载波上；
3. **送入天线**：天线中的电流变化激发电磁波；
4. **天线辐射**：高频振荡产生大幅度交变电磁场，向空间辐射。

#### 调制方式

| 方式 | 原理 | 应用 |
|---|---|---|
| **调幅 (AM)** | 载波**振幅**随信号变化 | 中波广播（AM 收音机） |
| **调频 (FM)** | 载波**频率**随信号变化 | FM 收音机、电视伴音 |
| **调相 (PM)** | 载波**相位**随信号变化 | 数字通信、卫星 |

### 接收

#### 步骤
1. **天线感应**：电磁波在天线中感应电流（含多种频率信号）；
2. **选频**（调谐）：用 LC 回路使其固有频率与所需电台频率匹配 → 通过共振只接收该电台信号；
3. **检波**（解调）：把高频载波"剥掉"，还原出音频信号；
4. **放大** + **扬声器**还原为声音。

### 调谐：LC 共振原理

- 接收回路的 LC 振荡频率 $f_0 = 1/(2\pi\sqrt{LC})$；
- 调节 $C$ 或 $L$ → 改变 $f_0$ → 与某电台载波频率共振；
- 实现"选台"。

### 检波（解调）原理

- 高频调幅波经**二极管整流** + **电容滤波** → 留下低频音频信号。

### 天线作用

- 发射天线：把高频电流转化为电磁波；
- 接收天线：把电磁波转化为高频电流；
- 天线长度与波长有关，一般 $L \sim \lambda/4$ 或 $\lambda/2$。

### 易错提示

- 调谐用的是 **LC 共振**——共振频率 = 载波频率才能高效接收；
- 调幅 / 调频是把"信号信息"加到载波上的不同方式；
- 检波是"取下信号"，与调制相反。

## 4. 电磁波谱

### 全谱概览（频率 / 能量由低到高）

| 波段 | 频率范围 (Hz) | 波长 | 产生机制 | 主要应用 |
|---|---|---|---|---|
| **无线电波** | $10^4$–$10^{11}$ | mm–km | LC 振荡电路 | 广播、电视、移动通信、雷达、卫星 |
| **微波** | $10^9$–$10^{11}$ | mm–dm | 磁控管、速调管 | 雷达、微波炉、卫星通信、5G |
| **红外线** | $10^{11}$–$4\times10^{14}$ | 760 nm–mm | 热辐射、原子振动 | 加热、遥控、夜视、热成像 |
| **可见光** | $4\times10^{14}$–$8\times10^{14}$ | 400 nm–760 nm | 原子外层电子跃迁 | 照明、视觉、光合作用 |
| **紫外线** | $8\times10^{14}$–$10^{17}$ | 1 nm–400 nm | 原子外层电子跃迁 | 消毒、荧光、化学反应 |
| **X 射线** | $10^{17}$–$10^{20}$ | 0.01–10 nm | 内层电子跃迁 / 高速电子撞击 | 医学影像、晶体衍射 |
| **γ 射线** | $> 10^{19}$ | $< 0.1$ nm | 原子核能级跃迁 | 放射治疗、核反应 |

### 各波段的特性与应用

#### 无线电波
- 长波：传播距离远，绕射能力强，用于远程通信；
- 中波 / 短波：电离层反射，远距离广播；
- 微波：定向性好，频率高带宽大，用于现代通信。

#### 红外线
- 热效应显著（"热射线"）；
- 容易被分子吸收 → 加热；
- 遥控器、热成像、红外通信、夜视。

#### 可见光
- 人眼可见的极窄波段（400–760 nm）；
- 颜色与波长对应：红（700 nm）→ 紫（400 nm）。

#### 紫外线
- 强烈化学作用、生理作用；
- 杀菌消毒、医疗（治疗皮肤病）、防伪标记；
- 过量伤害皮肤（晒伤）和眼睛。

#### X 射线（伦琴射线）
- 穿透力强（不同物质吸收差异大）；
- **医学影像**（X 光片）、**晶体结构分析**（劳厄、布拉格父子）；
- DNA 双螺旋结构由 X 射线衍射图像推出（沃森、克里克）。

#### γ 射线
- 极高能量电磁波；
- 来自原子核衰变；
- **放射治疗**（杀死肿瘤细胞）；
- 工业探伤（探测金属内部缺陷）。

### 电磁波识别口诀

> **无 → 微 → 红 → 可 → 紫 → X → γ**
>
> **波长 → 越来越短**
> **频率 → 越来越高**
> **能量 → 越来越大**
> **穿透力 → 总体越来越强**

### 太阳辐射光谱

- 太阳辐射 ≈ 黑体辐射（温度 5777 K）；
- 大部分能量集中在**可见光 + 红外**；
- 紫外线虽少但能量高，对地球生物影响大；
- 大气层吸收掉大部分 X 射线、γ 射线和部分紫外，保护地球生命。

### 电磁波的能量

- 同一波段：振幅越大、能量越大；
- 不同波段：频率越高、单光子能量越大（$E = h\nu$，量子论）；
- 这就是为什么 γ 射线伤害大、红外伤害小。

### 例题：微波频率与波长

> 微波炉频率 $f = 2450$ MHz。
- $\lambda = c/f = 3\times10^8 / 2.45\times10^9 = 0.122$ m = 12.2 cm；
- 这个波长与水分子转动共振频率匹配 → 高效加热。

### 易错提示

- 电磁波谱按**频率**或**波长**排序，注意方向（频率高 ↔ 波长短）；
- 紫外、X、γ 都是"高能"电磁波，区别在产生机制和频率范围；
- 微波是**无线电波**的一部分（高频段），不是独立波段；
- 颜色与波长对应——记住红色波长最长、紫色最短。
