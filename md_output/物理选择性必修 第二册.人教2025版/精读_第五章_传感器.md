# 选必第二册·第五章 传感器（精读）

> 主线：传感器的定义与共性 → 各类典型传感器的工作原理 → 利用传感器搭建简单自动控制电路。
> 核心思想：非电学量 → 电学量（电阻、电压、电流变化）→ 后续电路读取处理。

---

## 1. 认识传感器

### 传感器的定义

> **传感器 (sensor)**：能感受外界信息（如光、热、压力、湿度、化学物质等）并把这些信息按一定规律转换成可用电学信号（电阻、电压、电流、电容等）的元件或装置。

### 核心功能

非电学量 → 电学量

| 输入物理量 | 输出 |
|---|---|
| 光 | 电阻、电流 |
| 力 / 压力 | 电阻、电压 |
| 温度 | 电阻、电压 |
| 磁场 | 电压（霍尔效应） |
| 湿度 | 电阻 |
| 化学物质 | 电流、电势 |
| 声 | 电压（振膜→电信号） |

### 传感器的组成

- **敏感元件**：直接感受被测物理量的部分；
- **转换元件**：把感受到的量转换为电学量；
- **信号调理 / 输出电路**：进一步处理（放大、滤波、A/D 转换）。

> 简单传感器（如热敏电阻）敏感元件 = 转换元件，本身就是一个变阻器。

### 传感器在生活中的应用

- 智能手机：陀螺仪、加速度计、指纹、人脸、距离、光线；
- 智能家居：温湿度、烟雾、PIR 人体红外、燃气；
- 工业：压力、流量、位移、振动；
- 汽车：氧传感器、ABS 轮速、安全气囊加速度计；
- 医疗：心电、脉搏、血氧、血糖。

### 易错提示

- 传感器不是"测量仪表"——它输出的是电学信号，需后续电路放大显示。
- 一种物理量可以由多种传感器测量（如温度可用热敏电阻、热电偶、红外测温）。
- 同一个传感器输出对应输入的关系称为"特性曲线"，可能线性也可能非线性。

## 2. 常见传感器的工作原理及应用

### A. 光敏电阻 (LDR / 光电导)

#### 原理
- 半导体材料（如硫化镉 CdS）在**光照下电导率改变**——光子激发电子从价带跃迁到导带；
- 光照增强 → 自由电荷增多 → 电阻减小。

#### 特性
- 无光（黑暗）：电阻极大（兆欧级）；
- 强光：电阻很小（千欧级以下）；
- 响应速度毫秒级，可见光波段灵敏。

#### 应用
- 路灯自动开关（天黑亮、天亮灭）；
- 摄像机自动曝光；
- 自动门人体感应（红外光敏）；
- 烟雾报警器（光路被烟雾遮挡）。

### B. 热敏电阻 (Thermistor)

#### 原理
- 半导体的电阻**对温度极其敏感**。

#### 两类
| 类型 | 温度系数 | 电阻 vs 温度 |
|---|---|---|
| **NTC**（负温度系数） | 负 | 温度升高→电阻**减小** |
| **PTC**（正温度系数） | 正 | 温度升高→电阻**增大**；超过居里点剧烈增大 |

#### 应用
- 温度测量与控制（电饭锅、电热水器、空调）；
- 过流保护（PTC 保险丝）；
- 电池温度监控；
- 体温计（电子）。

### C. 金属热电阻 (RTD)

- 金属（多用铂 Pt100）的电阻**线性**随温度升高而增大；
- 比热敏电阻线性度好、稳定，常用于工业精密测温。

### D. 热电偶 (Thermocouple)

#### 原理
- 两种不同金属导线连成闭合回路；
- 两接点温度不同时，回路中产生**温差电动势**（塞贝克效应）。

#### 特性
- 温差电动势随温差**近似线性**增加；
- 测温范围广（部分类型 −200 ℃ ~ 2300 ℃）。

#### 应用
- 高温炉测温；
- 火电厂蒸汽温度测量；
- 火箭发动机喷口温度。

### E. 霍尔传感器 (Hall sensor)

#### 原理
- 通电导体板在磁场中运动 → 洛伦兹力使电荷在板的两侧积累 → 产生**横向电势差**（霍尔电压）；

$$U_\text{H} = \dfrac{BI}{nqd}$$

- $n$：载流子浓度，$d$：板厚。

#### 应用
- 测量磁场（高斯计）；
- 无接触电流测量（钳形电流表）；
- 转速测量（齿轮 + 霍尔）；
- 汽车点火、自动门、键盘按键。

### F. 压敏电阻 / 压力传感器

#### 原理 1：电阻式
- 压力使电阻形变 → 电阻改变（如金属应变片：$R$ 随 $L$、$S$ 改变）；
- $R = \rho L/S$，受力时 $L$、$S$ 变化 → $R$ 变化；
- 常做成"惠斯通电桥"提高灵敏度。

#### 原理 2：电容式
- 压力改变两极板间距 → 电容改变。

#### 原理 3：压电式
- 某些晶体（石英、压电陶瓷）在压力下产生**电压**（压电效应）；
- 输出电压与压力成正比。

#### 应用
- 电子秤、车胎压力表；
- 麦克风（声压 → 电信号）；
- 加速度计（质量受压电晶体作用）；
- 触屏（电容式触屏）。

### G. 电容式湿度传感器

- 湿敏材料吸水 → 介电常数变化 → 电容变化；
- 应用于温湿度计、智能家居。

### H. 红外热释电传感器 (PIR)

- 检测人体或动物释放的红外辐射变化；
- 应用：自动门、防盗报警、智能开关。

### I. 化学传感器

- **气敏电阻**（半导体氧化物）：感应气体浓度（CO、CO₂、酒精）；
- **离子选择电极**：感应特定离子浓度（pH 计）；
- 应用：燃气报警器、酒驾检测仪、水质监测。

### J. 光电管 (Photoelectric tube)

- 利用光电效应：光照阴极→发射电子→电路有电流；
- 应用：摄影曝光表、自动售货机（光电门）、电影放映机。

### K. 加速度传感器 / 陀螺仪

- 利用微机电系统（MEMS）：质量块受加速度作用产生位移 → 转换为电容 / 电阻变化；
- 应用：手机翻转、自动驾驶、运动手环、安全气囊。

### 易错提示

- NTC 和 PTC 方向相反，画 R-T 图要注意。
- 霍尔电压方向用左手定则（按正电荷方向）。
- 光敏电阻对**不同波长光**灵敏度不同。
- 选传感器时要考虑：量程、精度、响应速度、稳定性、抗干扰。

## 3. 利用传感器制作简单的自动控制装置

### 自动控制系统的基本结构

```
传感器 → 信号处理 / 比较器 → 控制器（开关 / 继电器）→ 执行器（灯、马达）
```

- **传感器**：感知输入；
- **比较器**：与"阈值"比较；
- **控制器**：根据比较结果切换；
- **执行器**：实施动作（点灯、开门）。

### 经典电路：光控开关

#### 元件
- 光敏电阻 R_LDR
- 定值电阻 R₁
- 比较器或晶体管开关
- 继电器 / 灯泡 / 电源

#### 工作流程
1. 白天：R_LDR 小 → 分压电路输出低电压 → 比较器使灯熄灭；
2. 夜晚：R_LDR 大 → 输出高电压 → 比较器使灯亮。

### 经典电路：温控开关

#### 元件
- 热敏电阻 R_T（NTC）
- 桥式电路（惠斯通电桥）
- 比较器
- 继电器 + 加热器 / 冷却器

#### 工作流程
- 温度低于设定值：R_T 大 → 电桥失衡 → 比较器输出 → 启动加热；
- 温度达到设定值：R_T 减小到平衡值 → 关掉加热。
- 类似原理用于冰箱、空调、电饭煲。

### 简单自动浇灌系统（湿度控制）

- 土壤湿度传感器（电极法测电导）；
- 干燥 → 输出信号变化 → 开水泵；
- 湿润 → 关水泵；
- 实际产品中含**继电器**用于切换大电流。

### 模数转换 (ADC) 与单片机

- 现代传感器多接入**单片机 / 嵌入式系统**：
  1. 传感器输出模拟信号；
  2. ADC 转换为数字信号；
  3. 单片机处理并输出控制信号；
  4. 通过功率驱动器控制执行器。
- Arduino / STM32 / 树莓派 / 51 单片机都可作为控制核心。

### 反馈与闭环控制

- 开环：不监测输出（如定时灯）；
- 闭环：用传感器监测输出并反馈（如恒温器持续监测室温）；
- 闭环可实现**精准、稳定**的自动控制。

### 例题：用热敏电阻做温控

> 设温度 $T$ 时 NTC 阻值 $R_T = R_0/(1 + \alpha T)$（简化模型），与定值 $R$ 串联分压。
- $T$ ↑ → $R_T$ ↓ → $U_{R_T}$ ↓ → 比较器输出反转 → 关掉加热。

### STSE 视角

- 智能家居：温度 + 光 + 湿度 + 烟雾 + 气体 + 摄像头 → 综合控制；
- 智慧农业：自动灌溉 + 自动施肥 + 大棚温控 + 远程监控；
- 工业 4.0：传感器网络 + 物联网 + 大数据 + AI 控制；
- 智能驾驶：激光雷达 + 摄像头 + 毫米波雷达 + GPS + IMU → 自动驾驶。

### 易错提示

- 自动控制系统需要"传感器 + 比较 + 执行"完整环节。
- 闭环控制比开环更稳定但更复杂。
- 阈值设定要避免**频繁切换**（如恒温器加滞回带）。
- 大电流执行器（如加热器）需要**继电器**或**功率管**驱动——比较器输出功率不够。

## 章末公式速查

```
传感器：非电学量 → 电学量

光敏电阻：光↑→R↓（CdS 等半导体）
热敏 NTC：T↑→R↓
热敏 PTC：T↑→R↑（保护用）
热电偶：温差→电动势
霍尔传感器：U_H = BI/(nqd) — 测磁场/电流
压敏电阻：力→形变→R 变化
压电效应：力→电压（石英）
光电管：光→电子→电流

自动控制：传感器 → 比较器 → 控制器（继电器）→ 执行器
闭环：用传感器反馈监测输出
```

## 高频易错速查

- [ ] NTC 与 PTC 方向搞反
- [ ] 光敏电阻"光强 → 电阻增大"（错，应减小）
- [ ] 霍尔电压方向不用左手定则
- [ ] 压电传感器输出与"压力大小"对应（实际多与"压力变化率"对应）
- [ ] 把比较器直接驱动大功率负载（需继电器）
- [ ] 自动控制系统漏掉反馈环节
