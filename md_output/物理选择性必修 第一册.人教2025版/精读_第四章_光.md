# 选必第一册·第四章 光（精读）

> 主线：光的折射定律 + 折射率 → 全反射与光纤 → 光的干涉（双缝）→ 测量光的波长 → 光的衍射 → 偏振与激光。
> 核心方程：$n_{12} = \sin\theta_1/\sin\theta_2$、$n = c/v$、$\sin C = 1/n$、双缝 $\Delta x = \lambda l/d$。

---

## 1. 光的折射

### 反射与折射的伴生现象

- 光从一种介质射到与另一种介质的界面时：
  - 一部分返回原介质 → **反射**；
  - 一部分进入第二种介质 → **折射**。

### 折射定律（斯涅耳，1621）

$$\boxed{\,\dfrac{\sin\theta_1}{\sin\theta_2} = n_{12}\,}$$

- $\theta_1$：入射角（与法线夹角）；
- $\theta_2$：折射角（与法线夹角）；
- $n_{12}$：第 1 种介质对第 2 种介质的相对折射率。
- 入射光线、折射光线、法线**共面**；折射线与入射线分居法线**两侧**。

### 光路可逆性

- 折射现象中**光路可逆**：若光线沿 OA→OB 折射，反向 BO→OA 也满足同一关系。
- 反射同样可逆。

### 绝对折射率

> 光从**真空**射入某种介质，入射角正弦与折射角正弦之比叫做该介质的**绝对折射率**（简称折射率）。

$$\boxed{\,n = \dfrac{\sin\theta_1}{\sin\theta_2}\,}$$

- $n_\text{真空} = 1$，$n_\text{空气} \approx 1.000\,28 \approx 1$（实际计算可视真空）。
- 任何介质的 $n > 1$。

### 折射率与光速的关系

$$\boxed{\,n = \dfrac{c}{v}\,}$$

- $c = 3 \times 10^8$ m/s（真空中光速）；
- $v$：光在该介质中的传播速度；
- $n$ 越大，介质中光速越小。
- 因 $v < c$ → 所有介质 $n > 1$ → 光从真空射入介质时折射角小于入射角。

### 几种介质的折射率

| 介质 | $n$ |
|---|---|
| 空气 | 1.000 28 |
| 水 | 1.33 |
| 酒精 | 1.36 |
| 玻璃 | 1.5–1.8 |
| 水晶 | 1.55 |
| 氯化钠 | 1.54 |
| 二硫化碳 | 1.63 |
| 金刚石 | 2.42 |

### 实验：测量玻璃的折射率

#### 原理
- 利用大头针定位光路：在玻璃砖两侧分别插 2 个大头针，让后插的针挡住前插的针 → 4 个针确定一条折射光路。
- 在白纸上画下入射线、折射线、法线，量出 $\theta_1$ 和 $\theta_2$ → 计算 $n$。

#### 关键操作
1. 玻璃砖两个面平行；
2. 大头针要垂直插入纸面、间距适当（太近误差大）；
3. 入射角不要太小（小角度时测量误差放大）；
4. 入射光线 AO 与从玻璃射出的光线 O'D **平行**——可用此性质验证。

#### 公式
$$n = \dfrac{\sin\theta_1}{\sin\theta_2}$$

### 例题：油桶看底

> 桶底直径 = 桶高 = $d$，无油时刚好看到 $B$（直径对边底沿），加油至半高后沿同方向看到 $C$，$CB = d/4$。
- $\angle AON = 45°$
- $\angle CON' = \arctan(CN'/ON') = \arctan(\dfrac{1}{2})$ 等
- 代入算得 $n = \sqrt{2}\cdot\dfrac{2}{\sqrt{5}}\cdot... \approx 1.58$
- $v = c/n \approx 1.9 \times 10^8$ m/s

### 易错提示

- 入射角 / 折射角都是与**法线**的夹角（不是与界面）。
- $n = \sin\theta_1/\sin\theta_2$ 中 $\theta_1$ 在真空（空气）侧，$\theta_2$ 在介质侧。
- 光路可逆要灵活使用——尤其分析观察角度问题。
- 光从光密射入光疏时折射角**大于**入射角，反之则小。

## 2. 全反射

### 光疏与光密介质

- **光密介质**：折射率较大（光在其中速度小）；
- **光疏介质**：折射率较小（光在其中速度大）；
- **相对的**：水对空气是光密，但水对水晶是光疏。

### 临界角

- 光从**光密**射入**光疏**时，折射角 > 入射角。
- 入射角增大到某值，使**折射角达到 90°** → 这个入射角叫作**临界角 $C$**。

$$\boxed{\,\sin C = \dfrac{1}{n}\,}$$

（$n$ 为光密介质的绝对折射率）

### 全反射

> 当入射角 **≥** 临界角时，光线全部反射回原介质，**没有折射光**——叫做**全反射 (total internal reflection)**。

#### 全反射条件（两个必须同时满足）

1. 光从**光密**介质射向**光疏**介质；
2. 入射角 **≥ 临界角**。

### 几种介质的临界角

| 介质 | $n$ | $C$ |
|---|---|---|
| 水 | 1.33 | 48.8° |
| 玻璃 | 1.5–1.8 | 32°–42° |
| 金刚石 | 2.42 | 24.4° |

→ 金刚石的临界角最小 → 入射角范围内绝大多数发生全反射 → 闪闪发光。

### 全反射的应用

#### 全反射棱镜
- 等腰直角玻璃棱镜，光垂直入射于一面 → 在斜面发生全反射（入射角 45° > 临界角约 41°）→ 反射率 ≈ 100%，比镀银平面镜更好。
- 应用：双筒望远镜、潜望镜、单反相机。

#### 光导纤维（光纤）
- **结构**：内芯（折射率大） + 外套（折射率小）；
- 光在内芯沿"锯齿形"传播，**在内/外界面上发生全反射**；
- 实用直径几微米到 100 微米，可弯曲；
- 应用：
  - **光纤通信**：传输容量 = 20 亿路电话 / 1 千万路电视；衰减小、抗干扰、保密性好；
  - **医学内窥镜**：胃镜、肠镜、气管镜；
  - **传像束**：把光纤成束，使两端排列相同 → 一端图像传到另一端。

### 例题：潜水员看到的"圆锥窗口"

> 潜水员在水下，岸上景物的光以 $0–90°$ 入射 → 折射角范围 $0–C$（$C = 48.8°$）。
- 因此岸上所有景物都集中在水下一个**顶角为 $2C \approx 97.6°$** 的倒立圆锥内；
- 圆锥外（折射角 > C 的方向）潜水员看到的是水底反射光（全反射）。

### 易错提示

- 全反射必须"光密→光疏"且"入射角 ≥ 临界角"，缺一不可；
- $\sin C = 1/n$ 中 $n$ 是光密介质的绝对折射率（不是相对折射率）；
- 水中的气泡 / 玻璃中的气泡看起来明亮是因为光从光密（水/玻璃）入光疏（空气）发生全反射，类似一面"小镜子"。

## 3. 光的干涉

### 杨氏双缝干涉实验（1801）

- 单色光 → 单缝（成线光源）→ 双缝 $S_1, S_2$ → 屏；
- 屏上出现**明暗相间的条纹**——证明光是波。

### 干涉条件（同机械波）

1. 频率相同；
2. 振动方向相同；
3. 相位差恒定。

### 双缝（间距 $d$）→ 屏（距离 $l$）的干涉条件

设屏上某点距中央亮纹 $x$：

- **路程差**：$\Delta r = \dfrac{d x}{l}$（小角近似下）
- **亮纹**（加强）：$\Delta r = n\lambda$ → $x = n\dfrac{\lambda l}{d}$
- **暗纹**（减弱）：$\Delta r = (n + \tfrac{1}{2})\lambda$

### 相邻亮纹（或暗纹）间距

$$\boxed{\,\Delta x = \dfrac{\lambda l}{d}\,}$$

- $\lambda$ 越大（红 → 紫），条纹**越疏**；
- $l$ 越大，条纹**越疏**；
- $d$ 越小，条纹**越疏**。

### 不同光的颜色

- 红光 $\lambda \approx 700$ nm，紫光 $\lambda \approx 400$ nm；
- 同一装置：红光条纹**最疏**，紫光**最密**。
- 白光下：中央亮纹为白色，两侧出现彩色条纹（从内到外按光色顺序排列）。

### 薄膜干涉

- 薄膜前后两面反射光叠加 → 干涉。
- 不同厚度处路程差不同 → 出现亮暗条纹。
- 应用：
  - 肥皂膜、油膜呈彩虹色；
  - 单色光 + 平整度检测（劈尖、牛顿环）；
  - 增透 / 增反膜（眼镜、镜头）。

### 例题：肥皂膜中黄光观察

- 食盐撒在酒精灯芯上 → 发出黄色钠光（589 nm）；
- 用肥皂膜反射钠光看灯焰 → 看到**水平的明暗条纹**；
- 不同位置厚度不同 → 路程差不同 → 干涉条纹。

### 光的本性发展史

| 时期 | 主张 | 代表人物 |
|---|---|---|
| 17 世纪 | 微粒说 | 牛顿 |
| 17 世纪 | 波动说 | 惠更斯 |
| 19 世纪初 | 干涉、衍射 → 波动说占优 | 杨、菲涅耳 |
| 19 世纪 60s | 麦克斯韦：光是电磁波 | — |
| 19 世纪末 | 光电效应 → 波动说无法解释 | — |
| 20 世纪初 | 光子说 → **波粒二象性** | 爱因斯坦 |

### 易错提示

- 双缝干涉中相邻**亮纹之间**是暗纹，故 $\Delta x$ 也是相邻暗纹间距；
- 中央亮纹的位置由"路程差 = 0"决定，**与波长无关** → 白光下中央是白色；
- 用 $\Delta x = \lambda l/d$ 反推 $\lambda$ 时必须确认是**相邻条纹**间距。

## 4. 实验：用双缝干涉测光的波长

### 原理

$$\lambda = \dfrac{d}{l} \Delta x$$

- $d$ 已知（双缝距离，由实验台标注）；
- 测量 $l$（双缝到屏距离）与 $\Delta x$（相邻条纹间距）。

### 实验装置

光源 → 滤光片（单色光）→ 单缝 → 双缝 → 遮光筒 → 毛玻璃屏 + 测量头。

### 关键测量

1. **$l$ 的测量**：用刻度尺直接测；
2. **$\Delta x$ 的测量**：通过**测量头**——
   - 把分划板中心刻线对准一条亮纹中心，记下手轮读数；
   - 转动手轮使刻线对准另一条亮纹中心，再记读数；
   - 两读数之差 = 间距；
   - **多测 $n$ 条亮纹间总距离 $a$**，$\Delta x = a/(n-1)$ → 减小相对误差。

### 注意事项

- 单缝、双缝必须**相互平行**；
- 缝中点尽量在遮光筒轴线上；
- 滤光片置于单缝前确保单色光；
- 不同颜色滤光片：观察条纹间距变化（红光最宽）。

### 数据处理

- 记录 $d, l, a, n$；
- $\Delta x = a/(n-1)$；
- $\lambda = d\Delta x/l$。

### 易错提示

- "$n$ 条亮纹间共 $n-1$ 个间距"——别多算或少算；
- 测量头要旋转方向一致，避免回程误差；
- 改变 $d, l$ 后条纹宽度变化是验证 $\Delta x \propto \lambda l/d$ 的好机会。

## 5. 光的衍射

### 现象

- 缝很宽时：直射，屏上一条与缝宽相当的亮带；
- 缝调窄到一定程度：亮带反而**变宽**且出现**明暗相间**条纹；
- 出现"光绕过缝边缘"传播 → **光的衍射**。

### 衍射明显条件

- 缝宽 / 障碍物尺寸**与波长相当或更小**时；
- 缝越窄 → 衍射越明显，中央亮条纹越宽。

### 白光衍射

- 白光通过单缝衍射 → 中央白色亮纹 + 两侧**彩色**条纹；
- 不同色光衍射条纹位置不同（波长不同）→ 彩色分开。

### 衍射光栅

- 大量等距等宽狭缝排列而成；
- 与单缝相比：条纹**更窄、更亮、更清晰**；
- 应用：光谱仪、激光器、CD/DVD 表面。

### 单缝衍射 vs 双缝干涉

| 项 | 双缝干涉 | 单缝衍射 |
|---|---|---|
| 来源 | 两条狭缝 | 一条狭缝 |
| 条纹间距 | 等间距 | 中央宽，两侧依次变窄 |
| 中央条纹 | 亮纹（最亮） | 亮纹（最宽最亮） |
| 各条纹亮度 | 大致相同 | 中央最亮，两侧迅速衰减 |

### 著名实验

- **泊松亮斑**（1818）：菲涅耳波动理论预测圆盘后影心有亮斑（计算结果），泊松想以此驳波动说，但阿拉果实验证实——反而证明了波动说。
- **劳厄实验**（1912）：X 射线通过晶体出现衍射图样——同时证实了 X 射线波动性 + 晶体周期点阵结构。爱因斯坦称为"物理学中最美的实验"。

### 易错提示

- 衍射 ≠ 干涉，但都是波动现象；
- 即使"光沿直线传播"也是衍射不明显时的近似；
- 缝越窄衍射越明显，但亮度也越低（能量分散）。

## 6. 光的偏振 · 激光

### 偏振

- 横波的振动方向与传播方向**垂直**——可以在垂直传播方向的平面内有不同方位；
- 偏振方向：横波的具体振动方向。
- 纵波**没有偏振**（振动方向 = 传播方向）。

### 用绳波类比

- 绳波通过狭缝：缝方向与振动方向**平行**则能通过、**垂直**则不能通过；
- 这种"挑选振动方向"的特性是偏振的本质。

### 光的偏振

- 光是横波（电磁波）；
- **偏振片**：只允许特定方向（"透振方向"）上的光振动通过；
- **自然光**：包含所有方向振动的等量混合（如太阳光、白炽灯、LED）；
- 自然光通过偏振片 → 透出的光强度减半，且变为**偏振光**（仅含一个振动方向）。

### 两偏振片串联（起偏 + 检偏）

- 偏振片 P 把自然光变偏振光；
- 偏振片 Q 让该偏振光通过：
  - Q 透振方向与 P 平行 → 几乎全透；
  - Q 透振方向与 P 垂直 → 几乎全挡。
- 转动 Q：透射光强度变化 → 证明光是横波。

### 自然光的偏振来源

- 自然光在玻璃、水、桌面等反射时：**反射光 + 折射光都成为部分偏振光**；
- 入射角等于布儒斯特角时，反射光为完全偏振光。

### 偏振的应用

| 应用 | 原理 |
|---|---|
| **3D 电影眼镜** | 两个透振方向垂直的偏振片，让两眼看到两个略有差异的偏振图像 |
| **偏振太阳镜** | 阻挡水面、雪面、玻璃面反射的部分偏振光，减小炫光 |
| **摄影偏振镜** | 减小水面 / 玻璃反射，看清水下物体或橱窗内 |
| **液晶屏幕** | 利用偏振光显示 |
| **应力检测** | 透明材料中应力分布会改变偏振方向 |

### 激光 (Laser)

> 1960 年梅曼研制成功的相干、单色、定向、高亮度光源。

#### 激光的四大特征

1. **相干性高**（频率、相位、振动方向都一致）；
2. **平行度极好**（发散角极小）；
3. **亮度极高**（小面积内集中大量能量）；
4. **单色性极好**（纯色）。

#### 激光的应用

| 应用 | 利用的特性 |
|---|---|
| 光纤通信 | 高相干 + 单色 + 可调制 |
| 测距（地月距离） | 平行度好、定向 |
| 切割、焊接、打孔 | 高亮度、能量集中 |
| 医疗（光刀、视网膜焊接） | 高亮度、可聚焦微小区域 |
| 激光武器、激光制导 | 定向、高功率 |
| 光盘、激光打印 | 单色、可聚焦小光斑 |
| 光谱分析、超精密测量 | 单色、高相干 |
| 原子钟 | 高频率稳定度 |

### 易错提示

- 偏振是横波**特有**现象——能用来判断波的类型。
- 自然光 → 偏振光通过偏振片**强度减半**；偏振光通过两偏振片夹角不同强度按 $\cos^2\theta$ 衰减（马吕斯定律，选学）。
- 激光"亮度高"并非"流明数大"，而是"单位面积单位立体角能量大"。
- 光纤通信里激光既是光源又是信号载体。

## 高频易错速查

- [ ] 入射角 / 折射角取与界面夹角而非法线
- [ ] 全反射条件只记"入射角大"忘了"光密→光疏"

## 选必第一册整册公式总表

```
动量 / 守恒
p = m v
F Δt = Δp
m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'
弹性碰撞：v₁' = (m₁−m₂)/(m₁+m₂)·v₁；v₂' = 2m₁/(m₁+m₂)·v₁
完全非弹性：v' = (m₁v₁+m₂v₂)/(m₁+m₂)
反冲：mΔv = −Δm·u

机械振动
x = A sin(ωt+φ)
ω = 2π/T = 2πf
F = −k x，a = −ω²x
弹簧振子 T = 2π√(m/k)
单摆 T = 2π√(l/g)
g = 4π²l/T²

机械波
v = λf = λ/T
相干：频率、振动方向、相位差相同
干涉加强 Δr = nλ，减弱 Δr = (n+½)λ
多普勒：靠近频率高、远离频率低

光
n = sinθ₁/sinθ₂ = c/v
sin C = 1/n      (全反射临界角)
双缝 Δx = λl/d
干涉 + 衍射 + 偏振 → 光的波动性
```
