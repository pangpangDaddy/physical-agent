# 选必第一册·第三章 机械波（精读）

> 主线：振动在介质中传播形成波 → 横波/纵波 → 波的描述（波形图、波长、频率、波速）→ 波的反射 / 折射 / 衍射 → 波的叠加与干涉 → 多普勒效应。
> 核心方程：$v = \lambda f = \lambda/T$。

---

## 1. 波的形成

### 波动的本质

- 振动在介质中传播——叫做**机械波 (mechanical wave)**。
- 介质中各**质点本身并不随波迁移**，只是依次重复波源的振动。
- 波传递的是**振动的形式**与**能量**（也可携带信息）。

### 形成过程

- 介质 = 由有弹性力相连的"小段质点"组成（绳、弹簧、水、空气）。
- 一个质点振动 → 通过弹性力带动相邻质点 → 一个个依次跟着振动。
- 后面的质点总比前面的**滞后一些**（相位差累积）。
- 整体上呈"凹凸"或"疏密"分布并向前传播。

### 横波 vs 纵波

| 类型 | 质点振动方向 | 例子 | 关键特征 |
|---|---|---|---|
| **横波 (transverse wave)** | **垂直**于波的传播方向 | 绳波、水波（近似）、电磁波 | 波峰、波谷 |
| **纵波 (longitudinal wave)** | **沿**波的传播方向（同一直线） | 声波、弹簧疏密波、地震 P 波 | 密部、疏部 |

### 机械波的传播条件

1. **波源**：使介质中某点振动；
2. **介质**：传递振动的媒介物（无介质则机械波不能传播）。
3. → 故机械波**不能在真空中传播**（与电磁波本质区别）。

### 易错提示

- 介质质点**不随波迁移**——只在各自平衡位置附近振动。
- 横波只能在固体或液体表面传播；纵波在固、液、气中都能传播。
- 同一介质中横波和纵波的速度一般不同（地震 P 波 vs S 波）。

## 2. 波的描述

### 波的图像（波形图）

- 横坐标 $x$：质点在波的传播方向上的**平衡位置**；
- 纵坐标 $y$：某一时刻各质点偏离平衡位置的位移。
- 描绘了"某一时刻"波形（**空间快照**）。

### 简谐波

- 介质中传播的波，若每个质点都做**简谐运动** → 称**简谐波**。
- 简谐波的波形图为正弦曲线。

### 波形图 vs 振动图像（再次强调）

| 项 | 振动图像 (x-t) | 波形图 (y-x) |
|---|---|---|
| 横轴 | 时间 $t$ | 位置 $x$ |
| 描述 | 某一**质点**在不同**时刻**的位移 | 某一**时刻**所有质点的位移 |
| "像" | 录像式 | 照相式 |

### 波长

$$\boxed{\,\lambda = \text{相邻同相点的距离}\,}$$

- 横波：相邻波峰之间或相邻波谷之间的距离；
- 纵波：相邻密部之间或相邻疏部之间的距离；
- 是空间周期。

### 周期与频率

- 介质中各质点的振动**周期 / 频率都相同**，等于**波源**的周期 / 频率。
- 这是波的**时间周期**。
- 一个周期内，波传播的距离恰为一个波长。

### 波速

$$\boxed{\,v = \dfrac{\lambda}{T} = \lambda f\,}$$

- 由**介质性质**决定（与波源无关）。
- 进入不同介质时：$v$ 变化、$f$ 不变（由波源决定）、$\lambda = v/f$ 变化。

### 0℃ 时几种介质中的声速

| 介质 | 声速 (m/s) |
|---|---|
| 空气 | 332 |
| 水 | 1450 |
| 铁 | 4900 |
| 玻璃 | 5000–6000 |
| 软木 | 430–530 |
| 橡胶 | 30–50 |

→ 一般规律：**固体 > 液体 > 气体**（弹性强、密度大利于传播）。

### 例题：判定波传播方向

> 实线为某时刻波形，虚线为 0.5 s 后的波形（$\lambda = 0.24$ m，$T > 0.5$ s）。
- 若**向左**：波形左移 6 cm = $\lambda/4$ → $v = 0.06/0.5 = 0.12$ m/s，$T = \lambda/v = 2$ s。
- 若**向右**：波形右移 18 cm = $3\lambda/4$ → $v = 0.36$ m/s，$T = 0.67$ s。

### 判断质点振动方向的方法

#### 方法 1：上下坡法（横波）
- 沿波传播方向看：
  - 波形向"上坡"的质点 → 正在**向下**振动；
  - 波形向"下坡"的质点 → 正在**向上**振动。
- "同侧法"：与前一个相同状态的质点（沿传播方向后退）方向同。

#### 方法 2：带动法（普适）
- 波传播方向上"前一个"质点带动"后一个"质点。
- 后者的下一刻就是前者的现在。

### 易错提示

- 波长是空间周期，周期是时间周期，二者不能混用。
- 同一波形可能对应多个传播方向——需补充信息（如某质点的振动方向）。
- 进入新介质：$v$ 改变 → $\lambda$ 改变；$f$ 不变。

## 3. 波的反射 · 折射 · 衍射

### 波的反射 (Reflection)

- 波遇到障碍物在分界面**返回原介质**继续传播。
- 反射定律：入射线、反射线、法线在同一平面内；入射线与反射线分居法线两侧；**反射角 = 入射角**。
- 反射过程中：**频率不变、波长不变、波速不变**（同一介质）。

### 波的折射 (Refraction)

- 波从一种介质进入另一种介质时方向改变。
- 折射定律（与光相同）。
- **频率不变、波速变、波长变**。

### 波的衍射 (Diffraction)

> 波**绕过障碍物**或穿过狭缝继续传播。

- **明显衍射条件**：障碍物或狭缝的尺寸**与波长相比拟或更小**。
- 衍射是**波特有的现象**——所有波都能衍射。

### 衍射的判据

| 情况 | 衍射现象 |
|---|---|
| 狭缝 ≫ 波长 | 不明显，近似直线传播（"阴影区"） |
| 狭缝 与 波长相当 | 明显衍射 |
| 狭缝 < 波长 | 衍射极明显，绕到背后 |

### 应用

- 声波波长 1.7 cm – 17 m，与一般障碍物相当 → 我们能"闻其声不见其人"；
- 光波波长极短（纳米级），日常衍射不明显（所以我们说"光沿直线传播"）；
- 蝙蝠超声 (40 kHz 级，波长 8 mm)：能避开毫米级铁丝——避障精度由波长决定。

### 易错提示

- "衍射现象不明显"≠"不能衍射"——所有波都能衍射。
- 反射时入射角是入射线与**法线**夹角，不是与界面夹角。
- 折射时入射角、折射角都是与法线夹角。

## 4. 波的干涉

### 波的叠加

- 两列波相遇时**互相穿过**，各自保持原状态继续传播；
- 在相遇区域，质点位移 = 各列波分别引起位移的**矢量和**。

### 波的干涉 (Interference)

> 满足相干条件的两列波相遇时，某些区域振动**总是加强**，另一些区域**总是减弱**，形成**稳定的干涉图样**。

### 相干条件（必须三个同时满足）

1. **频率相同**；
2. **振动方向相同**；
3. **相位差恒定**（一般也要求初相差固定）。

### 加强与减弱的位置

设两列波到 P 点的路程差 $\Delta r = |r_1 - r_2|$：

- **加强**：$\Delta r = n\lambda$（$n = 0, 1, 2, \dots$），即半波长偶数倍；振幅 = $A_1 + A_2$；
- **减弱**：$\Delta r = (n + \tfrac{1}{2})\lambda$，即半波长奇数倍；振幅 = $|A_1 - A_2|$（若 $A_1 = A_2$ 则该点始终不动）。

> **注**：以上结论的前提是两波源**同相**。若波源反相（相位差 $\pi$），则加强和减弱条件互换。

### 干涉是波特有的现象

- 跟衍射一起，是判断"是否为波"的两大决定性现象。
- 后续光的双缝干涉实验也是相同原理。

### 例题：水波干涉

- 两个同相波源 $S_1, S_2$，在水面上某点：
  - 蓝圆（波峰）与蓝圆相交、黑圆（波谷）与黑圆相交：加强区；
  - 蓝圆与黑圆相交：减弱区；
  - 形成稳定的明暗相间的干涉图样。

### 应用：主动降噪耳机

- 麦克风采集环境噪声 → 处理器生成与噪声**反相**的声波 → 与噪声叠加抵消。
- 利用波的"加强 / 减弱"中的减弱（破坏性干涉）。

### 易错提示

- "加强 / 减弱"指**始终**加强/减弱（稳定的图样），不是某一瞬间。
- 加强区的质点也在振动，只是振幅最大；减弱区的质点振幅最小（甚至不振动）。
- 必须满足相干条件才有稳定干涉；频率不同则相位差随时间变化，没有稳定图样。

## 5. 多普勒效应

### 定义

> 波源与观察者相互靠近或相互远离时，观察者**接收到的频率**与**波源频率**不同的现象。

### 频率变化规律

| 相对运动 | 接收频率 |
|---|---|
| 波源与观察者**相互接近** | **接收频率 > 波源频率**（变高） |
| 波源与观察者**相互远离** | **接收频率 < 波源频率**（变低） |
| 相对静止 | 接收频率 = 波源频率 |

### 直观解释

- 相互接近时：单位时间内有更多波峰到达 → 频率变高；
- 相互远离时：单位时间内到达波峰数减少 → 频率变低；
- 用"队伍迎面走来 / 顺方向同行"类比可帮助理解。

### 波源静止 vs 观察者静止

- 多普勒效应只取决于**相对运动**（波源动还是观察者动均可，但若考虑相对地面有差异——这需要相对论或更细致处理；高中只要求定性理解相对运动）。

### 应用

| 应用 | 原理 |
|---|---|
| 雷达测速 | 雷达发射电磁波，被运动物体反射后频率变化反映物体速度 |
| "彩超"医学影像 | 测血流速度（超声波被血液反射） |
| 测海底洋流速度 | 多普勒声呐 |
| 天文学 | 红移测星体远离速度，蓝移测靠近 |
| 多普勒雷达气象 | 测云团运动 |

### 例题：警车鸣笛

- 警车迎面驶来时：听到音调**变高**；
- 警车远离时：音调**变低**；
- 警车从你身边掠过的瞬间：音调有明显跳变。

### 易错提示

- 多普勒效应中**波源的频率没变**，变的是观察者**接收到**的频率。
- 适用于一切波（声、光、电磁、水波）。
- "光的多普勒效应"叫做"红移 / 蓝移"——天文学中用来测量星体的退行速度。

## 章末公式速查

```
波速公式      v = λ f = λ / T
反射 / 折射 / 衍射 / 干涉是所有波都有的现象
横波 / 纵波 由"质点振动方向 vs 波传播方向"区分

衍射明显条件：障碍物或狭缝与波长相比拟或更小

相干条件：频率相同 + 振动方向相同 + 相位差恒定
加强条件：Δr = n λ           （同相波源）
减弱条件：Δr = (n+½) λ       （同相波源）

多普勒效应
波源与观察者接近 → 频率变高
波源与观察者远离 → 频率变低
```

## 高频易错速查

- [ ] 把"机械波"和"电磁波"混为一谈（机械波需介质，电磁波不需）
- [ ] 同一介质中改变波源频率会改变波速（错，波速由介质决定）
- [ ] 不同介质间传播频率改变（错，频率不变）
- [ ] 衍射 / 干涉是"光特有"现象（错，是波特有）
- [ ] 干涉中"加强"点振幅 > 各源振幅之和（错，至多等于和）
- [ ] 多普勒效应认为波源频率变了（错，是接收频率变）
- [ ] 判断波形上各质点振动方向用了错方法
