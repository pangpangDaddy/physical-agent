# 选必第一册·第二章 机械振动（精读）

> 主线：用弹簧振子建立简谐运动模型 → 描述参数（振幅、周期、频率、相位）→ 回复力与能量 → 单摆作为另一个简谐运动模型 → 用单摆测重力加速度 → 阻尼、受迫振动、共振。
> 核心方程：$x = A\sin(\omega t + \varphi)$、$F = -kx$、$T_{\text{弹簧}} = 2\pi\sqrt{m/k}$、$T_{\text{单摆}} = 2\pi\sqrt{l/g}$。

---

## 1. 简谐运动

### 机械振动

- 物体或物体一部分在某一**平衡位置**附近的**往复**运动。
- 例：钟摆、水面浮标、扁担颤动、树梢摇摆、弹簧上下的小球。

### 弹簧振子（理想化模型）

- 小球套在光滑杆上 + 一端固定的弹簧；
- **假设**：弹簧质量可忽略，空气阻力可忽略；
- 平衡位置：弹簧未形变时小球所在位置（合力为零）。

### 简谐运动定义

> 如果物体的位移随时间的变化遵从**正弦函数**规律（$x$-$t$ 图像为正弦曲线），这样的振动是**简谐运动 (SHM)**。

$$x(t) = A\sin(\omega t + \varphi)$$

- 弹簧振子的运动就是简谐运动（小幅情况下）。

### x-t 图像（振动图像）

- 横轴 $t$，纵轴 $x$（**质点偏离平衡位置的位移**）。
- 频闪照相法 / 高速摄影可记录运动并描点。
- **振动图像 vs 波形图**：
  - 振动图像：某一质点在不同时刻的位移；
  - 波形图：某一时刻不同质点的位移；
  - 两者形状都可能是正弦曲线，但物理意义完全不同。

### 简谐运动几个关键性质

1. 位移 $x$ 是从**平衡位置**指向质点的有向线段（不是路程！）。
2. 经过平衡位置时，$x = 0$，速度最大、加速度为零。
3. 经过最大位移时，$x = \pm A$，速度为零、加速度最大。
4. 周期性运动——任意一个完整周期内位移、速度、加速度都重复。

### 易错提示

- 简谐运动是**振动**，不要写成"匀速"或"匀变速"。
- 位移**不等于路程**：一个周期内位移为 0，但路程为 $4A$。
- 振动方向与传播方向不同（机械波中再用）。

## 2. 简谐运动的描述

### 五个参量

$$x = A\sin(\omega t + \varphi_0)$$

| 量 | 含义 | 单位 | 决定因素 |
|---|---|---|---|
| $A$ | **振幅**：偏离平衡位置的最大距离 | m | 初始能量（与系统无关） |
| $T$ | **周期**：完成一次全振动所需时间 | s | 系统内部性质（与振幅无关） |
| $f$ | **频率**：单位时间完成全振动次数 | Hz | $f = 1/T$ |
| $\omega$ | **圆频率**（角频率） | rad/s | $\omega = 2\pi/T = 2\pi f$ |
| $\varphi_0$ | **初相位** | rad | 计时起点时的状态 |

### 一次全振动

- 从某状态出发回到该状态（位置 + 速度方向都恢复）的过程。
- 一个周期内通过的路程恰为 $4A$，位移为 $0$。
- 半个周期内通过路程为 $2A$，位移为 $-2A \sim +2A$（看起点）。

### 相位

- $(\omega t + \varphi)$ 称为相位，决定物体此刻处于一个周期中的哪个状态。
- 相位差 $\Delta \varphi = \varphi_1 - \varphi_2$ 用于比较两个**同频率**简谐运动的步调。
  - $\Delta\varphi = 0$：同相（同时到达最大、零位）；
  - $\Delta\varphi = \pi$：反相（一个向左另一个向右、最高点对最低点）；
  - $\Delta\varphi = \pi/2$：1 比 2 早 1/4 周期。

### 频率的不变性

- 简谐运动的周期 / 频率**与振幅无关**——这是简谐运动的核心特征。
- 改变释放高度，弹簧振子的周期不变（仅振幅变）。

### 测量周期

- 用秒表测 $n$ 次全振动总时间 $t$，求 $T = t/n$；
- $n$ 越大，相对误差越小（系统误差被均摊）。

### 例题：弹簧振子的位移函数

> 振幅 $A$，平衡位置在 $O$，从最右端 $B$ 处（$x = +A$）开始计时，到最左端 $C$ 处（$x = -A$）用 0.5 s。

- $T = 2 \times 0.5 = 1.0$ s
- 函数：$x = A\sin(2\pi t + \pi/2)$ 或等价的 $x = A\cos(2\pi t)$

### 易错提示

- 振幅 $A$ 是单侧最大距离，**不是**两侧之间的总长（总长为 $2A$）。
- 周期与振幅、相位**都无关**——这是简谐运动的核心。
- 圆频率 $\omega$ 是 $2\pi/T$，不要写成 $2\pi T$。
- 写位移函数时注意确定初相位 $\varphi_0$（看 $t=0$ 时的位移和运动方向）。

## 3. 简谐运动的回复力和能量

### 回复力

- 总是指向**平衡位置**的合力。
- 使物体不断回到平衡位置 + 在惯性带动下穿过 + 又被拉回。

### 简谐运动的回复力特征

$$\boxed{\,F = -kx\,}$$

- 大小与位移成正比；
- 方向始终与位移方向**相反**（指向平衡位置）；
- 这是**简谐运动的判据**——若一个振动满足 $F = -kx$，则它是简谐运动。

### 弹簧振子的回复力

- 由弹簧弹力提供：$F_\text{弹} = -kx$（其中 $k$ 是弹簧劲度系数）。
- → 弹簧振子在弹性限度内做简谐运动。

### 加速度

由牛顿第二定律：

$$a = \dfrac{F}{m} = -\dfrac{k}{m}x = -\omega^2 x$$

- 大小与 $x$ 成正比，方向与 $x$ 相反。
- $\omega = \sqrt{k/m}$，周期 $T = 2\pi/\omega = 2\pi\sqrt{m/k}$。
- 位移最大 → 加速度最大；位移 0 → 加速度 0。

### 简谐运动中各量的变化（重要！）

| 位置 | 位移 $x$ | 速度 $v$ | 加速度 $a$ | 回复力 $F$ | 动能 $E_k$ | 势能 $E_p$ |
|---|---|---|---|---|---|---|
| 平衡位置 $O$ | 0 | 最大 | 0 | 0 | 最大 | 0（弹性势能取参考时） |
| 最大位移处 | $\pm A$ | 0 | 最大 | 最大 | 0 | 最大 |

**判断口诀**：
- 远离平衡位置：$|x|$↑、$|a|$↑、$E_p$↑、$|v|$↓、$E_k$↓
- 靠近平衡位置：以上全反

### 简谐运动的能量

- 弹簧振子：动能 + 弹性势能（水平振动时不考虑重力势能）；
- 任意位置：$E = E_k + E_p = \text{const}$；
- 振幅决定总能量：$E = \tfrac{1}{2}kA^2$（弹簧振子）；
- 简谐运动是机械能守恒的**理想化**模型（忽略阻力）。

### 判断是否简谐运动的两种思路

1. 分析回复力：是否满足 $F = -kx$ 形式？
2. 分析位移函数：是否为正弦/余弦曲线？

### 例题：判断光滑圆弧面上小球的运动

> 半径 $R$ 的光滑碗底，小球被略微推开后释放。

- 偏角 $\theta$ 小时，$x \approx R\theta$，回复力 $F = -mg\sin\theta \approx -mg \cdot x/R = -\dfrac{mg}{R}x$。
- 满足 $F = -kx$ 形式（$k = mg/R$）→ 简谐运动。
- 周期 $T = 2\pi\sqrt{m/k} = 2\pi\sqrt{R/g}$（与单摆周期公式一致）。

### 易错提示

- 回复力是**合力**，不是某一具体力（如弹簧振子中回复力就是弹力；竖直弹簧的回复力是弹力与重力的合力）。
- 平衡位置不一定是"自然长度"位置——比如竖直弹簧悬挂物体的平衡位置在弹簧已被拉长的位置。
- 简谐运动的**机械能**只与振幅有关（与频率、质量等无直接简单关系，但 $\tfrac{1}{2}kA^2$ 中 $k$ 也间接含质量）。

## 4. 单摆

### 单摆（理想化模型）

- 细线长度不变 + 线质量可忽略 + 球的直径 ≪ 线长 + 阻力可忽略；
- 实际中选**质量大、体积小**的球 + **尽量细**的线。

### 单摆的回复力

设摆长 $l$、摆球 $m$、偏角 $\theta$（与竖直方向夹角）：

- 摆球受重力 $mg$ + 摆线拉力 $F_T$。
- 沿圆弧切线分量：$F_切 = -mg\sin\theta$（负号表示指向平衡位置）。
- 摆角较小时 $\sin\theta \approx \theta \approx x/l$：

$$F \approx -\dfrac{mg}{l}x$$

→ 满足 $F = -kx$ 形式，$k = mg/l$ → 单摆做**简谐运动**（小幅前提下）。

### 适用条件

- **偏角 $\theta$ 较小**（一般要求 $\theta < 5°$ 或更严格的 $\theta < 10°$）；
- 摆角大时 $\sin\theta \neq \theta$，回复力不再正比于位移，运动不严格是简谐运动。

### 单摆的周期公式（惠更斯）

$$\boxed{\,T = 2\pi\sqrt{\dfrac{l}{g}}\,}$$

**重要性质**：
- 周期与摆长 $l$、重力加速度 $g$ 有关；
- 周期与**摆球质量 $m$、振幅 $A$**（小角下）都**无关**——这是单摆等时性的体现；
- 摆长 $l$ 是**悬挂点到摆球重心**的距离（线长 + 球半径）。

### 应用

#### 用单摆计时（钟摆）
- 周期为 2 s 的单摆称为"秒摆"，摆长 $l = g/\pi^2 \approx 1$ m。
- 摆钟变慢 → 摆动周期变长 → 缩短摆长（让摆锤往上调）。

#### 在不同星球
- 同一单摆移到月球（$g \approx 1.6$ m/s²）：周期增大 $\sqrt{g_\text{地}/g_\text{月}} \approx 2.5$ 倍。

### 单摆能量

- 重力势能 + 动能；
- 平衡位置（最低点）：动能最大、势能最小；
- 最大位移处：动能为 0、势能最大。

### 易错提示

- 周期公式**只在小角下**才严格成立；大角时是近似。
- 单摆周期与摆球质量无关——这是与弹簧振子（$T = 2\pi\sqrt{m/k}$，质量影响周期）的重要区别。
- 摆长是悬挂点到**球心**的距离（不是到球面）。
- 在加速运动的电梯或非惯性系中，"等效重力加速度" $g'$ 替换 $g$ 使用。

## 5. 实验：用单摆测量重力加速度

### 实验原理

由 $T = 2\pi\sqrt{l/g}$ 反解：

$$\boxed{\,g = \dfrac{4\pi^2 l}{T^2}\,}$$

### 实验装置

- 铁架台 + 细线 + 小球 + 米尺 + 游标卡尺 + 停表（或秒表）。
- 铁架台靠桌边，便于调节摆长。

### 关键操作

1. **摆线**选细且不易伸缩的（如棉线、丝线）；
2. **摆球**选质量大、体积小（如金属球）；
3. **悬挂方式**：用细线一端打小结穿过球孔后悬挂（图甲），比"双线绕摆球"（图乙）更准确——后者实际摆长不易测；
4. **小角度**释放（$<5°$），避免大角误差；
5. **摆球到达平衡位置时**开始计时，并经过 $n$ 次（一般 50 次）取总时间求 $T = t/n$；
6. **摆长测量**：刻度尺测线长 + 卡尺测球直径 → $l = l_\text{线} + r$。

### 数据处理

#### 方法 1：单次求 g 取平均
- 改变摆长，测多组 $l$ 与 $T$，分别算 $g$ 取平均。

#### 方法 2：$l$-$T^2$ 图像法（推荐）
- 由 $T^2 = \dfrac{4\pi^2}{g}l$ → 作 $l$-$T^2$ 图（横 $T^2$、纵 $l$）应为过原点的直线；
- 斜率 $k = g/(4\pi^2)$ → $g = 4\pi^2 k$。
- 优点：自动消除偶然误差（多个数据线性拟合）；可消除**摆球质量分布不均**导致的"摆长不准"系统误差（截距非零反映系统误差）。

### 误差分析

- 偶然误差：计时不准、长度读数；多次测量平均。
- 系统误差：摆球非均匀（重心不在球心）、空气阻力、大角度。

### 例题（数据分析）

> $T^2$-$l$ 图过 $A(0.4, l_1)$、$B(0.8, l_2)$ → 斜率 $k = (l_2-l_1)/0.4$ → $g = 4\pi^2 (l_2-l_1)/0.4$。

### 易错提示

- 计时应"经过平衡位置开始计时"——平衡位置速度最大，过点时间判断误差小。
- 摆长是"悬挂点到球心"，不是"线长"。
- 数大次数减小相对误差，但太多易丢数；建议 30–50 次。

## 6. 受迫振动 · 共振

### 固有振动 & 固有频率

- 振动系统不受外界周期力作用（仅初始扰动后让其自由振动）的振动叫**固有振动**。
- 其频率由**系统自身性质**决定（如弹簧振子的 $f_0 = \dfrac{1}{2\pi}\sqrt{k/m}$，单摆的 $f_0 = \dfrac{1}{2\pi}\sqrt{g/l}$），称**固有频率**。

### 阻尼振动

- 实际振动系统因摩擦、空气阻力、辐射等损失能量，振幅随时间减小。
- 振动图像：包络线衰减的正弦曲线。
- 能量衰减方式：
  1. **内能耗散**：摩擦阻力 → 机械能转为内能；
  2. **波辐射**：振动激起周围介质振动 → 能量随波传走。

### 受迫振动

- 系统在**周期性外力（驱动力）**作用下的振动。
- 驱动力对系统做功补充能量损失 → 使振动能持续。

### 重要结论

> **物体做受迫振动达到稳定后，振动频率等于驱动力的频率**，与物体固有频率无关。

但**振幅**与频率关系密切——见共振。

### 共振

- 当**驱动力频率等于固有频率**时，受迫振动的振幅达到**最大值**——称为**共振**。
- A-f 曲线（共振曲线）：在 $f_0$ 处尖锐峰值，离 $f_0$ 越远振幅越小。

### 共振的应用

| 利用 | 避免 |
|---|---|
| 收音机选台（调谐电路与电磁波共振） | 桥梁倒塌（兵走桥要错开步频） |
| 共振筛分（同步驱动振动） | 飞机、汽车发动机基座共振 |
| 转速计（调整钢片找共振） | 摩天大楼抗风共振（设阻尼器） |
| 微波炉加热（与水分子共振频率匹配） | 高楼地震共振 |
| 核磁共振成像 | 唱歌震碎酒杯 |

### 历史案例：塔科马大桥

- 1940 年塔科马峡湾悬索大桥因风激起扭转**共振**坍塌，建成仅 4 个月。
- 此后土木工程发展抗共振设计：调谐质量阻尼器（如港珠澳大桥）。

### 例题

> 一辆汽车车身-弹簧系统固有周期 $T_0 = 1.5$ s。路面凸起每隔 8 m 一处。

- 驱动力周期 = $8/v$ 等于固有周期 → $v = 8/1.5 \approx 5.33$ m/s 时颠簸最剧烈。

### 易错提示

- 受迫振动稳定后频率 = **驱动力频率**（不是固有频率）。
- 共振 $\neq$ 大振幅一定有破坏——破坏要看振幅与系统耐受度。
- 大振幅可能源于驱动力大或共振；要分别讨论。

## 章末公式速查

```
简谐运动
位移：x = A sin(ω t + φ)
ω = 2π/T = 2π f
回复力：F = −k x
加速度：a = −ω² x
机械能：E = ½ k A²（弹簧振子，水平）

弹簧振子周期：T = 2π √(m/k)
单摆周期：    T = 2π √(l/g)         （小角度）

各位置特征
平衡位置：x=0，|v| 最大，|a|=0，E_k 最大
最大位移：x=±A，|v|=0，|a| 最大，E_p 最大

测重力加速度
g = 4π² l / T²
作 T²−l 图，斜率 = 4π²/g → g = 4π²/斜率

受迫振动 / 共振
受迫稳定频率 = 驱动力频率
共振条件：f_驱 = f_0
```

## 高频易错速查

- [ ] 振幅 $A$ 写成 $2A$（总长度）
- [ ] 简谐周期与振幅有关
- [ ] 单摆周期写错单位或漏 $2\pi$
- [ ] 摆长取线长而非到球心
- [ ] 大角度套用单摆周期公式
- [ ] 振动图像 vs 波形图混淆
- [ ] 受迫振动频率 = 固有频率
- [ ] 共振条件写错
- [ ] 共振一定有害（不一定，看是否破坏）
