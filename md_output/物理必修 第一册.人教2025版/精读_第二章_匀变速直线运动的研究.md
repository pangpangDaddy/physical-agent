# 必修第一册·第二章 匀变速直线运动的研究（精读）

> 主线：通过实验找规律 → 用公式精确描述 → 自由落体作为特殊匀变速运动。
> 核心方程：$v = v_0 + at$、$x = v_0 t + \tfrac{1}{2}at^2$、$v^2 - v_0^2 = 2ax$。

---

## 1. 实验：探究小车速度随时间变化的规律

### 装置与思路

- **器材**：一端带滑轮的长木板、小车、纸带、打点计时器、槽码、刻度尺、低压交流电源。
- **思路**：小车在槽码牵引下加速 → 纸带与小车相连 → 用纸带上各打点间的位移和时间反推**瞬时速度** → 作 $v$-$t$ 图像找规律。

### 操作步骤

1. 把打点计时器固定在长木板无滑轮的一端，纸带穿过限位孔与小车相连。
2. **先接通电源，再放开小车**（顺序错了会丢失起始数据）。
3. 改变槽码数量或在小车上放重物，重复 2–3 次取平均，提高可靠性。
4. 舍掉开头过密的点，选合适计时起点；每 5 个间隔取一个计数点（间隔 0.1 s）。
5. 测各计数点到起点距离 $x$，相邻两计数点的位移 $\Delta x$ 与 $\Delta t$。
6. 用前一节的"短间隔平均速度近似瞬时速度"方法，得各点 $v$。

### 数据处理

- 在坐标纸上以 $t$ 为横轴、$v$ 为纵轴描点。
- 描出的点若大致落在一条**倾斜直线**附近，则用平滑直线"拟合"，让点均匀分布在直线两侧。
- 直线 → 小车做加速度不变的匀变速直线运动；可用 WPS / Excel 选"平滑散点图 + 线性趋势线"自动拟合。

### 易错提示

- 必须先开电源后释放小车。
- 槽码质量需远小于小车质量，才能近似认为拉力等于槽码重力（这是下一章实验"探究 a 与 F、m 关系"的核心约束）。
- 选计数点要避开打点过密区。
- 误差来源：纸带与限位孔的摩擦、打点不稳、测量读数。

## 2. 匀变速直线运动的速度与时间的关系

### 概念

- **匀变速直线运动**：沿一条直线、加速度保持不变的运动。
  - $a > 0$ 与 $v_0$ 同向 → 匀加速；
  - $a < 0$（即 $a$ 与 $v_0$ 反向）→ 匀减速。
- v-t 图像：一条**倾斜的直线**。

### 公式推导

由加速度定义 $a = \dfrac{\Delta v}{\Delta t} = \dfrac{v - v_0}{t}$，得

$$\boxed{\,v = v_0 + at\,}$$

- 物理意义：$at$ 是 $t$ 时间内速度的变化量；末速度 = 初速度 + 变化量。

### 例题（汽车刹车）

> 汽车 $v_0 = 36$ km/h = 10 m/s，加速 $a = 0.6$ m/s²，加速 10 s 后紧急刹车，减速 $a' = -6$ m/s²，求 10 s 末速度 + 停止用时。

- 10 s 末速度：$v = 10 + 0.6 \times 10 = 16$ m/s
- 刹车时间：$t = \dfrac{0 - 16}{-6} = 2.67$ s

> **关键提示**：减速 $a$ 取负，初末速度按一维坐标取正负号。"刹车类"问题要先算理论停车时间，再判断给定时间是否超过——超过则取停止状态而不是反向加速。

## 3. 匀变速直线运动的位移与时间的关系

### 用 v-t 图像求位移（梯形面积法）

- 匀速直线运动：位移 = v · t = 矩形面积。
- 匀变速直线运动：位移 = v-t 图线下的**梯形面积**。

### 位移公式

$$\boxed{\,x = v_0 t + \tfrac{1}{2}at^2\,}$$

由梯形面积 $x = \tfrac{1}{2}(v_0 + v)t$，把 $v = v_0 + at$ 代入即得。

- 初速度为 0 时简化为 $x = \tfrac{1}{2}at^2$。
- 若计时起点不在原点：$x - x_0 = v_0 t + \tfrac{1}{2}at^2$。

### 速度—位移关系（消去 t）

由 $v = v_0 + at$ 与 $x = v_0 t + \tfrac{1}{2}at^2$ 消去 t：

$$\boxed{\,v^2 - v_0^2 = 2ax\,}$$

- 已知量和未知量都不涉及时间时优先选用。
- 例：刹车距离 $x = \dfrac{v_0^2}{2|a|}$。

### 中间时刻速度 / 平均速度

$$\bar v = \dfrac{v_0 + v}{2} = v_{t/2}$$

- 仅适用于**匀变速直线运动**。
- 中间位置速度 $v_{x/2} = \sqrt{\dfrac{v_0^2 + v^2}{2}}$，且 $v_{x/2} > v_{t/2}$（加速运动）。

### 初速度为 0 的匀加速直线运动比例规律

设各连续相等时间间隔 $T$ 内的位移为 $s_1, s_2, s_3, \dots$，连续相等时间末瞬时速度为 $v_1, v_2, v_3, \dots$。

| 比例 | 关系 |
|---|---|
| 前 $T$、$2T$、$3T$... 内的位移 | $x_1:x_2:x_3:\dots = 1:4:9:\dots = 1^2:2^2:3^2:\dots$ |
| 第 1 个、第 2 个、第 3 个... $T$ 内的位移 | $s_1:s_2:s_3:\dots = 1:3:5:\dots = (2n-1)$ |
| $T$、$2T$、$3T$... 末瞬时速度 | $v_1:v_2:v_3:\dots = 1:2:3$ |
| 相邻相等时间间隔内位移之差 | $\Delta s = s_{n+1} - s_n = aT^2$（**逐差法基石**） |
| 通过相等位移所用时间 | $t_1:t_2:t_3:\dots = 1:(\sqrt 2-1):(\sqrt 3-\sqrt 2):\dots$ |

### 逐差法求加速度（纸带处理）

把纸带分成 6 段相等时间间隔 $T$，位移为 $s_1, s_2, \dots, s_6$：

$$a = \dfrac{(s_4 + s_5 + s_6) - (s_1 + s_2 + s_3)}{9T^2}$$

> 比"$a = \dfrac{s_2-s_1}{T^2}$ 取一次"误差小，因为利用了全部数据。

### 易错提示

- 减速运动 $a$ 必须代负号；先判断是否在限定时间内已经停止。
- $x$、$v$、$a$ 都是矢量，建立一维坐标系后用正负号代入计算。
- $v^2 - v_0^2 = 2ax$ 中 $x$ 是位移（带正负），不是路程。

## 4. 自由落体运动

### 历史脉络

- 亚里士多德："重的物体下落得快"（直觉但错）。
- 伽利略：用反证逻辑指出矛盾 → 提出"轻重物体下落同样快"。
- 通过**斜面冲淡重力**实验：测得 $x \propto t^2$，验证 $v \propto t$（匀加速运动）。
- 外推到 90° → 自由落体；并指出空气阻力是日常观察差异的根源。
- 牛顿管演示：抽真空后铁片与羽毛**同时落地**。

### 定义

- **自由落体运动**：物体只在**重力**作用下从**静止**开始下落的运动。
- 严格存在条件：真空。空气阻力可忽略时近似成立。

### 自由落体加速度（重力加速度）$g$

- **方向**：竖直向下（注意：不是垂直地面）。
- **大小**：随纬度变化，赤道 9.780 m/s²，北极 9.832 m/s²，标准值 $g_0 = 9.806\ 65$ m/s²。
- 计算常取 $g = 9.8$ m/s² 或 $g = 10$ m/s²（题目给定时按题目）。

### 公式（$v_0 = 0$，$a = g$ 代入匀变速公式）

| 量 | 公式 |
|---|---|
| 末速度 | $v = gt$ |
| 位移 | $h = \tfrac{1}{2}gt^2$ |
| 速度—位移 | $v^2 = 2gh$ |
| 第 $n$ 秒下落距离 | $h_n = g(n - 0.5)$（第 $n$ 秒内） |

### 实验：用打点计时器测 $g$

1. 重物 + 纸带，纸带穿过竖直放置的打点计时器。
2. 先开电源再松手，让重物自由下落。
3. 选取间距适中的若干点，用逐差法计算加速度即得 $g$。
4. **注意**：因纸带摩擦与空气阻力，测得 $g$ 通常偏小。

### 现代实验：手机加速度传感器

- 让手机自由下落，黄色加速度图线显示 $-10$ m/s²，被手接住时出现向上波峰。

### 易错提示

- "自由落体"严格要求：**只受重力 + 初速为零**。
- 斜抛 / 平抛 / 上抛阶段不是自由落体，但都满足"a = g 竖直向下"。
- 测 $g$ 时纸带要竖直、重物要选稍重的（让阻力相对小）、纸带不能弯折。
- 估算井深一类题，往往要扣除"声音传播时间"才不偏大。

## 章末公式速查（必背）

```
匀变速直线运动（一维，取正方向后代入正负号）
─────────────────────────────────────
v   = v₀ + a t                速度-时间
x   = v₀ t + ½ a t²          位移-时间
v² − v₀² = 2 a x              速度-位移
x̄/t = (v₀ + v)/2 = v_{t/2}    平均速度 = 中间时刻瞬时速度

特例（自由落体，v₀ = 0，a = g 竖直向下）
v = g t,  h = ½ g t²,  v² = 2 g h

初速度为 0 的匀加速运动比例
xₙ : 1 ∶ 4 ∶ 9 ∶ …             (前 nT 位移)
sₙ : 1 ∶ 3 ∶ 5 ∶ …             (第 n 个 T 内位移)
Δs = a T²                      (逐差关系)
```

## 高频易错速查

- [ ] 减速运动公式代入时忘记 $a$ 取负
- [ ] 刹车类问题没判断"是否已停"，导致出现"倒退"假解
- [ ] 把路程当位移代入 $v^2 - v_0^2 = 2ax$
- [ ] 中点位置速度 $v_{x/2}$ 与中点时刻速度 $v_{t/2}$ 混淆
- [ ] 自由落体题忘记 $v_0 = 0$ 的前提
- [ ] 测 $g$ 实验：先松手再开电源
- [ ] 用逐差法只用了 $s_2 - s_1$ 而非全部数据
