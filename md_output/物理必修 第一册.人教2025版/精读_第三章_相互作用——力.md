# 必修第一册·第三章 相互作用——力（精读）

> 主线：把"物体间相互作用"抽象为"力"，研究 3 种典型力（重力、弹力、摩擦力）的产生、大小、方向 → 矢量运算（平行四边形定则）→ 共点力平衡。
> 核心方法：**受力分析 + 正交分解 + 平衡方程**。

---

## 1. 重力与弹力

### 重力

| 项 | 内容 |
|---|---|
| 定义 | 由于地球的吸引而使物体受到的力 |
| 施力物体 | 地球 |
| 大小 | $G = mg$（其中 $g$ 在赤道 9.78、北极 9.83 m/s²） |
| 方向 | **竖直向下**（不是"垂直地面向下"，斜面上仍指向地心方向） |
| 作用点 | **重心** |
| 单位 | 牛顿 N |

**$g$ 的双重单位**：$1\ \text{N/kg} = 1\ \text{m/s}^2$。前者用于"已知质量求重力"，后者用于"已知重力产生的加速度"。

### 重心

- 重力作用线在物体上的等效作用点。
- **质量均匀 + 形状规则**的物体：重心在几何中心（细直棒中点、均匀球的球心、均匀圆柱轴线中点）。
- 不规则物体 → **二次悬挂法**：先在 A 点悬挂，画铅垂线 AB；再换 D 点悬挂，画铅垂线 DE；交点 C 即重心。
- 三角形薄板的重心与几何中心（三条中线交点）重合（前提：均匀等厚）。
- 重心可以在物体**外部**（如空心环）。

### 力的图示与示意图

- **力的图示**：有向线段长度按比例表示力的大小，箭头表方向，箭尾（或箭头）表作用点——精确表达。
- **力的示意图**：只表示方向 + 作用点，不需准确刻度——分析受力时常用。

### 弹力

- **形变**：物体在力的作用下形状或体积发生改变。微小形变可借助"激光反射放大"或"光弹效应"观察。
- **弹性形变**：撤去外力能恢复原状。
- **弹性限度**：超过它后形变不能完全恢复。
- **弹力**：发生形变的物体由于要恢复原状，对与它接触的物体产生的力。
- **产生条件**：① 接触；② 接触处发生弹性形变。
- **方向规律**：
  - 压力 / 支持力：**垂直接触面**，指向被压（被支撑）物体。
  - 绳的拉力：**沿绳指向绳收缩方向**。
  - 杆的弹力：可沿杆，也可不沿杆（活动连接看具体情境）。
  - 弹簧弹力：沿弹簧轴线，与形变方向相反。

### 胡克定律

$$\boxed{\,F = k x\,}$$

- $F$：弹力大小（N）；$x$：弹簧伸长或压缩量（m），**不是弹簧总长度**！
- $k$：劲度系数（N/m），由弹簧本身决定，与拉力或伸长量无关。
- 适用条件：**弹性限度内**。
- "硬"弹簧 → $k$ 大；"软"弹簧 → $k$ 小。

### 实验：探究弹簧弹力与形变量的关系

- 装置：铁架台 + 弹簧 + 钩码 + 刻度尺。
- 记录"弹力 F vs 伸长量 $x = l - l_0$"；作 $F$-$x$ 图像应为**过原点的直线**，斜率即 $k$。
- 易错：用 "总长度 $l$" 而非"形变量 $x$"作图；超过弹性限度后图线偏离直线。

### 易错提示

- 重力 ≠ 万有引力（重力是万有引力扣除随地球自转的离心效应后的合力，只在两极近似相等）。
- 弹力的方向"垂直接触面" → 对于**点—面**接触，沿"过接触点的法线"；对**面—面**沿公共接触面的法线。
- 绳上的弹力只能是**拉力**，不能压；杆可拉可压。
- 同一根轻绳张力处处相等（理想假设）。

## 2. 摩擦力

### 滑动摩擦力

- **产生条件**：① 接触并相互挤压；② 接触面粗糙；③ 有**相对滑动**。
- **方向**：沿接触面，与**相对运动方向**相反。
- **大小**：

$$\boxed{\,F_f = \mu F_N\,}$$

- $\mu$（动摩擦因数）：由接触面材料与粗糙度决定，无单位。
- $F_N$：**正压力**，即两接触面间的弹力大小（不一定等于重力）。

| 接触材料对 | $\mu$ |
|---|---|
| 钢—钢 | 0.25 |
| 木—木 | 0.30 |
| 木—金属 | 0.20 |
| 钢—冰 | 0.02 |
| 木—冰 | 0.03 |
| 橡胶轮胎—干路面 | 0.71 |

### 静摩擦力

- **产生条件**：① 接触并相互挤压；② 接触面粗糙；③ 有**相对运动趋势**但仍相对静止。
- **方向**：沿接触面，与**相对运动趋势方向**相反。
- **大小**：
  - 由其他力的平衡条件决定：$0 < F_s \le F_{\text{max}}$。
  - 不能写成 $\mu F_N$，最大静摩擦力 $F_{\text{max}}$ 略大于 $\mu F_N$，高中常近似 $F_{\text{max}} = \mu F_N$。

### 流体阻力（拓展）

- 方向与物体相对流体的运动方向相反。
- 大小随**速度**与**横截面积**增大而增大。
- 雨滴下落到达"重力 = 阻力"时变为匀速。
- 流线型设计可减小阻力（轮船水下部分、磁浮列车）。

### 易错提示

1. **摩擦力方向 ≠ 物体运动方向**：静摩擦力常常与物体运动方向**同向**（如皮带运输机带着货物向前的就是静摩擦力）。
2. $F_N$ 不一定等于 $mg$：斜面、压力、拉力都会改变 $F_N$。例如斜面上 $F_N = mg\cos\theta$。
3. 动摩擦因数 $\mu = F_f / F_N$，不能反推为 $\mu \propto F_f$、$\mu \propto 1/F_N$——这是定义，与 $F_f, F_N$ 实际取值无关。
4. 滑动摩擦力大小与接触面积无关（只与正压力和动摩擦因数有关）。
5. 静止的物体可能受到滑动摩擦力（如皮带上有静止货物，皮带相对货物向前滑动 → 货物受到向前的滑动摩擦力）——判据是"相对运动"还是"相对运动趋势"。

## 3. 牛顿第三定律

### 作用力与反作用力

- 两个物体间的相互作用：一个施力的同时必然受力。
- **作用力**与**反作用力**互相依赖，同时存在、同时消失。

### 牛顿第三定律

> 两个物体之间的作用力和反作用力，**大小相等、方向相反、作用在同一条直线上**。

**实验验证**：两个相钩的弹簧测力计 / 力传感器，无论拉动还是静止、加速还是减速，两示数始终大小相等、方向相反。

### 与"二力平衡"的区别（重点）

| 维度 | 作用力 / 反作用力 | 一对平衡力 |
|---|---|---|
| 作用对象 | **两个不同物体** | **同一物体** |
| 力的性质 | **必同性质**（同为弹力 / 摩擦力 / 引力） | 可不同性质 |
| 是否同时存在 | 同生同灭 | 一个撤去另一个可独立存在 |
| 合力 | 不可叠加（不在同一物体上） | 合力为零 |
| 大小方向 | 等大反向、共线 | 等大反向、共线 |

**典型反例**：猴子吊在树枝上静止——
- 猴子受重力 $G$ + 树枝拉力 $F$：**一对平衡力**（作用在猴子上，性质不同）。
- 树枝拉猴子 $F$ + 猴子拉树枝 $F'$：**一对作用力反作用力**（作用在不同物体上，都是弹力）。

### 受力分析方法

1. **确定研究对象**——只画这个物体所受的力。
2. 按顺序找力：**重力 → 弹力 → 摩擦力 → 其他外力**。
3. 反作用力不画进研究对象的受力图。
4. 每个力都要写明"谁施加给谁"。

### 应用举例

- 划船：桨向后推水（作用力），水向前推桨（反作用力）。
- 汽车驱动：车轮向后推地面，地面向前推车（驱动力本质是地面给轮胎的摩擦力）。
- 火箭：发动机向下喷气，气体向上推火箭。

### 易错提示

- 重的物体压在轻的物体上，对地面的压力等于"自身重力"——错；要看是否还有其他外力。
- "对地面的压力" ≠ "受到的重力"：压力是作用在地面上的，不是受到的；二者大小可能相等（仅当只受重力与支持力且无加速度）。

## 4. 力的合成和分解

### 合力与分力

- **合力 (resultant force)**：单独作用与几个力的共同作用效果相同的那个力。
- **分力 (component force)**：几个力的共同作用与某个力的单独作用效果相同，这几个力叫该力的分力。
- 合力与分力是**等效替换**关系。

### 平行四边形定则

> 两个力合成时，以表示这两个力的有向线段为邻边作平行四边形，**对角线**就代表合力的大小和方向。

实验验证（橡皮筋等效法）：
1. 用两个测力计 $F_1, F_2$ 把橡皮筋一端拉到 O 点（记下方向和示数）。
2. 改用一个测力计 $F$ 把橡皮筋拉到同一 O 点（效果等价）。
3. 画力的图示，验证 $F$ 是以 $F_1, F_2$ 为邻边作出的平行四边形对角线。

### 合力大小的范围

$|F_1 - F_2| \le F_{\text{合}} \le F_1 + F_2$

| 夹角 $\theta$ | 合力大小 |
|---|---|
| $\theta = 0°$ | $F_1 + F_2$（最大） |
| $\theta = 180°$ | $|F_1 - F_2|$（最小） |
| $\theta = 90°$ | $\sqrt{F_1^2 + F_2^2}$ |
| $\theta$ 任意 | $F = \sqrt{F_1^2 + F_2^2 + 2F_1F_2\cos\theta}$ |

### 多力合成

- 逐次两两合成：$F_1, F_2 \to F_{12}; F_{12}, F_3 \to F_{123}$...
- 共点力前提：所有力**作用线交于一点**。

### 力的分解

- 一个力可以**任意方向**分解 → 不唯一。
- 实际分解要按"产生的实际效果"或"研究问题的需要"。
- 三角形定则：把两个力首尾相接，则合力是从第一个起点到第二个终点的有向线段（等价于平行四边形定则）。

### 正交分解（高频）

把力 $F$ 沿互相垂直的 $x, y$ 轴分解：
$$F_x = F\cos\theta, \quad F_y = F\sin\theta$$

- 坐标轴方向选取：① 平行/垂直于斜面（斜面问题）；② 水平/竖直（一般地面）。
- 选取原则：尽可能让**多数力沿轴**，减少需要分解的力。

### 矢量与标量再认识

| 矢量 | 标量 |
|---|---|
| 位移、速度、加速度、力、动量、电场强度 | 路程、质量、时间、温度、功、电流（高中范围) |
| 平行四边形定则（合成） | 算术法则 |

## 5. 共点力的平衡

### 共点力

- 几个力都作用在物体的**同一点**，或它们的作用线**相交于一点**。

### 平衡状态

- 静止 **或** 匀速直线运动。
- 加速度恒为 0。

### 平衡条件

$$\boxed{\,F_{\text{合}} = 0\,}$$

- 正交分解形式：$\sum F_x = 0,\ \sum F_y = 0$。
- 三力平衡的"任意两力的合力" = "第三个力的等大反向"。

### 三种解题方法

#### 方法一：合成法（适合三力平衡）
取任两力合成，与第三个力等大反向。

#### 方法二：分解法（适合三力以上 / 复杂方向）
把不沿坐标轴的力沿坐标轴分解。

#### 方法三：正交分解法（最通用）
1. 画受力图。
2. 选坐标系（尽量让多数力沿轴）。
3. 把不沿轴的力分解。
4. 列方程：$\sum F_x = 0,\ \sum F_y = 0$。
5. 解联立方程。

### 典型例题分析

#### 例 1：斜面上匀速下滑的物体（滑梯）

- 物体受 $G$、$F_N$、$F_f$。
- 沿斜面方向：$G\sin\theta = F_f = \mu F_N$。
- 垂直斜面方向：$F_N = G\cos\theta$。
- 综合：$\tan\theta = \mu$ → 斜面倾角的正切等于动摩擦因数。

> 推论：物体在斜面上恰好匀速下滑时 $\tan\theta = \mu$；若 $\tan\theta > \mu$ 物体加速下滑；若 $\tan\theta < \mu$ 物体保持静止（前提是初速度为 0）。

#### 例 2：吊灯绳索的拉力

悬绳 AO 与竖直方向成 $\theta$，水平绳 BO 拉住 O 点，重物 $G$ 挂在 O 点下方：
- $F_{AO} = \dfrac{G}{\cos\theta}$
- $F_{BO} = G\tan\theta$

→ **结论**：$\theta$ 越大，两根绳拉力越大；夹角过大可能崩断绳子。

### 动态平衡

> 物体所受某个力的大小或方向缓慢变化，整个过程一直保持平衡。

- **图解法**：把矢量画成三角形，让某一边随 $\theta$ 旋转，观察其他边长度变化。
- **正切函数法**：找出可表达成 $F_1 = G\tan\theta$、$F_2 = G/\cos\theta$ 等关系，按 $\theta$ 变化讨论。

### 易错提示

- 共点力 ≠ 共线力。
- 平衡 ≠ 静止：匀速直线运动也是平衡态。
- 三力平衡时三个力一定共面，且按矢量首尾相接可构成封闭三角形。
- 多力平衡先化简为二力平衡或三力平衡再处理。
- 分解时不要混淆"产生效果方向"与"沿坐标轴方向"——两种分解都对，但意义不同。

## 章末公式与方法速查

```
重力 G = m g                    （g 取 9.8 或 10 m/s²）
胡克 F = k x                    （x 为形变量，弹性限度内）
滑动摩擦 F_f = μ F_N            （F_N 为接触面正压力）
静摩擦  0 < F_s ≤ μ F_N         （由其他力决定）
合力范围 |F₁ − F₂| ≤ F ≤ F₁ + F₂
正交分解 F_x = F cosθ, F_y = F sinθ
平衡条件 ΣF_x = 0, ΣF_y = 0

斜面三角分解
G_x = G sinθ  (沿斜面向下)
G_y = G cosθ  (垂直斜面)
F_N = G cosθ
临界：tanθ = μ  → 恰好匀速下滑
```

## 高频易错速查

- [ ] 把"作用力反作用力"当成"一对平衡力"
- [ ] 误以为 $F_N = mg$（忽略其他力对 $F_N$ 的影响）
- [ ] 静摩擦力用 $\mu F_N$ 求（应由平衡条件求出）
- [ ] 摩擦力方向写成与"运动方向"相反（应为"相对运动 / 相对运动趋势"方向）
- [ ] 弹簧伸长量取成总长度
- [ ] 重心位置默认"在物体内部"
- [ ] 分解力时未画清坐标系
- [ ] 多力合成时漏掉某个力
