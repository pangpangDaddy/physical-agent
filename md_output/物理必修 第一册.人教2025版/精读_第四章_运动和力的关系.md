# 必修第一册·第四章 运动和力的关系（精读）

> 主线：从"力是维持运动的原因"（亚里士多德的错）到"力是改变运动状态的原因"（伽利略 / 牛顿）。
> 三大牛顿运动定律 + 单位制 + 实际应用 + 超重失重。
> 核心方程：$F = ma$。

---

## 1. 牛顿第一定律

### 历史认识演化

| 学者 | 观点 |
|---|---|
| 亚里士多德 | 必须有力推/拉物体，物体才能持续运动；停止用力→停下 |
| 伽利略 | 通过**理想斜面实验**推出：若无摩擦，水平面上小球将永远运动下去 |
| 笛卡儿 | 不受力的物体保持原速度沿原直线运动 |
| 牛顿 | 综合伽利略与笛卡儿，提出第一定律 |

### 伽利略理想斜面实验

1. 两个对接斜面，小球从 A 斜面释放，冲上 B 斜面。
2. 减小 B 斜面倾角，小球仍能到达原高度（但路程变长）。
3. 外推：当 B 倾角 = 0°（水平面），小球将永远运动下去到达原高度（不可能）。

> 结论：力**不是维持运动的原因**，而是**改变运动状态的原因**。
> 这是一个**理想实验**——实际无法做到，但通过逻辑推理把摩擦等次要因素剔除，抓住本质。

### 牛顿第一定律（惯性定律）

> 一切物体总保持**匀速直线运动状态**或**静止状态**，除非作用在它上面的力迫使它改变这种状态。

- **惯性 (inertia)**：物体保持原有运动状态的性质。
- 一切物体都有惯性，惯性是物体的固有属性，与是否受力、运动状态无关。
- 描述惯性大小的物理量是**质量** $m$（标量，单位 kg）。

### 惯性与质量

- 质量越大，惯性越大（越难改变运动状态）。
- 与速度无关：高速行驶的汽车惯性不会"变大"。
- 与运动状态无关：转弯前后惯性不变。
- 与外界无关：是物体固有属性。

### 惯性参考系

- **惯性系**：牛顿第一定律成立的参考系（如地面、太阳、做匀速直线运动的车厢）。
- **非惯性系**：定律不成立（如加速运动、转弯中的车厢）——拉动桌面上的纸时，小球相对纸"自动"向后运动，并非受力，而是参考系本身加速。

### 易错提示

- "运动状态改变" = 速度大小或方向之一改变 → 任一改变都意味着有合外力。
- 不受力 ≠ 匀速直线运动（前者不存在，后者是后者状态的描述）。
- 惯性不是力，没有方向，不能说"被惯性推着"。
- 飞机投弹/抛体：物体离开后水平方向**保持原有速度**——这是惯性的应用，不是力的作用。

## 2. 实验：探究加速度与力、质量的关系

### 控制变量法

> 当某一物理量与多个因素相关时，每次让其中一个变化、其他不变。

- 本实验中：
  - 保持 $m$ 不变，研究 $a$-$F$；
  - 保持 $F$ 不变，研究 $a$-$m$。

### 实验装置

- 长木板（一端垫高补偿摩擦）+ 小车 + 打点计时器 + 纸带 + 槽码 + 滑轮 + 细线。
- 拉力 $F$ ≈ 槽码所受重力（**条件：槽码质量 ≪ 小车质量**）。

### 平衡摩擦力

- 把木板**无滑轮一端**垫高。
- 调节倾角直到小车在**不挂槽码**时能拖动纸带做**匀速直线运动**（打点间距均匀）→ 重力沿斜面分量恰好等于摩擦力。
- 平衡后不要再调整木板倾角。
- **更换小车质量后无需重新平衡摩擦力**（因为 $mg\sin\theta$ 与摩擦力 $\mu mg\cos\theta$ 同时与 $m$ 成正比，$\tan\theta = \mu$ 关系不变）。

### 数据处理

#### a 的测量
1. 用打点纸带 + 逐差法计算加速度；
2. 或用相同时间内两小车位移之比 = 加速度之比（"参考案例 2"）。

#### 结论 1：$a$-$F$ 关系
- $m$ 一定时，$a \propto F$。$a$-$F$ 图像是**过原点的直线**。

#### 结论 2：$a$-$m$ 关系
- $F$ 一定时，$a \propto 1/m$。
- 直接作 $a$-$m$ 图像是**双曲线**，难判断 → 改作 $a$-$\dfrac{1}{m}$ 图像，应为**过原点的直线**才能确证反比关系。

### 易错提示

- 槽码质量必须远小于小车质量（一般 $m_{\text{槽}} < m_{\text{小车}}/10$）；否则 $F \neq m_{\text{槽}} g$。
- 不能仅靠 $a$-$m$ 图像"看起来像双曲线"得结论，必须作 $a$-$1/m$ 验证。
- 平衡摩擦力时不能挂槽码。
- 打点计时器需采用先开电源再放小车的顺序。

## 3. 牛顿第二定律

### 内容

> 物体加速度的大小跟它所受的合力成正比，跟它的质量成反比，加速度的方向跟合力的方向相同。

$$\boxed{\,F_{\text{合}} = ma\,}$$

### 关键性质

1. **同体性**：$F, m, a$ 必须针对**同一物体**。
2. **矢量性**：$F$ 与 $a$ 同方向。
3. **瞬时性**：$F$ 改变的瞬间 $a$ 立即随之改变（与 $v$ 不同——$v$ 是积累的结果）。
4. **独立性**：合力的各个分量分别对应各方向上的加速度分量。
5. **相对性**：定律只在**惯性系**中成立。

### 力的单位"牛顿"的定义

$F = ma$ 中取 $k = 1$，$m = 1$ kg、$a = 1$ m/s² 时所对应的力定义为 **1 N**。

$$1\ \text{N} = 1\ \text{kg·m/s}^2$$

### 例题应用思路

#### 例 1：从受力求运动

> 汽车 $m = 1100$ kg，初速 100 km/h 取消动力，70 s 停下。求阻力 + 重新起步时（牵引 2000 N）的加速度。

- 减速阶段：$a_1 = -v_0/t = -27.8/70 \approx -0.397$ m/s² → $F_{\text{阻}} = ma_1 \approx -437$ N。
- 重新起步：$F_{\text{合}} = 2000 - 437 = 1563$ N → $a_2 = 1563/1100 \approx 1.42$ m/s²。

#### 例 2：用悬挂小球测车厢加速度

> 车厢匀加速启动，车顶悬线偏离竖直方向 $\theta$。
- 悬球受重力 $mg$ + 拉力 $F_T$，合力水平且 $F_{\text{合}} = mg\tan\theta$。
- 加速度 $a = g\tan\theta$（与车厢相同）。

→ 这是动力学测加速度的常用方法（手机加速度计同理）。

### 用动力学方法测质量（拓展）

> 太空失重 → 天平无效。
> 用已知恒力 $F$ 拉物体，测加速度 $a$ → $m = F/a$。
> 应用于神舟系列、天宫课堂演示。

### 易错提示

- $F$ 必须是**合外力**，不是单个力（除非该方向只有一个力）。
- $F_{\text{合}} = 0$ 时 $a = 0$，物体处于平衡态（包括匀速直线运动）。
- 加速度的方向 = 合外力方向，与速度方向无关（可能同向也可能反向）。
- "无论多小的力都能产生加速度" —— 但日常推不动重物是因为有摩擦力抵消，合力为 0。

## 4. 力学单位制

### 基本量与基本单位

| 物理量 | 符号 | 单位 | 单位符号 |
|---|---|---|---|
| 长度 | $l$ | 米 | m |
| 质量 | $m$ | 千克 | kg |
| 时间 | $t$ | 秒 | s |
| 电流 | $I$ | 安培 | A |
| 热力学温度 | $T$ | 开尔文 | K |
| 物质的量 | $n$ | 摩尔 | mol |
| 发光强度 | $I_v$ | 坎德拉 | cd |

### 导出量与导出单位

- 由基本量按物理关系推导出来。
- 例：
  - 速度 m/s
  - 加速度 m/s²
  - 力 1 N = 1 kg·m/s²
  - 功 1 J = 1 N·m = 1 kg·m²/s²
  - 功率 1 W = 1 J/s = 1 kg·m²/s³

### 国际单位制 (SI)

- 1960 年第 11 届国际计量大会制定。
- 现代基于普朗克常量、光速、铯-133 原子跃迁频率等**自然常数**定义。
- 2018 年起 1 kg 不再由"国际千克原器"定义，改由普朗克常量 $h$ 定义：

$$1\ \text{kg} = \dfrac{h}{6.626\,070\,15 \times 10^{-34}\ \text{m}^2\cdot\text{s}^{-1}}$$

### 用单位制检查公式

- 公式两边单位必须相同；
- 例：$V = \tfrac{1}{3}\pi R^3 h$ 单位 m⁴ ≠ 体积单位 m³ → **公式错**（应为 $V = \tfrac{1}{3}\pi R^2 h$）。
- 解题时建议**先用字母推导**到最终表达式，再代入数值；统一国际单位制后，过程中可不写单位。

### 易错提示

- 题中给的"千克"、"克"、"克拉"——代入公式前必须换算为 SI。
- "牛顿每千克"和"米每二次方秒"在重力加速度上等价，但在其他物理量上不能混用。
- $g$ 取 9.8 还是 10 看题目要求。

## 5. 牛顿运动定律的应用

### 两类基本问题

#### 类型 A：已知力 → 求运动

1. 受力分析 → 求合力 → 用 $F = ma$ 求加速度。
2. 用运动学公式 ($v = v_0 + at$, $x = v_0 t + \tfrac{1}{2}at^2$ 等) 求位置、速度。

#### 类型 B：已知运动 → 求力

1. 用运动学公式求加速度 $a$。
2. 受力分析 → 用 $F = ma$ 求未知力（或动摩擦因数等）。

### 例题：冰壶滑行（类型 A）

> $v_0 = 3.4$ m/s，$\mu_1 = 0.02$，求滑行距离。
- $a = -\mu_1 g = -0.2$ m/s²
- $x = \dfrac{-v_0^2}{2a} = \dfrac{3.4^2}{0.4} = 28.9$ m

### 例题：滑雪下坡（类型 B）

> $m = 75$ kg，$v_0 = 2$ m/s，$\theta = 30°$，$t = 5$ s，$x = 60$ m。
- 由 $x = v_0 t + \tfrac{1}{2}at^2$ 得 $a = 4$ m/s²
- 沿斜面：$mg\sin\theta - F_f = ma$ → $F_f = m(g\sin\theta - a) = 75$ N
- 垂直斜面：$F_N = mg\cos\theta \approx 650$ N

### 解题套路

1. **明确研究对象**——单独画受力图，每次只对一个物体。
2. **正交分解**——沿运动方向 / 垂直方向。
3. **列两个方向方程**：沿运动方向 $F_{\text{合}} = ma$，垂直方向 $\sum F = 0$（如果无加速度分量）。
4. **代入数据求解**——注意单位统一与正负号。
5. **复杂过程分段**：每个匀变速段独立分析；前段末速 = 后段初速。

### 易错提示

- 注意"刹车"问题先判停止时刻，再代入。
- 多过程问题分阶段处理：摩擦因数变化、是否打开降落伞等转折点要单独建模。
- 牵引力 ≠ 净力；阻力可能与速度有关（如空气阻力、流体阻力）。

## 6. 超重和失重

### 重力的测量

1. **动力学法**：$G = mg$，先测 $m$（天平）和 $g$（自由落体）。
2. **平衡法**：用测力计悬挂或承托——示数等于重力，**前提是物体静止 / 匀速**。

### 超重和失重的本质

人 / 物体的**实际重力不变**，变化的是**对支持物的压力或对悬挂物的拉力**。

设竖直向下为正方向。

#### 加速向下 / 减速向上（加速度方向向下）

$$mg - F_N = ma \implies F_N = m(g - a) < mg$$

→ **失重**：对支持物的压力小于重力。

#### 加速向上 / 减速向下（加速度方向向上）

$$F_N - mg = ma \implies F_N = m(g + a) > mg$$

→ **超重**：对支持物的压力大于重力。

#### 完全失重

- $a = g$ 向下 → $F_N = 0$。
- 物体对支持物完全无压力。
- 例：自由下落的电梯、绕地球运行的航天器。

### 典型现象

| 情境 | 状态 |
|---|---|
| 火箭发射升空（加速度向上） | 超重，航天员承受数倍体重 |
| 卫星 / 空间站轨道运行 | 完全失重 |
| 蹦极弹性绳拉直前 | 失重（自由落体） |
| 蹦极弹性绳拉直后减速 | 超重 |
| 自由下蹲（下蹲启动阶段） | 失重 |
| 下蹲到最低后开始减速 | 超重 |
| 站起启动阶段 | 超重 |
| 站起接近顶部减速 | 失重 |
| 自由落体水瓶 | 水不会从小孔流出（完全失重） |

### 例题：电梯中体重计示数

> $m = 60$ kg，电梯以 $a = 0.25$ m/s² 匀加速上升，求人对电梯压力。
- $F_N - mg = ma$ → $F_N = m(g + a) = 60 \times 10.05 \approx 603$ N
- 由牛三：人对电梯压力 = 603 N，方向竖直向下。

### 自制"竖直加速度测量仪"

- 轻弹簧上端固定，下端挂质量已知的重物；
- 静止时指针位置标 0，对应 $a = 0$；
- 当电梯加速上升 $a$ 时弹簧拉伸量变化与 $a$ 成正比（$F = m(g+a)$，弹簧 $F = k\Delta x$ → $\Delta x \propto a$）。

### 易错提示

- 超重 / 失重 = 加速度方向的判断，**与速度方向无关**：
  - 加速向上 + 减速向下 → 超重；
  - 加速向下 + 减速向上 → 失重；
- 重力大小没变（除非物体到了不同纬度或天体上）。
- 完全失重 ≠ 没有重力：重力仍然作用，只是对支持物无压力。
- "天宫"实验中水球呈球形是因为表面张力 + 完全失重。

## 章末综合公式速查

```
牛顿三定律
─────────────
N1 (惯性定律)：F_合 = 0  ⇔  匀速直线 / 静止
N2 (牛顿第二定律)：F_合 = m a
N3 (作用力反作用力)：F = −F'，同性质，同时存在，分别作用在两物体上

超重失重判据（设向上为正）
F_N − m g = m a
a 向上 → F_N > m g → 超重
a 向下 → F_N < m g → 失重
a = g 向下 → F_N = 0 → 完全失重

单位关系
─────
1 N = 1 kg·m/s²
1 J = 1 N·m = 1 kg·m²/s²
1 W = 1 J/s
```

## 解题流程模板

```
1. 选研究对象（单个物体或系统）
2. 受力分析（重力→弹力→摩擦力→其他外力）
3. 画受力图（每个力标明大小符号与方向）
4. 选坐标系（按"多数力沿轴"原则）
5. 列方程：
     - 平衡：ΣF_x = 0, ΣF_y = 0
     - 加速：沿运动方向 F_合 = ma；垂直方向 ΣF = 0
6. 代入数据 → 检查正负号 → 写明物理意义
```

## 高频易错速查

- [ ] 把"惯性"当作"力"
- [ ] 用 $\mu mg$ 而不是 $\mu F_N$（忽略斜面、外力使 $F_N$ 变化）
- [ ] 牛顿第二定律的 $F$ 写成单个力而非合力
- [ ] 多过程问题忘记切换初末速
- [ ] 超重 / 失重看速度方向而非加速度方向
- [ ] 完全失重时认为"重力消失"
- [ ] 单位制混用（km、g 没换算成 m、kg）
- [ ] 公式化简时丢失矢量正负号
