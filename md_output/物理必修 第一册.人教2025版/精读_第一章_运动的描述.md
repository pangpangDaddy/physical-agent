# 必修第一册·第一章 运动的描述（精读）

> 教材定位：序言 + 第一章；建立"机械运动 / 质点 / 参考系 / 位移 / 速度 / 加速度"五个核心描述工具。

---

## 序言：物理学——研究物质及其运动规律的科学

- **物理学定义**：定量地研究物质的存在形式、基本性质以及运动和转化规律。它既是实验科学，也是依赖逻辑推理与数学工具的理论科学。
- **发展脉络**：古希腊（亚里士多德、阿基米德）→ 伽利略、牛顿创立系统力学（1687《自然哲学的数学原理》） → 19 世纪经典物理（力、热、电磁）→ 20 世纪相对论 / 量子力学 → 21 世纪标准模型、引力波、量子信息。
- **科学精神三要素**：
  1. 学以致知（求知的探索精神）；
  2. 实事求是（以实验和逻辑为唯一判据）；
  3. 理性分析（环环相扣，没有诡辩者的位置）。

## 1. 质点 · 参考系

### 核心概念

| 概念 | 内涵 | 关键点 |
|---|---|---|
| **机械运动** | 物体的空间位置随时间的变化 | 自然界最基本的运动形态 |
| **质点 (mass point)** | 用来代替物体的、具有质量但没有大小和形状的点 | **理想化模型**：忽略次要因素，突出主要因素 |
| **参考系 (reference frame)** | 用来作为参考的物体 | 运动是绝对的，描述是相对的 |

### 能否看作质点（判据）

> 物体的形状、大小对所研究问题影响可以忽略不计时，才能视为质点。

- 同一物体在不同问题中"是否可视为质点"的答案可以不同：
  - 研究地球公转 → 可视为质点（直径 ≈ 1.3×10⁴ km vs 日地距离 ≈ 1.5×10⁸ km，万分之一）
  - 研究地球自转 → 不可视为质点
  - 研究列车整体行驶 → 可
  - 研究列车上的"香蕉球"旋转 → 不可
- 物体上各点运动情况完全相同（平动）时，整体也可简化为一个质点。

### 参考系选取

- 任意性 + 方便性：通常以地面为参考系；研究天体常以太阳为参考系。
- 提到运动必须说明参考系，否则无意义。
- **运动的相对性**：陈与义《飞花》"卧看满天云不动，不知云与我俱东"——以船为参考系云"不动"；以地面为参考系云"东行"。

### 易错提示

- 体积小≠质点，体积大也未必不是质点；判据始终是"问题情境"。
- "绝对静止"不存在；"绝对运动 + 相对静止"才是描述自然的正确说法。

## 2. 时间 · 位移

### 时刻 vs 时间间隔

- **时刻**：数轴上的"点"，对应"什么时候"——8 时、第 3 s 末。
- **时间间隔**：数轴上的"线段"，对应"用了多久"——45 min、前 3 s、最后 3 s、第 3 s 内。

### 位置与位移

- **位置**：在参考系上建立坐标系（一维、二维、三维），用坐标定量描述。
- **路程 (path)**：标量，运动轨迹的长度。
- **位移 (displacement)**：矢量，由初位置指向末位置的有向线段。
  - 一维记法：$\Delta x = x_2 - x_1$
  - 大小：$|\Delta x|$；方向：指向 x 轴正方向（$\Delta x > 0$）或负方向。

### 矢量 vs 标量

| 矢量 | 标量 |
|---|---|
| 位移、速度、加速度、力 | 路程、温度、时间、质量、功 |
| 既有大小又有方向 | 只有大小 |
| 合成遵循平行四边形定则 | 加减遵循算术法则 |

### x-t 图像（位置/位移—时间图像）

- 横轴 t，纵轴 x。若以初始位置为原点，则 $x = \Delta x$，x-t 即 位移-t 图像。
- 直观读出任意时刻位置、任意区间位移。

### 实验：打点计时器使用

| 仪器 | 工作电压 | 打点 |
|---|---|---|
| **电磁打点计时器** | 4–6 V 低压交流（教材标 8 V） | 振针 + 复写纸打点 |
| **电火花计时器** | 220 V 交流 | 电火花 + 墨粉打点 |

- 电源频率 50 Hz → 每 **0.02 s** 打一个点。
- 操作要点：先**通电再放纸带**；纸带先穿过限位孔；舍弃开头过密的点，选合适计时起点。

### 易错提示

- "前 3 s"是时间间隔，"第 3 s 末"是时刻，"第 3 s 内"是时间间隔（即第 2 s 末到第 3 s 末）。
- 路程 ≥ |位移|，仅在单向直线运动中两者相等。
- 出租车计价的"千米"是路程，不是位移。

## 3. 位置变化快慢的描述——速度

### 平均速度 vs 瞬时速度

- **平均速度** $\bar v = \dfrac{\Delta x}{\Delta t}$——一段过程的"平均快慢与方向"，矢量。
- **瞬时速度** $v = \lim\limits_{\Delta t \to 0}\dfrac{\Delta x}{\Delta t}$——某时刻或某位置的快慢与方向，矢量。
- **速率 (speed)**：瞬时速度的大小，标量。汽车速度计显示的是速率。

### 国际单位制单位

- 速度：m/s（或 km/h、cm/s）
- 换算：$1\ \text{m/s} = 3.6\ \text{km/h}$

### 匀速直线运动

- 瞬时速度保持不变；平均速度与瞬时速度恒等；x-t 图像为一条**斜直线**，斜率 = 速度。

### v-t 图像

- 横轴 t，纵轴 v；斜率 = 加速度；图线下面积 = 位移。
- 与时间轴重合 → 静止；平行 t 轴 → 匀速；倾斜直线 → 匀变速；曲线 → 非匀变速。

### 实验：用打点计时器测平均速度/瞬时速度

- **平均速度**：在纸带上选两点 D、G，$\bar v_{DG} = \dfrac{x_{DG}}{t_{DG}}$。
- **瞬时速度**：用包含 E 点的足够短间隔 DF 的平均速度近似代替 E 点瞬时速度。区间越小越精确，但过小测量误差又增大——需折中。
- **拓展**：超声波位移传感器（红外触发 + 超声波回声）配合计算机自动绘 v-t 图像，时间精度可达 0.02 s。

### 易错提示

- 平均速度 ≠ "速度的平均"。位移 / 时间 是定义，不能写成 $(v_1+v_2)/2$（除匀变速外）。
- 速度方向 = 位移方向。
- v-t 图像与 x-t 图像形状相同但物理意义完全不同：x-t 斜率是速度，v-t 斜率是加速度。

## 4. 速度变化快慢的描述——加速度

### 定义

$$a = \dfrac{\Delta v}{\Delta t} = \dfrac{v - v_0}{t}$$

- **物理意义**：单位时间内速度的变化量；描述速度变化的快慢。
- **单位**：m/s²（米每二次方秒），符号 m·s⁻²。
- **矢量**：方向与 $\Delta v$ 方向相同。

### 直线运动中加速度方向的判断

| 情形 | $a$ 与 $v$ 方向 |
|---|---|
| 加速（速率增大） | 同向 |
| 减速（速率减小） | 反向 |

### v-t 图像中的加速度

- 直线斜率 = $\Delta v / \Delta t$ = 加速度。
- 倾斜程度越大，加速度越大。
- 图线为水平直线（v 恒定）→ $a = 0$；倾斜直线 → 匀变速；曲线 → 加速度随时间变化。

### 数量级感

| 物体 | a / (m·s⁻²) |
|---|---|
| 子弹在枪筒中 | 5×10⁴ |
| 伞兵着陆 | −25 |
| 汽车急刹车 | −5 |
| 赛车起步 | 4.5 |
| 汽车起步 | 2 |
| 高速列车起步 | 0.35 |

### 易错提示（典型反例题）

1. $a = 0$ 时 $v$ 不一定为 0 —— 匀速直线运动。
2. $|\Delta v|$ 大、$a$ 不一定大 —— 还要看 $\Delta t$。
3. $a$ 与 $v$ 方向可以相反 —— 减速运动。
4. $a$ 减小，$v$ 仍可能增大 —— 只要 $a$ 与 $v$ 同向且 $a > 0$，速度就在增加。
5. 速度大不等于加速度大；速度变化大也不等于加速度大。

### 变化率思想

- 任意物理量 $D$ 对时间的变化率：$\dfrac{\Delta D}{\Delta t}$。
- 速度是位置的变化率，加速度是速度的变化率——同一种方法的递推。

## 章末小结

```
                    ┌─ 质点（研究对象的理想模型）
研究运动的基本概念 ─┤
                    └─ 参考系（观察运动的参照物体）

                    ┌─ 位移（位置的变化）
描述运动的物理量 ──┼─ 速度（位置的变化率）
                    └─ 加速度（速度的变化率）
```

### 研究方法
1. 模型化：质点是把物体抽象为有质量的点。
2. 极限思维：用极短时间内的平均速度逼近瞬时速度。
3. 图像法：x-t、v-t 图像；斜率 / 面积分别对应导数 / 积分。
4. 实验工具：打点计时器、传感器 + 计算机。

### 高频易错速查

- [ ] 时刻与时间间隔混淆
- [ ] 位移与路程混淆
- [ ] 用 $(v_1+v_2)/2$ 计算非匀变速的平均速度
- [ ] 把"减速"理解成加速度为零
- [ ] v-t 图像与 x-t 图像看错斜率含义
- [ ] 把"速度大"等同于"加速度大"
