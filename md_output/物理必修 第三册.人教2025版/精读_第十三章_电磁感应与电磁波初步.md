# 必修第三册·第十三章 电磁感应与电磁波初步（精读）

> 主线：电流的磁效应（奥斯特）→ 磁场的描述（磁感应强度 $B$、磁感线、磁通量 $\Phi$）→ 法拉第电磁感应 → 麦克斯韦电磁场理论 + 电磁波 → 能量量子化（普朗克）。
> 核心方程：$B = F/(Il)$、$\Phi = BS$、电磁波 $c = \lambda f$、能量子 $\varepsilon = h\nu$。

---

## 1. 磁场 · 磁感线

### 电与磁的统一

- 1820 年丹麦物理学家**奥斯特**：把通电导线放在小磁针上方时，磁针偏转 → 发现**电流的磁效应**。
- 安培、法拉第等进一步研究：磁体—磁体、磁体—通电导线、通电导线—通电导线之间都有相互作用。
- 这些作用通过**磁场**发生。

### 磁场

- 磁场是一种**客观存在的物质**（与电场并列）。
- 既不依赖人的感觉，又能通过对磁体 / 通电导线的作用力表现出来。
- 不依赖介质传递（与电场相同）。

### 磁感线

- 形象描述磁场的假想曲线；曲线上**任一点的切线方向 = 该点磁场方向**。
- 磁针 N 极所指方向 = 该点磁场方向。
- **性质**：
  1. 磁感线**不相交**；
  2. 磁感线**首尾相连闭合**（与静电场线"从正到负"不同）——磁单极子不存在；
  3. 磁体外部从 N 出发到 S；内部从 S 到 N（首尾相连）；
  4. 疏密表示 $B$ 大小。

### 常见磁感线分布（必记）

| 场源 | 磁感线 |
|---|---|
| 条形磁体 | 从 N 极出发，经外部回到 S 极；两极附近最密 |
| 蹄形磁体 | 两极间近似匀强磁场 |
| 直线电流 | 以导线为圆心的同心圆，平面垂直导线 |
| 环形电流 | 中心处沿轴线方向，与电流方向满足右手定则 |
| 通电螺线管 | **外部类似条形磁体**，内部沿轴方向且较均匀 |

### 安培定则（右手螺旋定则）

#### 直线电流
- 右手握住导线，**伸直拇指指向电流方向**，**四指弯曲方向 = 磁感线环绕方向**。

#### 环形电流 / 螺线管
- 右手四指弯曲指向电流方向，**伸直拇指指向轴线上 N 极方向**（也即磁感线内部方向）。

### 地磁场

- 地球本身是个大磁体；地磁两极与地理两极**不重合**——存在**地磁偏角**。
- 地磁北极位于地理南极附近，地磁南极位于地理北极附近（小磁针 N 指北是因为地理北极附近是地磁南极）。
- 地磁偏角随地点、时间变化。

### 分子电流假说（安培）

- 物质内部存在环形电流（"分子电流"）→ 每个微粒都是小磁体。
- 排列杂乱 → 对外不显磁性；
- 受外磁场后排列趋同 → 显示磁性 → 磁化。
- 高温或剧烈撞击 → 重新打乱 → 退磁。
- 现代观点：分子电流对应于原子内电子的运动。

### 易错提示

- "N 极指北"是因为**地理北极附近是地磁的南极**。
- 螺线管外部磁感线是连续的（不像静电场线"消失"在负电荷）。
- 直线电流的磁感线是同心圆，不是辐射线。
- 用右手定则要分清楚"直线"和"螺线管"两种握法。

## 2. 磁感应强度 · 磁通量

### 磁感应强度 B

- 用通电导线在磁场中的受力定量描述磁场。
- 把**很短一段通电导线 + 电流** $Il$ 看作"电流元"。
- 通电导线与磁场**垂直**时，$F = BIl$ → 由此定义：

$$\boxed{\,B = \dfrac{F}{Il}\,}$$

- **矢量**，方向 = 小磁针静止时 N 极所指方向。
- 单位：特斯拉 **T**，$1\ \text{T} = 1\ \text{N/(A·m)}$。

### 典型数量级

| 场源 | $B$ (T) |
|---|---|
| 人体器官 | $10^{-13}$–$10^{-9}$ |
| 地磁场（地表） | $5\times10^{-5}$ |
| 电动机铁芯 | 0.8–1.7 |
| 核磁共振 | 3 |
| 中子星表面 | $10^6$–$10^8$ |
| 原子核表面 | $\sim 10^{12}$ |

### 匀强磁场

- 各点 $B$ 大小相等、方向相同；
- 磁感线为间距相等的**平行直线**；
- 典型情境：相距很近的两个平行异名磁极间、两个共轴线圈中间。

### 磁通量 Φ

$$\boxed{\,\Phi = BS\,}$$

- $S$：与 $B$ 方向**垂直**的面积；如果 $B$ 与平面法线成 $\alpha$ 角，则用投影面积 $S\cos\alpha$。
- **标量**（虽有"穿过方向"概念），单位 韦伯 **Wb**：$1\ \text{Wb} = 1\ \text{T·m}^2$。
- **物理意义**：穿过某个面的"磁感线总数"（或可看作 $B$ 在该面上的"流量"）。

### 磁感应强度与磁通量的关系

- $B = \Phi / S$ →磁感应强度数值上等于"穿过垂直磁场方向单位面积的磁通量"。
- 一个面的 $\Phi$ 取决于：
  1. $B$ 的大小；
  2. 面积 $S$；
  3. 面与 $B$ 的夹角。

### 易错提示

- $B = F/(Il)$ 是**定义式**，$B$ 由磁场本身决定，与 $F, I, l$ 都无关。
- 求 $F$ 时只有当导线与 $B$ 垂直才能直接用 $F = BIl$；非垂直时 $F = BIl\sin\theta$。
- 磁通量不分正负？严格说 $\Phi$ 是有正负的（依规定法线方向），但中学常作标量处理。
- 一个闭合曲面的总磁通量恒为 0（磁感线连续闭合，进出相等）——这是"无磁单极"的体现。

## 3. 电磁感应现象及应用

### 法拉第的发现

- 法拉第坚信"电与磁有内在联系"，从奥斯特发现后探索了 10 年。
- 1831 年，他把两个线圈绕在同一铁环上 → 一个线圈接电源开关，另一个接电流表 → **通电或断电的瞬间**，另一线圈出现电流。
- 由此得出："磁产生电"必须在**变化、运动**中发生。

### 产生感应电流的条件

> **穿过闭合导体回路的磁通量发生变化时，闭合回路中产生感应电流。**

变化方式可以是：
1. 磁感应强度 $B$ 大小变化（如向线圈中插入磁铁、改变邻近线圈电流大小）；
2. 回路面积 $S$ 变化（如导体棒在磁场中切割磁感线）；
3. 回路与 $B$ 的夹角变化（如线圈转动）；
4. 以上的组合。

### 各种产生感应电流的典型情境

| 操作 | 是否变化 $\Phi$ | 感应电流 |
|---|---|---|
| 线框始终与场平行、上下移动 | $\Phi$ 不变 | 无 |
| 线框始终与场平行、左右移动（边界穿过场区） | 部分线框出/入场 → $\Phi$ 变 | 有 |
| 线框绕轴旋转（与场夹角变） | $\Phi$ 变 | 有 |
| 把磁铁靠近 / 远离闭合线圈 | $B$ 变 | 有 |
| 闭合开关的瞬间（变压器初级） | 次级 $\Phi$ 突变 | 有（瞬时） |
| 稳定后开关闭合 | $\Phi$ 不变 | 无 |
| 断开开关瞬间 | $\Phi$ 突变 | 有 |
| 在匀强磁场中线圈不动 | $\Phi$ 不变 | 无 |

### 切割磁感线（导线穿越场区）

- 是 $S$ 变化的特殊情形。
- 导体棒在匀强磁场中以速度 $v$ 切割磁感线时（导线、$v$、$B$ 互相垂直）：

$$\varepsilon = Blv$$

（**注**：这是选必二的内容；必修三只要求理解切割是产生感应电流的方式之一。）

### 历史意义

- 1831 年圣诞节前夕，法拉第演示"铜盘 + 磁极"——人类第一台**圆盘发电机**。
- 当时有人取笑"婴儿有什么用"，法拉第回答"新生婴儿又有什么用"。
- 后世发电机改变了整个人类社会 → 进入电气时代。

### 应用

- 发电机、变压器、电磁炉、感应加热、磁卡、电磁制动…

### 易错提示

- 感应电流出现的条件是"磁通量**变化**"，不是"磁通量为零或不为零"。
- 即使回路不动，只要 $B$ 在变 → 仍产生感应电流。
- 线圈在均匀磁场中**平动**（与场方向不平行）不会产生感应电流，除非边界穿入穿出。
- "切割磁感线"是 $S$ 变化的一种说法；"$\Phi$ 变化"是更普遍表述。

## 4. 电磁波的发现及应用

### 麦克斯韦的预言

- 法拉第：变化的磁场产生**电场**（电磁感应）。
- 麦克斯韦推广：变化的电场也产生**磁场**。
- 由此得出"电磁场"概念：变化的电场与变化的磁场相互"激发"，形成不可分割的统一体。
- 进一步推断：周期性变化的电磁场在空间传播 → **电磁波**。
- 推算电磁波速度 = 光速 → 提出"光的电磁理论"：光是电磁波的一种。
- 1888 年**赫兹**通过实验证实电磁波存在。

### 电磁波的特性

| 特性 | 说明 |
|---|---|
| 传播 | **不需要介质**，可在真空中传播 |
| 速度 | 真空中 $c = 3 \times 10^8$ m/s |
| 关系 | $c = \lambda f$（波速 = 波长 × 频率） |
| 横波 | 电场、磁场都垂直于传播方向 |
| 能量 | 电磁波传播能量 |

### 电磁波谱（频率/波长由小到大）

| 波段 | 频率范围 | 典型波长 | 应用 |
|---|---|---|---|
| **无线电波**（长波/中波/短波/微波） | 10⁴–10¹¹ Hz | km–mm | 广播、通信、雷达、卫星、微波炉 |
| **红外线** | 10¹¹–10¹⁴ Hz | mm–μm | 加热、热成像、遥控 |
| **可见光** | 4–7.5×10¹⁴ Hz | 400–760 nm | 照明、显示、视觉 |
| **紫外线** | 7.5×10¹⁴–10¹⁷ Hz | nm | 消毒、荧光 |
| **X 射线** | 10¹⁷–10¹⁹ Hz | 0.01 nm 级 | 医学影像、晶体衍射 |
| **γ 射线** | $>10^{19}$ Hz | <pm | 放射治疗、核物理 |

### 电磁波的能量

- 法拉第用"力线"描述电磁场；麦克斯韦用数学方程描述；赫兹证实它是**真实物质**。
- 微波炉：微波使食物中水分子振动加剧 → 内能增加 → 加热。
- 收音机收到电台广播：电磁波在天线中感应电流 → 解调出声音。
- → 电磁波**具有能量**，是**物质存在的一种形式**。

### 电磁波通信

- 5G/4G/Wi-Fi/卫星：都是电磁波在不同频段的应用。
- 信号 → 调制（AM/FM/数字）→ 天线发射 → 空间传播 → 接收天线 → 解调 → 还原信号。
- 卫星宽带通信使全球随时随地接入互联网成为可能。

### 例题：微波炉

- 频率 $f = 2450$ MHz → $\lambda = c/f = 3\times10^8/(2.45\times10^9) \approx 0.122$ m ≈ 12.2 cm。
- 微波加热"内部不易过热"是因为微波穿透深度有限，由表及里。

### 易错提示

- 电磁波传播**不需要介质**——这是与机械波（声波、水波）的根本区别。
- 真空中所有电磁波速度都等于光速 $c$。
- 进入介质后速度变慢、频率不变、波长缩短。
- 光速并非"光独有"：所有电磁波在真空中都以 $c$ 传播。
- 微波是无线电波的一部分，不是独立波段。

## 5. 能量量子化

### 热辐射

- 任何物体都辐射电磁波，与温度有关 → **热辐射**。
- 温度升高 → 辐射强度增大，短波长成分增多 → 物体由暗红→赤红→橘红→黄白色。
- **黑体**：能完全吸收入射各种波长电磁波而不反射的物体；其辐射强度按波长的分布**只与温度有关**。
- 经典电磁理论解释黑体辐射 → 紫外灾难（推出短波长辐射强度无限大），与实验不符。

### 能量子（普朗克 1900）

- **大胆假设**：振动的带电微粒能量只能取某一最小值 $\varepsilon$ 的整数倍。
- 最小能量单位 $\varepsilon$ 叫**能量子**：

$$\boxed{\,\varepsilon = h\nu\,}$$

- $\nu$：电磁波频率（注意：在原子物理中频率常用希腊字母 $\nu$）；
- $h$：**普朗克常量**：

$$h = 6.626\,070\,15 \times 10^{-34}\ \text{J·s}$$

- 借此推出与实验吻合的黑体辐射分布公式 → 量子论的开端。

### 光子（爱因斯坦 1905）

- 爱因斯坦推广能量子假设：**电磁场本身**是不连续的，由一个个**光子**组成。
- 频率 $\nu$ 的光，每个光子能量 $\varepsilon = h\nu$。

### 能级

- 微观粒子（原子、分子）的能量也是量子化的，取**分立**值，称**能级**。
- 原子从高能级 $E_n$ 跃迁到低能级 $E_m$，放出光子能量：

$$h\nu = E_n - E_m$$

- 因此原子发射光谱呈**分立亮线**（不是连续光谱）。

### 数量级感

- 红光 $\lambda = 700$ nm，$\nu = 4.3\times10^{14}$ Hz → 单光子能量 $h\nu \approx 2.8\times10^{-19}$ J ≈ 1.8 eV
- 蓝光 $\lambda = 450$ nm → 单光子 $\approx 4.4\times10^{-19}$ J ≈ 2.8 eV
- → 蓝光每个光子能量高于红光（虽然两者速度一样）

### 例题：激光的光子数

> 激光器波长 $\lambda = 632.8$ nm，功率 $P = 18$ mW。
- 单光子能量 $\varepsilon = hc/\lambda = (6.63\times10^{-34} \times 3\times10^8)/(6.328\times10^{-7}) \approx 3.14\times10^{-19}$ J
- 每秒光子数 $N = P/\varepsilon = 0.018 / 3.14\times10^{-19} \approx 5.7\times10^{16}$ 个/秒

### 量子力学的意义

- 普朗克 → 爱因斯坦 → 玻尔 → 德布罗意 → 海森堡 / 薛定谔（1920 年代）共同奠定量子力学。
- 现代科技：核能、计算机、智能手机、激光、半导体、超导，皆基于量子力学。

### 易错提示

- 能量子 $\varepsilon = h\nu$ 中 $\nu$ 是**频率**，不是波长。
- 普朗克常量 $h$ 的单位是 J·s，不是 J/s（瓦特）。
- 经典电磁理论解释微观辐射会出现"紫外灾难"，要意识到这是量子论的诞生背景。
- 高温物体颜色偏向短波长（蓝色一端），低温偏向长波长（红外）——星色判定温度的依据。
