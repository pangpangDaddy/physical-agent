# 必修第三册·第十二章 电能 能量守恒定律（精读）

> 主线：电流做功的本质 → 焦耳定律 → 电源的非静电力做功（电动势）→ 闭合电路欧姆定律 → 测电源电动势和内阻 → 能源利用与可持续发展。
> 核心方程：$W = UIt$、$P = UI$、$Q = I^2 R t$、$\varepsilon = W/q$、$I = \varepsilon/(R+r)$。

---

## 1. 电路中的能量转化

### 电流做功的本质

- 电场对自由电荷的静电力做功 → 电荷电势能减少 → 转化为其他形式的能。
- 电热水壶：电能 → 内能；
- 电动机：电能 → 机械能 + 内能（线圈发热）；
- 蓄电池充电：电能 → 化学能 + 内能。

### 电功

$$\boxed{\,W = qU = UIt\,}$$

- 一段电路两端电压 × 通过电流 × 时间。
- 单位：J；$1$ kW·h = $3.6 \times 10^6$ J。
- **普适**：对任何用电器都成立（纯电阻、电动机、电池）。

### 电功率

$$\boxed{\,P = \dfrac{W}{t} = UI\,}$$

- 单位：W（$1$ W = $1$ J/s）。
- **普适**：电流做功的功率。

### 焦耳定律（纯电阻情况）

$$\boxed{\,Q = I^2 R t\,}$$

- 电流通过纯电阻**发热**的热量。
- 由欧姆定律 $U = IR$ 与 $W = UIt$ 联立得：
  - $Q = UIt = I^2 R t = \dfrac{U^2 t}{R}$
- 发热功率：

$$P_{\text{热}} = I^2 R = \dfrac{U^2}{R}$$

### 纯电阻 vs 非纯电阻电路

| 类型 | 电能转化 | 关系 |
|---|---|---|
| **纯电阻**（电热器、白炽灯、电饭锅加热） | 电能全部→内能 | $W = Q$，$P = P_{\text{热}}$，$U = IR$ |
| **非纯电阻**（电动机、电池充电） | 电能→机械能/化学能 + 部分内能 | $W > Q$，$P_{\text{电}} = P_{\text{机}} + P_{\text{热}}$，$U \neq IR$ |

### 例题：电动机功率分配

> $R = 6$ Ω，$U = 220$ V，$I = 5$ A。
- 消耗电功率 $P_{\text{电}} = UI = 1100$ W
- 发热功率 $P_{\text{热}} = I^2 R = 150$ W
- 机械功率 $P_{\text{机}} = P_{\text{电}} - P_{\text{热}} = 950$ W

### 易错提示

- 电动机两端电压 $U \neq IR$！应理解为 $U = U_{\text{反}} + IR$（反电动势 + 内阻分压）。
- 焦耳定律 $Q = I^2 R t$ **任何时候都成立**（适用于任意电路，描述的是电流通过电阻发热），与"纯/非纯"无关。
- $W = UIt$ 描述的是电流总做的功（电能消耗），与电阻是否纯无关。
- "电功 = 电热"只在纯电阻成立。

## 2. 闭合电路的欧姆定律

### 内、外电路

- **外电路**：电源外部，由导线和用电器组成。
- **内电路**：电源内部。
- 内电路存在**电阻**（内阻 $r$）。

### 电动势

- 电源中**非静电力**把正电荷由负极搬到正极的能力。
- 定义：

$$\boxed{\,\varepsilon = \dfrac{W_{\text{非}}}{q}\,}$$

- $W_{\text{非}}$：非静电力把电荷 $q$ 从负极搬到正极所做的功。
- 单位：V（与电压相同）。
- **由电源本身决定**，与外电路无关、与体积无关。
- 数值上 = 1 C 正电荷在电源内部由负极移到正极时，非静电力所做的功。
- 国家标准用 $E$ 表示，与电场强度 $E$ 易混淆，本书统一用 $\varepsilon$（教材也用 $E$，注意区分）。

### 闭合电路欧姆定律

$$\boxed{\,I = \dfrac{\varepsilon}{R + r}\,}$$

- $R$：外电阻；
- $r$：内阻；
- $\varepsilon$：电动势。

### 等价形式

$$\varepsilon = U_{\text{外}} + U_{\text{内}} = IR + Ir$$

**能量解释**：单位时间内非静电力做的功 = 外电路电流做的功 + 内电路发热。

### 路端电压 $U$（外电路电压）

$$U = \varepsilon - Ir = IR$$

- 闭合电路下 $U-I$ 图像为一条直线，斜率为 $-r$，纵截距为 $\varepsilon$。
- 负载电阻 $R$ 变化 → $I$ 和 $U$ 都变化。

### 极端情形

| 情况 | $R$ | $I$ | $U$ |
|---|---|---|---|
| **断路** | $\infty$ | 0 | $\varepsilon$（外电压 = 电动势） |
| **短路** | 0 | $\varepsilon / r$（极大） | 0 |
| 一般负载 | $R$ | $\varepsilon/(R+r)$ | $IR$ |

→ 测电动势的常用方法："让外电路断开，测两极电压 = $\varepsilon$"。

### $R$ 变化对 $U$、$I$ 的影响

- $R$ ↑ → $I$ ↓ → $Ir$ ↓ → $U$ ↑（路端电压升高）
- $R$ ↓ → $I$ ↑ → $U$ ↓（路端电压降低）

> 解释"汽车启动时车灯变暗"：电动机启动瞬间电流大 → 内阻分压大 → 路端电压低 → 车灯变暗。

### 电源输出功率与最大功率

- 输出功率：$P_{\text{出}} = UI = I^2 R = \dfrac{\varepsilon^2 R}{(R+r)^2}$
- 当 **$R = r$** 时，$P_{\text{出}}$ 最大：$P_{\text{出,max}} = \dfrac{\varepsilon^2}{4r}$
- 电源效率：$\eta = U/\varepsilon = R/(R+r)$；$R$ 越大效率越高。
- 设计电源时要权衡输出功率与效率。

### 电阻表（拓展）

- 由表头 + 电池 + 调节电阻构成；红黑表笔短接时电流最大（满偏）→ 标为 $0$ Ω。
- $R_x = \dfrac{\varepsilon}{I_x} - R_\Omega$
- 刻度**不均匀**（非线性）。

### 易错提示

- 电动势 $\varepsilon$ 与端电压 $U$ 不同：$\varepsilon$ 只与电源有关，$U$ 与负载有关。
- 短路电流不是无穷大，受内阻限制。
- 同一回路里串联电池时电动势和内阻都加（$\varepsilon_{\text{总}} = \sum \varepsilon$，$r_{\text{总}} = \sum r$）。
- 并联同型号电池：电动势不变，内阻减小到 $r/n$。
- 路端电压 $U$ 比电动势 $\varepsilon$ 小一个 $Ir$，**外电路只能测到 $U$**。

## 3. 实验：电池电动势和内阻的测量

### 实验思路

- 由 $\varepsilon = U + Ir$，测多组 $U, I$ → 作 $U$-$I$ 图像 → 截距 = $\varepsilon$，斜率绝对值 = $r$。

### 标准电路

- 电源 + 电压表 V + 电流表 A + 滑动变阻器 R + 开关 S。
- 改变 R → 多组数据。

### 数据处理（图像法 vs 联立方程法）

| 方法 | 优劣 |
|---|---|
| 联立两组方程 | 误差大、仅作初步估计 |
| **U-I 图像法** | 多组数据线性拟合，误差小 |

### 由 U-I 图像求 ε 和 r

$$U = -rI + \varepsilon$$

- 纵截距 = $\varepsilon$；
- 斜率绝对值 = $r$；
- $U$ 坐标轴起点**不一定从 0 开始**——干电池内阻小，$U$ 变化小，要选合适坐标范围使图像分布均匀。

### 实验注意

1. 选择**旧干电池**（内阻较大）实验，效果明显；
2. 测量过程中尽量缩短通电时间，避免电池电动势漂移；
3. **滑动变阻器电阻调到最大**再闭合开关（保护电路）；
4. 多次测量分散数据，避免数据集中。

### 替代电路（无电流表 / 无电压表）

- **只有电压表 + 电阻箱**：$U = \varepsilon - \dfrac{U}{R}r$ → 测多组 $(U, R)$ 拟合。
- **只有电流表 + 电阻箱**：$I = \dfrac{\varepsilon}{R + r + R_A}$ → 测多组 $(I, R)$ 拟合 $1/I$ vs $R$。

### 误差分析

- 电压表分流 → 测得 $\varepsilon$ 偏小、$r$ 偏小（流过电流表的不全是流过 $R$ 的）；
- 电流表内阻不计 → 测得 $r$ 偏大（实际含 $R_A$）；
- 准确度要求高时用电桥 / 高精度数字仪表。

### 易错提示

- 滑动变阻器初始位置必须电阻最大；
- $U$ 坐标轴起点不能死板取 0；
- 不能根据一两组数据就下结论，必须多组拟合。

## 4. 能源与可持续发展

### 能量守恒定律

> 能量既不会凭空产生，也不会凭空消失，它只能从一种形式转化为其他形式，或者从一个物体转移到别的物体；在转化或转移过程中**总量保持不变**。

### 能量转化的方向性

- 自然界中宏观过程是**不可逆的**——能量守恒，但**可利用品质降低**（熵增）。
- 例：燃料燃烧释放热能 → 散逸到环境后无法再聚集利用 → 能量"耗散"。
- 这是节约能源的根本原因。

### 能源分类

| 类型 | 例 |
|---|---|
| **不可再生能源** | 煤、石油、天然气（化石能源）、核裂变燃料 |
| **可再生能源** | 水能、风能、太阳能、生物质能、地热能、潮汐能 |
| **一次能源** | 自然界直接获得的（煤、风、太阳） |
| **二次能源** | 由一次能源加工后的（电能、汽油、氢能） |
| **清洁能源** | 燃烧或使用过程对环境影响小（天然气、清洁煤、风、太阳、核能） |

### 各种发电方式简述

- **水力发电**：水能→机械能→电能；中国装机量世界第一（2015 年 $3\times10^8$ kW）。
- **风力发电**：风能→机械能→电能；中国装机量世界第一。
- **太阳能发电**：光生伏特效应直接将光能转化为电能；中国光伏产品制造世界第一。
- **核电**：
  - 裂变：重核分裂释能（铀-235）；
  - 聚变：轻核聚合释能（氢→氦），产能效率高、废料少；
  - 中国"华龙一号"——具有完全自主知识产权的三代百万千瓦级核电技术。

### 估算题型

- 水电站功率：$P = \rho_\text{水}Q\cdot g h \cdot \eta$
  - $Q$：流量（m³/s）；$h$：水头；$\eta$：转化效率。
- 风机功率：$P = \tfrac{1}{2}\rho_\text{空气} S v^3 \cdot \eta$
- 太阳能热水器：$P = E_0 S \cdot \eta$
  - $E_0$：太阳辐射照度（J/m²·s）。

### 能源利用与社会

- 三阶段：柴薪 → 煤炭 → 石油 → 多能源结构（核能 + 可再生）。
- 化石燃料燃烧 → CO₂ 温室效应、SO₂ 酸雨、NOₓ 光化学烟雾、粉尘。
- 可持续发展：满足当代人需求且不损害子孙后代需求 → **节能 + 清洁能源**。

### 易错提示

- 能量守恒不意味着能量"取之不尽"——品质会下降。
- 计算效率时要分清"输入能"与"输出能"。
- 估算题数据较粗，注意有效数字。

## 章末公式速查

```
电功 / 电热 / 功率
─────────────────
W = UIt              （电流做功，普适）
Q = I²Rt = U²t/R     （焦耳定律，发热）
P_电 = UI            （消耗总功率，普适）
P_热 = I²R = U²/R    （发热功率）
纯电阻：W = Q，U = IR
非纯电阻：P_电 = P_机 + P_热

闭合电路
ε = U_外 + U_内 = IR + Ir
I = ε / (R + r)
U = ε − Ir         （路端电压）

输出功率与效率
P_出 = εI − I²r = ε²R/(R+r)²
P_出,max = ε²/(4r)，当 R = r
η = U/ε = R/(R+r)

特殊情况
断路：I = 0，U = ε
短路：U = 0，I = ε/r
```

## 高频易错速查

- [ ] 电动机两端 $U = IR$（错，应是 $U \neq IR$，要算反电动势）
- [ ] 焦耳定律只用 $Q = UIt$（实际 $Q = I^2 R t$，非纯电阻时 $W > Q$）
- [ ] 把电动势与路端电压混为一谈
- [ ] 短路电流写成无穷大
- [ ] 串联电池漏算总内阻
- [ ] U-I 图像中 $U$ 坐标从 0 开始导致图线压缩
- [ ] 节能理由说成"能量减少"（错，应是品质下降）
