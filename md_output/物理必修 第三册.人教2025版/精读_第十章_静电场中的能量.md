# 必修第三册·第十章 静电场中的能量（精读）

> 主线：静电力做功的"与路径无关"特性 → 电势能 → 电势 → 电势差 → 电势差与电场强度的关系 → 电容器储能 → 带电粒子在电场中的加速与偏转。
> 核心方程：$W_{AB} = qU_{AB}$、$U_{AB} = Ed$、$C = Q/U$、加速 $qU = \tfrac{1}{2}mv^2$、偏转 类比平抛。

---

## 1. 电势能和电势

### 静电力做功的特点

匀强电场 $E$ 中，正电荷 $q$ 从 A 沿任意路径运动到 B：
- 直线 AB：$W_{AB} = F \cos\theta \cdot |AB| = qE\cdot|AM|$（其中 AM 为 AB 在场线方向的投影）。
- 折线 AMB / 任意曲线：垂直 $E$ 方向不做功；总功仍 $= qE|AM|$。

> **结论**：静电力做功**与路径无关**，只与初末位置有关。匀强电场 + 非匀强电场都成立。

→ 与重力做功类似 → 静电力是**保守力** → 电场中存在**电势能**。

### 电势能

- 电荷在电场中由于位置而具有的能量，记 $E_p$。
- 静电力做的功 = 电势能的减少量：

$$\boxed{\,W_{AB} = E_{pA} - E_{pB}\,}$$

| 静电力做功 | 电势能变化 |
|---|---|
| $W_{AB} > 0$（做正功） | 电势能减少（$E_p$ 减少） |
| $W_{AB} < 0$（做负功） | 电势能增加 |
| $W_{AB} = 0$ | 电势能不变 |

- 电势能是**电荷+电场系统所共有的**（类似重力势能是物体+地球共有）。
- **零势能位置**：通常取**无穷远**或**大地**为零电势能位置（理论计算取无穷远，实际工程取大地）。

### 电势

把电势能除以电荷量 → 得到与试探电荷无关的场点性质：

$$\boxed{\,\varphi = \dfrac{E_p}{q}\,}$$

- 单位：伏特 V，$1\ \text{V} = 1\ \text{J/C}$。
- **标量**，可正可负（取决于零电势位置选择）。
- **特性**：
  - 沿电场线方向 → 电势**降低**；
  - 在电场中**正电荷在电势高处电势能大**，**负电荷在电势低处电势能大**（关键易错点）；
  - 改变零电势位置 → 各点电势值改变，但任意两点的**电势差不变**。

### 由电势能判断电势的高低（典型方法）

| 已知 | 判断 |
|---|---|
| 正电荷 在 A 比 B 电势能大 | $\varphi_A > \varphi_B$ |
| 负电荷 在 A 比 B 电势能大 | $\varphi_A < \varphi_B$ |
| 沿电场线方向 | 电势降低 |

### 易错提示

- 电势能和电势是**两个不同的量**——电势能与电荷有关，电势是场点性质。
- 电势能正负取决于"零势能点"的选取，不等同于电势的正负。
- "无穷远电势为零"通常使**正电荷激发的电场**各点电势为正、负电荷激发的电场各点电势为负。
- 电势能变化 = 静电力做的功（带正负号）；不是 = 外力做的功。

## 2. 电势差

### 定义

$$\boxed{\,U_{AB} = \varphi_A - \varphi_B\,}$$

- 也叫**电压**，单位伏特 V。
- $U_{AB} = -U_{BA}$。
- $U_{AB}$ 可正可负，正负反映"A 比 B 高还是低"。
- 与零电势位置选择**无关**——这是电势差比电势更基本的原因。

### 静电力做的功与电势差

$$\boxed{\,W_{AB} = qU_{AB}\,}$$

- 推导：$W_{AB} = E_{pA} - E_{pB} = q\varphi_A - q\varphi_B = q(\varphi_A - \varphi_B)$。
- **重要应用**：已知 $U_{AB}$，从 A 到 B 移动电荷 $q$，立刻可求 $W$，不需要管路径。

### 例题（必修三典型）

> 匀强电场中 $q = 2 \times 10^{-9}$ C 从 A 到 B 做功 $1.6 \times 10^{-7}$ J，从 B 到 C 做功 $-4 \times 10^{-7}$ J。
- $U_{AB} = W_{AB}/q = 80$ V
- $U_{BC} = -200$ V
- $U_{AC} = U_{AB} + U_{BC} = -120$ V
- 电势高低：$\varphi_C > \varphi_A > \varphi_B$
- 把 $-1.5 \times 10^{-9}$ C 电荷从 A 到 C：$W = q'U_{AC} = (-1.5\times10^{-9})\times(-120) = 1.8\times10^{-7}$ J（正功）

### 等势面

- 电场中电势相同的点构成的曲面。
- **性质**：
  1. 等势面与电场线**处处垂直**；
  2. 电场线由高电势等势面指向低电势等势面；
  3. 等势面**不相交**；
  4. 同一图中相邻等势面间电势差**相等**——则等势面密的地方 $E$ 大；
  5. 在等势面上移动电荷，静电力**不做功**。
- 典型形态：
  - 点电荷：以电荷为中心的同心球面；
  - 匀强电场：与场线垂直的等距平行平面；
  - 等量异种电荷：左右对称的曲面，连线中垂面为零等势面。

### 电子伏（拓展）

- $1$ eV = 一个电子经 $1$ V 电势差加速后所获得的动能：

$$1\ \text{eV} = e \cdot 1\ \text{V} = 1.60 \times 10^{-19}\ \text{J}$$

### 易错提示

- $U_{AB}$ 的下标顺序很重要，写错正负就反了。
- 沿电场线方向 → 电势降低（不是升高！）。
- 正电荷沿电场线方向移动时，静电力做正功、电势能减少。
- 同一等势面上电势相等，但电场强度可以不同（如点电荷的球面，不同方向 $E$ 方向不同）。

## 3. 电势差与电场强度的关系

### 匀强电场中

$$\boxed{\,U_{AB} = Ed\,}\quad \text{或}\quad E = \dfrac{U_{AB}}{d}$$

- $d$ 是 A、B **沿电场方向**的距离（**不是 AB 直线距离**！）。
- 推导：在匀强电场中移动电荷 $q$：$W = qEd$；又 $W = qU_{AB}$ → $U_{AB} = Ed$。

### 几何意义

> $E$ 数值上等于"沿电场方向单位距离上电势的降落"。

- 单位等价：$1\ \text{V/m} = 1\ \text{N/C}$。
- 推论：相邻等势面间电势差相等时，$E$ 大处等势面密。

### 一般电场

- 上式只严格适用于**匀强电场**。
- 非匀强电场也满足"沿场线方向电势降低"，但 $E$ 与 $U$ 不再有简单乘积关系。

### 典型例题：平行板间带电粒子运动

> 平行板 $d = 0.04$ m，加电压 $U = 200$ V，粒子 $q = 8\times10^{-15}$ C，$m = 2\times10^{-15}$ kg，从 M 板静止释放。
- $E = U/d = 5000$ V/m
- $F = qE = 4 \times 10^{-11}$ N（远大于重力 $2\times 10^{-14}$ N，重力可忽略）
- 由动能定理 $qU = \tfrac{1}{2}mv^2$ → $v = \sqrt{2qU/m} = 40$ m/s
- **注意**：$d$ 变为 $2d$ 时，$E$ 减半、$F$ 减半、$a$ 减半，但加速距离加倍 → **末速度不变**！这是因为电压 $U$ 决定动能。

### 易错提示

- $d$ 必须沿场强方向取，不能取斜距离。
- $U_{AB} = Ed$ 中 $U_{AB}$ 有正负，$d$ 是有向距离（从 A 到 B 沿 $E$ 方向为正）。
- 加速电场中：粒子能量由 $U$ 决定，与 $d$ 无关（$qU = \tfrac{1}{2}mv^2$ 是关键公式）。

## 4. 电容器的电容

### 电容器

- **构造**：两个相距很近的导体（极板）夹介质（电介质，可以是空气）。
- **充电**：电源把电子从一极板搬到另一极板 → 两板带等量异种电荷 → 板间产生电场。
- **放电**：连接外电路后正负电荷中和，电场消失。
- 任意两个绝缘相距很近的导体都可看作电容器。

### 电容（定义）

$$\boxed{\,C = \dfrac{Q}{U}\,}$$

- $Q$ 是**一个极板**所带电荷量的绝对值。
- 单位：法拉 F，$1\ \text{F} = 1\ \text{C/V}$。常用 $\mu\text{F} = 10^{-6}$ F、$\text{pF} = 10^{-12}$ F。
- **物理意义**：电容是电容器储存电荷的"容量"，**与 Q、U 无关**——只由结构决定。

### 平行板电容器电容（结构决定式）

$$C = \dfrac{\varepsilon_r S}{4\pi k d}$$

- $S$：正对面积
- $d$：板间距
- $\varepsilon_r$：电介质相对介电常数（真空为 1）
- $k$：静电力常量

**结论**：
- $S$ ↑ → $C$ ↑
- $d$ ↑ → $C$ ↓
- 插入电介质（$\varepsilon_r > 1$） → $C$ ↑

### 充放电图像

- I-t 图像下面积 = 电荷量 $Q$（$Q = \int I\ dt$）。
- 充电：电流由电源正极流向电容器正极板，逐渐减小到 0。
- 放电：电流方向与充电相反，幅值随两板电势差减小。

### 击穿与额定电压

- 加在电容器上的电压不能超过**击穿电压**（介质击穿失去绝缘）；
- 标牌上的是**工作电压（额定电压）**，低于击穿电压留余量。

### 动态分析

| 条件 | 关系 |
|---|---|
| 电容器始终与电源相连（$U$ 不变） | $Q \propto C$；改 $d, S, \varepsilon_r$ → $Q$ 跟 $C$ 同变；$E = U/d$ 随 $d$ 反比变 |
| 充电后断开电源（$Q$ 不变） | $U = Q/C$；改 $d, S, \varepsilon_r$ → $U$ 跟 $C$ 反变；$E = U/d = Q/(C d)$ 实际 $E = \sigma/(\varepsilon_0\varepsilon_r)$ 只与 $\sigma$ 有关，与 $d$ 无关 |

→ 这是高频考点："连着电源" vs "断开电源"两种情形要分别讨论。

### 易错提示

- $C = Q/U$ 是**定义式**，不能反推"$C$ 与 $U$ 成反比"或"$C$ 与 $Q$ 成正比"。
- $C$ 由电容器自身决定。
- 插入电介质后 $C$ 增大（除非真空 / 空气，否则 $\varepsilon_r > 1$）。
- "电容器所带的电荷量"指一个极板的电荷量绝对值，不是两板之和（两板代数和 = 0）。

## 5. 带电粒子在电场中的运动

### 两类典型问题

#### 加速（速度方向沿电场或反场方向）

- 沿场方向运动：受恒力 $F = qE$ → 匀加速直线运动。
- **动能定理法**（普适，匀场 / 非匀场均可）：

$$qU = \dfrac{1}{2}mv^2 - \dfrac{1}{2}mv_0^2$$

- 当 $v_0 = 0$ 时：$v = \sqrt{\dfrac{2qU}{m}}$
- **不需要管加速器内的细节**——只看入口、出口的电势差。

#### 偏转（速度方向与电场方向垂直）

- 类比**平抛运动**：水平匀速 + 竖直匀加速。

设带电粒子以速度 $v_0$ 平行于板射入长 $l$、间距 $d$、电压 $U$ 的平行板间：

$$a = \dfrac{qE}{m} = \dfrac{qU}{md}$$

| 物理量 | 公式 |
|---|---|
| 飞行时间 | $t = \dfrac{l}{v_0}$ |
| 偏移量 | $y = \dfrac{1}{2}at^2 = \dfrac{qUl^2}{2md v_0^2}$ |
| 出射纵向速度 | $v_\perp = at = \dfrac{qUl}{md v_0}$ |
| 偏转角 | $\tan\theta = \dfrac{v_\perp}{v_0} = \dfrac{qUl}{md v_0^2}$ |

### 加速 + 偏转的复合问题（示波管原理）

> 电子枪加速 $U_1$ → 偏转电场 $U_2$ → 荧光屏成像。
- 加速：$eU_1 = \tfrac{1}{2}mv_0^2$ → $v_0^2 = 2eU_1/m$
- 偏转角：$\tan\theta = \dfrac{eU_2 l}{md v_0^2} = \dfrac{U_2 l}{2dU_1}$
- 偏移量与质量无关、与电荷量无关（在 $U_1, U_2$ 给定时）；只取决于几何与电压比。

### 示波管 / 电子直线加速器

- **示波管**：电子枪 + 偏转电极 XX′（扫描电压）+ YY′（信号电压）+ 荧光屏。
- **直线加速器**：交流电源驱动一排金属圆筒，电子在圆筒内做匀速（屏蔽），在缝隙间被加速；圆筒长度与序号 $n$ 成 $\sqrt{n}$ 关系。

### 例题：直线加速器圆筒长度

> 电子质量 $m$、电荷 $e$、交流电压 $u$、周期 $T$。
- 电子进入第 $n$ 个圆筒后：$neu = \tfrac{1}{2}mv_n^2$ → $v_n = \sqrt{2neu/m}$
- 第 $n$ 个圆筒长度（要求电子在筒中恰好走半个周期）：

$$l_n = v_n \cdot \dfrac{T}{2} = \dfrac{T}{2}\sqrt{\dfrac{2neu}{m}}$$

### 解题思路总结

| 题型 | 优先方法 |
|---|---|
| 匀强电场、涉及时间 / 加速度 | 牛顿第二定律 + 匀变速运动公式 |
| 匀强 / 非匀强电场、只问初末位移与速率 | 动能定理 + $W = qU$ |
| 加速场 + 偏转场组合 | 加速段用 $qU = \tfrac{1}{2}mv_0^2$；偏转段类比平抛 |
| 是否考虑重力 | 看 $qE$ 与 $mg$ 比；电子、α粒子等微观粒子的重力通常忽略 |

### 易错提示

- 是否忽略重力：先比 $qE$ 与 $mg$ 的数量级；微观粒子（电子、质子、α）几乎都可以忽略，带电小球类问题则要考虑。
- 偏转后离开电场仍做匀速直线运动（不再受静电力），轨迹是切线方向延长线，延长线**反向延长过两板正中**（"中点定理"）。
- 加速电压 $U$ 决定动能，与板间距、加速距离无关。

```
电势能 / 电势 / 电势差
─────────────────────────
W_AB = E_{pA} − E_{pB} = q U_{AB}
φ = E_p / q
U_{AB} = φ_A − φ_B = −U_{BA}
W_{AB} = q U_{AB}
1 eV = 1.60×10⁻¹⁹ J

匀强电场
U_{AB} = E d        (d 为沿场方向 A→B 距离)
E = U / d
1 V/m = 1 N/C

电容器
C = Q / U                  (定义)
C = ε_r S / (4πkd)         (平行板结构)
储能 W = ½ Q U = ½ C U² = Q²/(2C)

带电粒子运动
加速：qU = ½ m v² − ½ m v₀²
偏转：a = qU/(md)，y = qUl²/(2mdv₀²)，tanθ = qUl/(mdv₀²)
示波管偏移：y = U₂l²/(4dU₁) × l_后/l 等扩展
"中点定理"：出射粒子速度反向延长过偏转板正中点
```

## 高频易错速查

- [ ] 把"电势能减少"与"电势减小"混淆
- [ ] 把正电荷的电势能正负 = 电势正负
- [ ] $U_{AB} = Ed$ 用了 AB 直线距离而非沿场距离
- [ ] $C$ 与 $Q, U$ 的依赖关系搞反
- [ ] "连着电源 vs 断开电源"动态分析没分类讨论
- [ ] 偏转粒子忽略静电力以外的力（电子的重力一般可忽略，带电油滴不能）
- [ ] 加速 + 偏转复合问题中没用动能定理化简
