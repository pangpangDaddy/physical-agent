# 必修第三册·第九章 静电场及其应用（精读）

> 主线：电荷的本质 → 电荷之间的相互作用规律（库仑定律）→ 用"场"描述这种相互作用（电场、电场强度、电场线）→ 静电的防护与利用。
> 核心方程：$F = k\dfrac{q_1 q_2}{r^2}$、$E = \dfrac{F}{q}$、$E = k\dfrac{Q}{r^2}$。

---

## 1. 电荷

### 电荷与电荷量

- **电荷**：物体由于摩擦等过程而具有的吸引轻小物体的性质所对应的"东西"。自然界只有两种电荷——**正电荷**与**负电荷**。
- **电荷量** $Q$（或 $q$）：电荷的多少，标量。单位：库仑 C（$1$ A 在 $1$ s 内传送的电荷量为 $1$ C）。
- 同种相斥、异种相吸。

### 起电方式

| 方式 | 原理 | 例子 |
|---|---|---|
| **摩擦起电** | 两物体摩擦时部分受束缚较弱的电子转移 | 琥珀吸羽毛、塑料梳子起电 |
| **接触起电** | 带电体接触不带电导体后电荷分配 | 验电器 |
| **静电感应** | 带电体靠近导体，导体上自由电子受静电力重新分布 | C 靠近 A-B 组合导体，使近端带异号、远端带同号 |

### 物质的微观结构（金属）

- 原子 = 原子核（质子+中子）+电子；原子整体电中性。
- 金属：**自由电子**穿梭于规则排列的正离子之间，因此导电。
- 绝缘体：几乎没有可自由移动的电荷。

### 电荷守恒定律

> 电荷既不会创生，也不会消灭，它只能从一个物体转移到另一物体或从物体的一部分转移到另一部分；总量保持不变。

更普遍表述：**一个与外界没有电荷交换的系统，电荷的代数和保持不变**（包含正负电子对生成 / 湮没）。

### 元电荷

- 最小电荷量 = 电子电荷量绝对值：

$$e = 1.602\,176\,634 \times 10^{-19}\ \text{C} \approx 1.60 \times 10^{-19}\ \text{C}$$

- 所有带电体的电荷量都是 $e$ 的整数倍（电荷量量子化）。
- 比荷：$\dfrac{e}{m_e} = 1.76 \times 10^{11}\ \text{C/kg}$。

### 易错提示

- "感应起电"过程中电荷只是**转移**，没有"产生"新电荷。
- 验电器只能判断带电与否、大致多少，**不能判断电性正负**。
- "元电荷"是 $e$ 这个数值，**不是某种粒子**。

## 2. 库仑定律

### 点电荷

- **理想模型**：带电体之间距离 ≫ 带电体大小，且电荷分布对作用力影响可忽略。
- 类似于"质点"，是个模型。

### 库仑扭秤实验

- 库仑用扭秤通过悬丝扭转角度比较力的大小。
- 通过"接触均分"法把电荷量分为 $q/2, q/4, q/8\dots$ 验证电荷量关系。

### 库仑定律

$$\boxed{\,F = k\dfrac{q_1 q_2}{r^2}\,}$$

- 适用条件：**真空中的两个静止点电荷**。
- $k = 9.0 \times 10^9\ \text{N·m}^2/\text{C}^2$（静电力常量）。
- 方向：沿两电荷连线。同号相斥，异号相吸。
- $r = 1$ m、$q_1 = q_2 = 1$ C 时，$F = 9 \times 10^9$ N（约 100 万吨重物！可见 1 C 是巨大单位）。

### 多电荷情形

- **叠加原理**：任意点电荷受到的合力 = 各其他点电荷单独作用力的**矢量和**。
- 两点电荷间的作用力**不因第三个电荷存在而改变**。

### 例题：氢原子内静电力 vs 万有引力

- 氢原子核 - 电子距离 $r = 5.3 \times 10^{-11}$ m。
- $F_\text{库} \approx 8.2 \times 10^{-8}$ N
- $F_\text{引} \approx 3.6 \times 10^{-47}$ N
- $F_\text{库}/F_\text{引} \approx 2.3 \times 10^{39}$
- 推论：**微观粒子间万有引力远小于库仑力，可忽略**。

### 易错提示

- 库仑定律的 $r$ 是距离，必须 $r > 0$；当 $r \to 0$ 时点电荷模型失效，不能直接代入。
- 球形带电体可视为点电荷处理时，距离取**球心间**距离（前提：球之间距离远大于球半径）。
- 两个相同金属球接触后再分开 → 电荷量平均分配（同种电荷直接平分；异种电荷先中和后平分）。

## 3. 电场 · 电场强度

### 电场的物质性

- 电荷之间的相互作用通过**电场**发生（法拉第提出，麦克斯韦完善）。
- 场是**物质存在的一种形式**，与实物粒子并列。
- 静止电荷产生的电场叫**静电场**。

### 试探电荷 vs 场源电荷

- **场源电荷**：激发电场的电荷。
- **试探电荷**：用来检测电场的电荷，要求电荷量 $q$ 很小（不影响原电场）、体积很小（便于测点电场）。

### 电场强度定义

$$\boxed{\,E = \dfrac{F}{q}\,}$$

- $F$：试探电荷在电场中某点受到的静电力。
- 单位：N/C（牛每库），也等价于 V/m（伏每米）。
- **矢量**：方向规定为**正电荷在该点所受静电力**的方向；负电荷所受力方向与 $E$ 相反。
- **重要性质**：$E$ 是场点性质，与试探电荷的电荷量 $q$ 无关，与是否放试探电荷无关。

### 点电荷的电场

$$E = k\dfrac{Q}{r^2}$$

- 距点电荷 $r$ 的球面上各点 $E$ 大小相等，方向沿径向（正电荷指向外，负电荷指向内）。
- 均匀带电球体在球外的电场等价于"同总电荷量集中于球心"的点电荷电场。

### 电场强度的叠加（矢量和）

- 多个场源时，某点 $E$ = 各场源单独在该点产生的 $E$ 的矢量和。
- 按平行四边形定则或正交分解处理。

### 电场线

- 形象描述电场的假想曲线；切线方向 = 该点 $E$ 方向。
- 性质：
  1. 始于正电荷或无穷远，终于负电荷或无穷远；
  2. 不相交（每点 $E$ 方向唯一）；
  3. 疏密表示 $E$ 大小（同一图中比较）；
  4. **电场线不闭合**（静电场）。
- 几种典型电场线（必记）：
  - 正点电荷：沿径向辐射向外；
  - 负点电荷：沿径向指向中心；
  - 等量异种点电荷（电偶极子）：从正电荷出发到负电荷；
  - 等量同种点电荷：两组排斥状曲线，中点处 $E = 0$；
  - 带等量异种电荷的平行金属板：板间为匀强电场，板外几乎为零。

### 匀强电场

- 各点 $E$ 大小相等、方向相同。
- 电场线为间隔相等的平行直线。
- 典型：相距很近的等量异种电荷平行金属板间（除边缘外）。

### 易错提示

1. $E = F/q$ 是**定义式**，不是决定式——$E$ 由场源决定，与 $F$ 和 $q$ 无关；
2. $E = kQ/r^2$ 是点电荷电场的**决定式**；
3. 电场线不是带电粒子的运动轨迹（轨迹要看初速度方向，与场线只在特殊情形重合）；
4. 电场线疏密的比较**只在同一幅图中**才有意义；
5. 正电荷沿场线方向运动，电势能减小；负电荷反之。

## 4. 静电的防止与利用

### 静电平衡

- 把不带电导体放入电场 $E_0$ 中 → 自由电子受静电力反向移动 → 两侧出现感应电荷 → 感应电荷产生反向场 $E'$ → 内部合场 $E = E_0 - E' = 0$ 时达到**静电平衡**。
- **特征**：
  1. 内部场强处处为 0；
  2. 电荷只分布在外表面；
  3. 内部各点电势相等（**等势体**）；
  4. 表面任意一点 $E$ 方向沿该点**外法线方向**；
  5. **越尖锐的位置电荷面密度越大**（→ 尖端放电）。

### 尖端放电

- 导体尖端处电荷密度大、附近 $E$ 大 → 使空气电离（中性分子电离为正离子 + 自由电子）→ 异号粒子被尖端吸收、与尖端电荷中和。
- **应用**：**避雷针**——雷雨云接近建筑物 → 避雷针顶部感应出异号电荷 → 尖端放电中和云电荷 → 避免雷击。
- **危害**：高压设备需把表面做光滑，避免尖端放电造成电能损失（电晕现象）。

### 静电屏蔽

- 空腔导体处于静电平衡时，**内表面无电荷、空腔内部 $E = 0$**——无论外界电场如何变化。
- 应用：把电学仪器置于金属壳内屏蔽外界电场；高压输电线上方架两条接地导线形成"金属网"屏蔽雷电。
- **金属网也能屏蔽**（无需密闭金属容器）。

### 静电吸附（利用）

| 应用 | 原理 |
|---|---|
| 静电除尘 | 烟气中粉尘被电离的电子带上电荷，被吸到正极板上回收 |
| 静电喷漆 | 油漆雾化器喷出的带电微粒在静电力作用下沉积到工件上，覆盖均匀 |
| 静电复印 | 光导体鼓充电 → 曝光 → 显影 → 转印 → 放电 5 步循环 |

### 静电的危害

- 印刷厂纸张相吸；
- 印染厂织物吸尘；
- 制药纯度下降；
- 油气运输火花引起爆炸；
- 手术台麻醉剂火花引爆。

### 防静电措施

- 接地（最直接）；
- 增加湿度；
- 减少摩擦或使用抗静电材料；
- 静电释放器（加油站要求触摸释放器后再加油）；
- 高压作业人员穿"金属丝织物"屏蔽服。

### 易错提示

- 静电平衡 ≠ 电荷不动：电荷已经达到分布平衡，**宏观电流为零**。
- 静电屏蔽只屏蔽外电场对内部的影响；**内部电场不影响外部仍然成立**（只屏蔽进入，不屏蔽离开——通过接地才能彻底屏蔽）。
- 验电器外面的金属杆顶部做成圆球而非尖端，是为了避免尖端放电造成电荷损失。
