# 必修第三册·第十一章 电路及其应用（精读）

> 主线：电流的概念 → 欧姆定律 → 影响电阻的因素（电阻定律）→ 电阻率测量实验 → 串并联规律 + 电表改装 → 多用电表使用。
> 核心方程：$I = q/t$、$R = U/I$（欧姆定律）、$R = \rho l/S$（电阻定律）。

---

## 1. 电源和电流

### 电源

- **作用**：通过**非静电力**做功，把正电荷从负极（低电势）搬运到正极（高电势），维持电路两极间稳定电势差。
- **电极**：正极（高电势）、负极（低电势）。
- 非静电力的物理本质因电源类型而不同：
  - 化学电池：化学反应力；
  - 发电机：电磁感应中的安培力等；
  - 太阳能电池：光生伏特效应。

### 恒定电场

- 电源 + 导线 + 用电器构成回路 → 电路中各处电荷分布稳定 → 产生**稳定**的电场 → 叫**恒定电场**。
- 性质与静电场相同：电势、电势差、$E$-$U$ 关系都适用。

### 电流

- 定义：电荷的定向移动；用单位时间内通过导体横截面的电荷量度量。

$$\boxed{\,I = \dfrac{q}{t}\,}$$

- 单位：安培 A（$1$ C = $1$ A·s）；常用 mA = $10^{-3}$ A、μA = $10^{-6}$ A。
- 电流方向：**正电荷定向移动方向**；电子定向移动方向相反。
- **标量**（虽然有"方向"但不参与矢量合成）。

### 恒定电流

- 大小、方向都不随时间变化的电流。

### 电流的微观表达式

$$I = nesv$$

- $n$：单位体积内自由电荷数；
- $e$：每个电荷的电荷量；
- $S$：导体横截面积；
- $v$：自由电荷定向移动平均速率。

> 自由电子无规则热运动速率约 $10^5$ m/s，定向移动平均速率约 $10^{-4}$ m/s；电场以光速建立 → 闭合开关瞬间各处电场几乎同时建立，电流"几乎同时"形成。

### 电池容量

- 单位 A·h 或 mA·h：电池放电时能输出的总电荷量。
- 例：$4000$ mA·h 电池 = $4000 \times 10^{-3} \times 3600 = 14400$ C。
- 估算待机/工作电流：$I = Q_{\text{容量}}/t$。

### 易错提示

- 电流方向与电子运动方向**相反**。
- $I$ 是标量，但有方向意义；电流的"代数和"在节点分析中需要正负号约定。
- 电池容量与电压无关；选择电池组要分串联（电压加）和并联（容量加）。

## 2. 导体的电阻

### 部分电路欧姆定律

$$\boxed{\,I = \dfrac{U}{R}\,}\quad \text{或}\quad R = \dfrac{U}{I}$$

- 适用条件：**金属导体、电解质溶液**等线性元件；温度变化不大。
- $R$ 由导体本身决定，与 $U$、$I$ 无关（在温度不变时）。

### U-I 图像（伏安特性曲线）

- **线性元件**（金属、电解液）：U-I 图像过原点直线，斜率 = $R$。
- **非线性元件**（气态导体、半导体二极管）：曲线，欧姆定律**不适用**。
- 习惯上 I-U 图像（横 $U$ 纵 $I$）：斜率 = $1/R$。

### 电阻定律

$$\boxed{\,R = \rho\dfrac{l}{S}\,}$$

- $l$：导体长度（m）；
- $S$：横截面积（m²）；
- $\rho$：电阻率（Ω·m），由材料决定。

### 几种常见材料的电阻率（20℃）

| 材料 | $\rho$ (Ω·m) |
|---|---|
| 银 | $1.6 \times 10^{-8}$ |
| 铜 | $1.7 \times 10^{-8}$ |
| 铝 | $2.9 \times 10^{-8}$ |
| 钨 | $5.3 \times 10^{-8}$ |
| 铁 | $1.0 \times 10^{-7}$ |
| 锰铜合金 | $4.4 \times 10^{-7}$（温度不敏感 → 制作标准电阻） |
| 镍铬合金 | $1.0 \times 10^{-6}$（电热丝） |

### 电阻率与温度的关系

- 金属导体：温度升高 → $\rho$ 增大（电阻温度计原理）；
- 半导体：温度升高 → $\rho$ **减小**；
- 锰铜 / 镍铜合金：$\rho$ 几乎不随温度变化（标准电阻）；
- 超导体：低于临界温度时 $\rho = 0$（1911 年发现，1986 年高温超导突破，1987 年中国赵忠贤等钇钡铜氧 90 K）。

### 易错提示

- $\rho$ 是材料本身性质，与 $l, S, R$ 无关；同一根铜丝拉细或剪短 $\rho$ 不变。
- 求拉长前后电阻比时注意：体积不变，$l$ 变 $k$ 倍则 $S$ 变 $1/k$ 倍 → $R$ 变 $k^2$ 倍。
- 求不同方向通电的电阻比时（长方体），公式中 $l$ 是"沿电流方向长度"，$S$ 是"垂直电流的截面积"。

## 3. 实验：导体电阻率的测量

### 测量原理

- 测出 $R, l, d$（圆截面 $S = \pi d^2/4$）→ $\rho = \dfrac{\pi d^2 R}{4 l}$。

### 工具与精度

| 工具 | 量程 / 分度 | 精度 |
|---|---|---|
| 刻度尺 | mm 级 | 0.1 mm 估读 |
| **游标卡尺**（10/20/50 分度） | 主尺 mm + 游标 | 0.1 / 0.05 / 0.02 mm，**不估读** |
| **螺旋测微器（千分尺）** | 0–25 mm，螺距 0.5 mm | 0.01 mm，最后一位**估读** |

### 游标卡尺读数

- **10 分度**：主尺 mm 整数 + 游标对齐刻度 × 0.1 mm
- **20 分度**：主尺 mm 整数 + 游标对齐刻度 × 0.05 mm
- **50 分度**：主尺 mm 整数 + 游标对齐刻度 × 0.02 mm
- **不估读**！

### 螺旋测微器读数

- 主尺读 mm（含半 mm 刻度线）+ 微分筒读数 × 0.01 mm + 估读位
- 例：6.5 + 22.5 × 0.01 = 6.725 mm

### 测电阻的电路（伏安法）

- **安培计内接（电压表外接）**：测的 $R$ 偏大（含电压表内阻），适合测**大电阻**（$R \gg R_V$ 时误差小）。
- **安培计外接（电压表内接）**：测的 $R$ 偏小（含电流表内阻），适合测**小电阻**（$R \ll R_A$ 时误差小）。

> 经验法则：$R_x \gtrsim \sqrt{R_A \cdot R_V}$ → 安培计**外接**；反之 → **内接**。

### 测细金属丝直径的方法

1. **直接法**：游标卡尺或螺旋测微器；
2. **累积法**：把细丝紧密缠绕在圆柱体上 N 圈，测总宽度 $L$，则 $d = L/N$。

### 数据分析

- 用 U-I 图像取直线斜率求 $R$；
- 测电阻丝的有效长度 $l$（接入电路部分，不是总长度）；
- 多次测量取平均，减小偶然误差。

### 易错提示

- 测量时电阻丝可能因发热改变电阻 → 通电不宜过久；
- 螺旋测微器读数注意主尺上半 mm 线是否露出；
- 缠绕法应排紧、不留缝、不重叠。

## 4. 串联电路和并联电路

### 串联

| 物理量 | 公式 |
|---|---|
| 电流 | $I = I_1 = I_2 = \dots = I_n$（处处相等） |
| 电压 | $U = U_1 + U_2 + \dots + U_n$ |
| 电阻 | $R = R_1 + R_2 + \dots + R_n$（总比每个都大） |
| 分压 | $U_k / U = R_k / R$ |
| 功率 | $P_k / P = R_k / R$ |

### 并联

| 物理量 | 公式 |
|---|---|
| 电流 | $I = I_1 + I_2 + \dots + I_n$ |
| 电压 | $U = U_1 = U_2 = \dots = U_n$（处处相等） |
| 电阻 | $\dfrac{1}{R} = \dfrac{1}{R_1} + \dfrac{1}{R_2} + \dots$（总比每个都小） |
| 两个并联简化 | $R = \dfrac{R_1 R_2}{R_1 + R_2}$ |
| 分流 | $I_k / I = R / R_k$ |
| 功率 | $P_k / P = R / R_k$ |

### 电压表 / 电流表的改装

#### 改装为电压表（量程大）
- 把表头与一个**大阻值分压电阻 $R_1$ 串联**：

$$R_1 = \dfrac{U}{I_g} - R_g$$

- 电压表可看作"电压可读的大电阻"。

#### 改装为电流表（量程大）
- 把表头与一个**小阻值分流电阻 $R_2$ 并联**：

$$R_2 = \dfrac{I_g R_g}{I - I_g}$$

- 电流表可看作"电流可读的小电阻"。

### 例题：表头改大量程电流表

> 表头 $R_g = 30$ Ω，$I_g = 1$ mA，改装为 0–0.6 A 电流表。
- 通过分流电阻的电流 $I_R = 0.6 - 0.001 = 0.599$ A
- 分流电阻 $R = U_g/I_R = I_g R_g / I_R = 30 \times 10^{-3}/0.599 \approx 5\times10^{-2}$ Ω
- 改装后内阻 $R_A = R_g R / (R_g + R) \approx 5\times10^{-2}$ Ω

### 分压电路（电位器接法）

- 滑动变阻器三端接法：两端接电源，滑动端取出电压。
- 与"限流接法"（两端只接一个）的区别：**分压**输出从 0 到 U 连续可调，**限流**只能在某范围调节。

### 易错提示

- 电压表实际并联在被测元件两端；电流表实际串联进电路。
- 改装后电压表读数 = 表头电流 × （$R_g + R_1$）= 总电压；电流表读数 = 总电流。
- 表头本身不能直接测大电流 / 大电压，会烧表。
- 不能把电流表当电压表使（直接并联在电源上 → 烧表，因为内阻很小）。

## 5. 实验：练习使用多用电表

### 多用电表的功能

- 直流电流、直流电压、交流电压（一般 2.5–250 V）、电阻。

### 表盘与选择开关

- 顶部表盘：上层是电阻刻度（**右零左∞**，**非线性、不均匀**），下层是电压电流刻度（线性）。
- 选择开关：旋至对应物理量及量程。
- 不使用时旋至 OFF（或交流电压最高挡）。

### 测量步骤

1. **机械调零**：旋指针定位螺丝使指针指 0（每次开机检查）。
2. **选择量程**：超过待测最大估计值。
3. **接入电路**：
   - 电压：并联，红表笔接高电位端；
   - 电流：串联，电流从红表笔流入；
   - 电阻：表笔分别接电阻两端，**红负黑正**（电池正极在黑表笔内）。
4. **欧姆调零**（测电阻前必做）：表笔短接，调"欧姆调零旋钮"使指针指右端 0 刻度。
5. **换挡前必须重新欧姆调零**。
6. **读数**：电阻值 = 表盘读数 × 倍率；指针偏在**刻度盘中间区域**时误差最小。
7. **测完旋至 OFF**（或交流最高挡），防止下次误用大电流烧表。

### 红表笔与黑表笔（电阻挡）

- 多用电表内部有干电池为电阻测量供电；
- **黑表笔接内部电池正极**，红表笔接负极；
- 测电阻时电流由黑表笔流出 → 进入待测电阻 → 由红表笔流回；
- 这是测**二极管单向导电性**、判断三极管管脚的依据。

### 数字多用电表

- 直接以数字形式显示读数，无需估读；
- 内部有 A/D 转换电路，可测多种物理量。

### 故障诊断（电压挡）

- 把红表笔固定在电源正极端，黑表笔依次接触电路各节点。
- 表盘读数为 0 表示该点与正极等势（电流未中断）；
- 读数 = 电源电压表示该点与正极间断路。

### 易错提示

- **电阻挡严禁带电测量**——电池电压会与外电压叠加，毁坏表头。
- 换挡前先把表笔与电路分离。
- 二极管反向接通时电阻很大；正反向电阻之比可以判断二极管好坏。
- 电压表测电阻**不能**直接用 V 挡放在电阻两端读（没有电流没有数）；必须**电阻挡**。

## 章末公式速查

```
电流              I = q / t = neSv
欧姆定律          I = U / R     (部分电路)
电阻定律          R = ρ l / S

串联              I 同，U 加，R 加
并联              U 同，I 加，1/R = Σ 1/R_k；两阻并 R = R₁R₂/(R₁+R₂)

分压               U_k / U = R_k / R          (串联)
分流              I_k / I = R / R_k          (并联)

电压表改装        R 串 = U / I_g − R_g
电流表改装        R 并 = I_g R_g / (I − I_g)
```
